 **MSIT: 現実の世界移行ストーリー Gentran 5.1 から BizTalk 2010**  
  
 **作成者:** Amit Kumar Dua、Microsoft & #124 です。 Banupriya Thirumaran、Microsoft  
  
 **校閲者:** Mandi Ohlinger、Microsoft & #124 です。 Nitin Mehrotra、Microsoft

 **適用対象:** BizTalk Server 2010、Gentran サーバー  
  
 **概要:** Microsoft IT (MSIT) 統合プラットフォームは、BizTalk Server と Gentran を使用します。 最近まで、Microsoft IT Gentran フット プリントの適切な支持されていました。 クラウド コンピューティングと AS2 と EDI 機能をサポートする BizTalk Server、Microsoft BizTalk Server 2010 と Gentran を置換する機会が存在します。  Windows Server 2003 および SQL server 2005; Gentran をサポートしています拡張のサポートがすぐに終了することであります。 Windows Server 2008/2012 および 2014/SQL Server 2008 を含む、新しいプラットフォームをサポートするために Gentran のロードマップではありません。  
  
 MSIT は、これをする機会として表示。  
  
-   新しいテクノロジ プラットフォームへの取得します。  
  
-   サポートの制約を削除します。  
  
-   統合プラットフォームの全体的なアーキテクチャを簡素化します。  
  
-   コスト効果の高い、堅牢な統合プラットフォームを行う  
  
 BizTalk Server 2010 は、実証済みの統合プラットフォームはの多数の機能と機能、および新しいプラットフォーム テクノロジをサポートしています。  
  
 このホワイト ペーパーでは、戦略と Gentran 5.1 から BizTalk Server に移行する手順について説明します。
