# <a name="msit-real-world-migration-story-from-gentran-51-to-biztalk-2010"></a>BizTalk 2010 へ Gentran 5.1 から MSIT: 実際の移行プロセス
BizTalk の技術記事  
  
 **MSIT: Gentran 5.1 から BizTalk 2010 への実際の移行事例**  
  
 **作成者:** Amit Kumar Dua, Microsoft &#124; Banupriya Thirumaran, Microsoft  
  
 **校閲者:** Mandi Ohlinger、Microsoft &#124; Nitin Mehrotra, Microsoft  
  
 **寄稿者:** Nikhil Tayal, Microsoft &#124; Anil チャンドラ Lingam, Microsoft  
  
 **公開日:** 2014 年 10 月  
  
 **適用対象:** BizTalk Server 2010、Gentran サーバー  
  
 **概要:** Microsoft IT (MSIT) の統合プラットフォームは、BizTalk Server と Gentran を使用します。 最近まで、Microsoft IT は Gentran フット プリントの適切なプレゼンスがありました。 クラウド コンピューティングおよび AS2 または EDI 機能をサポートしている BizTalk Server では、営業案件に Gentran を Microsoft BizTalk Server 2010 に置き換えます。  Windows Server 2003 および SQL server 2005; Gentran をサポートしています延長サポートが間もなく終了することであります。 Windows Server 2008/2012、2014/SQL Server 2008 などの新しいプラットフォームをサポートする Gentran のためのロードマップではありません。  
  
 MSIT は、これをする機会と見なします。  
  
- 新しいテクノロジ プラットフォームに取得します。  
  
- サポート性制約を削除します。  
  
- 統合プラットフォームの全体的なアーキテクチャを簡略化します。  
  
- コスト効率の高い、堅牢な統合プラットフォームを作成します。  
  
  BizTalk Server 2010 は、実績のある統合プラットフォームですには、多数の機能と機能、および新しいプラットフォーム テクノロジをサポートしています。  
  
  このホワイト ペーパーでは、戦略と Gentran 5.1 から BizTalk Server に移行する手順について説明します。  
  
  ドキュメントを確認するには、ダウンロード、 [MSIT: Gentran 5.1 から BizTalk 2010 に実際の世界の移行状況](http://download.microsoft.com/download/6/D/E/6DEE8EE9-0F26-4991-8FE5-B0E5239C0980/Real%20World%20Migration%20Story%20from%20Gentran%20to%20BizTalk.docx)Word 文書です。
