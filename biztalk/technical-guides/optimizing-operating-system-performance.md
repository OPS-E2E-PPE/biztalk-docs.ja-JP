# <a name="optimizing-operating-system-performance"></a>オペレーティング システムのパフォーマンスを最適化します。
このセクションでは、実稼働の BizTalk Server 環境で使用する BizTalk Server コンピューターのパフォーマンスを最適化するための推奨事項を説明します。 ここで示した最適化は、BizTalk Server をインストールして、構成後に適用されます。  
  
> [!IMPORTANT]  
>  このセクションの推奨事項の一部には、Windows Server のレジストリへの変更が必要があります。 レジストリ エディターの不適切な使用により問題が発生し、オペレーティング システムの再インストールが必要となることがあります。 レジストリ エディターは、ご自身の責任で使用してください。 バックアップ、復元、およびレジストリを変更する方法の詳細については、Microsoft サポート技術情報の記事を参照してください。[上級ユーザー向けの Windows レジストリ情報](http://go.microsoft.com/fwlink/?LinkId=62729)(http://go.microsoft.com/fwlink/?LinkId=62729)。
