- [Operations Manager 2007 R2 または 2012 がサポートされている構成](http://go.microsoft.com/fwlink/?LinkId=90676)  
  
- [Microsoft サポート ライフ サイクル ポリシー](http://go.microsoft.com/fwlink/?LinkId=123820)  
  
  [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 監視管理パックは、次のバージョンの BizTalk Server と関連付けられているロケールをサポートします。  
  
|                                             バージョン                                              | Supported |
|---------------------------------------------------------------------------------------------------|-----------|
| [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] 2013 (英語) |    はい    |
|                              その他のすべてのバージョンの BizTalk とロケール                               |    いいえ     |
