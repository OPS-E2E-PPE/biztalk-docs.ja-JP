# <a name="installing-biztalk-server-2010-accelerator-for-swift"></a>Installing BizTalk Server 2010 Accelerator for SWIFT
![Logo](../technical-guides/media/bts-10-installaccelerator-logo.gif "BTS_10_InstallAccelerator_Logo")  
  
 BizTalk の技術記事  
  
 **BizTalk Server 2010 Accelerator for SWIFT のインストール**  
  
 **ライター:** Maria Quian、Microsoft Corporation  
  
 **技術校閲者:** Andres Naranjo、Mandi Ohlinger  
  
 **公開日。** 2012 年 6 月  
  
 **適用対象します。** BizTalk Server 2010 Accelerator for SWIFT  
  
 **概要:** Microsoft BizTalk Server 2010 Accelerator for SWIFT (A4SWIFT) のインストールが成功の基盤は、A4SWIFT の機能と各機能が依存する BizTalk コンポーネントの理解。 A4SWIFT の各機能は、適切に構成されたその他の BizTalk コンポーネントに依存します。 A4SWIFT 環境の構築に影響するその他の考慮事項には、既存のネットワーク インフラストラクチャと構成済みの BizTalk Server 環境が含まれます。 このドキュメントは参照し、BizTalk Server 環境を構築するために既存のドキュメントを使用します。 A4SWIFT では、BizTalk Server 2010 のすべてのソフトウェアとハードウェア要件を継承します。  
  
 インストール ガイドでは、各 A4SWIFT 関連コンポーネントですが、1 つコンパイルされたドキュメントではなく、インストール/構成手順について詳しく説明する使用できます。 このドキュメントはそれらをまとめると、いくつかのインストールと構成の中に発生した一般的な問題を識別します。  
  
 ドキュメントを確認するには、ダウンロード、[をインストールする BizTalk Server 2010 Accelerator for SWIFT](http://go.microsoft.com/fwlink/?LinkId=255118) Word 文書です。
