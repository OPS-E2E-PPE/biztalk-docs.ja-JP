# <a name="mscit-supply-chain-migration-from-biztalk-server-2009-hub-to-biztalk-server-2013-r2-virtualized-hub"></a>MSCIT:BizTalk Server 2009 ハブから BizTalk Server 2013 R2 仮想化ハブへのチェーンの移行を指定します。
アプリケーションの移行は、BizTalk Server 2010 から BizTalk Server 2013 R2。  
  
 **作成者**:Arvind Chaudhary, Microsoft &#124; Piyush、, Microsoft  
  
 **発行**:2015 年 9 月  
  
 **適用対象**:BizTalk Server 2013 R2  
  
 **概要**:上の理由から、アプリケーションを BizTalk Server 2009 および 2010 から BizTalk Server 2013 R2 に移行する Microsoft IT が行った移行プロセスについて説明します。 MSIT で、メッセージ ストリームを識別、Visual Studio で BizTalk プロジェクトを変換、取引先パートナー管理のパーティーと契約に移動、および、アプリケーションのバインドを更新します。  
  
 MSIT 直面する課題および使用のベスト プラクティスについて読むこともできます。
