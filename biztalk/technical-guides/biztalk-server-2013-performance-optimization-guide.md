## <a name="microsoft-biztalk-server-2013-performance-optimization-guide"></a>Microsoft BizTalk Server 2013 のパフォーマンス最適化ガイド  
 **まとめ**  
  
 BizTalk Server 2013 のパフォーマンス最適化ガイドには、BizTalk Server を使用して広範なきた IT プロフェッショナルの実践的な経験に基づいて、BizTalk Server のパフォーマンスを最適化するための規範的なガイダンスが含まれています。 このガイドには、次の 4 つの主要なセクションが含まれています。検索して、ボトルネック、パフォーマンスの最適化、運用環境の BizTalk Server 環境のスケーリング、および BizTalk Server のパフォーマンス テストの方法論を削除します。  
  
 オペレーティング システム、SQL Server の構成、およびその他のパフォーマンス設定は、BizTalk Server 2013 R2 にも適用されます。  
  
 ドキュメントを確認するには、ダウンロード、 [Microsoft BizTalk Server 2013 のパフォーマンス最適化ガイド](http://download.microsoft.com/download/6/D/E/6DEE8EE9-0F26-4991-8FE5-B0E5239C0980/BizTalkServer2013-PerformanceGuide.docx)Word 文書です。
