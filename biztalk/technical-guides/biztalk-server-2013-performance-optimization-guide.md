## <a name="microsoft-biztalk-server-2013-performance-optimization-guide"></a>Microsoft BizTalk Server 2013 パフォーマンス最適化ガイド  
 **概要**  
  
 BizTalk Server 2013 のパフォーマンス最適化ガイドには、BizTalk Server を使用して広範なきた IT 技術者の実践的な経験に基づいて、BizTalk Server のパフォーマンスを最適化するための規範的なガイダンスが含まれています。 このガイドには、次の 4 つの主要なセクションが含まれています。 検索とボトルネックを排除すること、パフォーマンスの最適化、スケーリング実稼働の BizTalk Server 環境、および BizTalk Server のパフォーマンスのテスト方法です。  
  
 オペレーティング システム、SQL Server の構成、およびその他のパフォーマンス設定は、BizTalk Server 2013 R2 にも適用されます。
