# <a name="viewing-information-in-the-operations-manager-console"></a>Operations Manager コンソールによる情報の表示
提供されるビューを使用して、[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]管理パックを現在の可用性、構成、正常性、および BizTalk Server 環境のパフォーマンスを理解します。 表示には、オブジェクトの長い一覧を含むことができます。 特定のオブジェクトまたはオブジェクトのグループを検索するには、Operations Manager のツールバーで、スコープおよび検索ボタンを使用できます。 詳細については、管理の監視データを使用して、スコープ、検索する方法を参照してくださいし、Operations Manager 2007 R2\2012 ヘルプ トピックを検索します。  
  
 次のビューがすぐ下に表示**[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]** オペレーション コンソールの [監視] ウィンドウでノード。  
  
-   **アプリケーションのビュー**:BizTalk 管理者は、さまざまな BizTalk Server アイテムの正常性と状態を監視およびオーケストレーションなどのアプリケーションは、送信ポート、受信場所、および、具合です。  
  
-   **展開ビュー**:企業の IT 管理者は、監視対象の状態とさまざまなエンタープライズ展開ホスト インスタンス コンピューター、IIS、SSO サービスをホストするマシンの SQL Server データベースをホストするコンピューターの正常性がネットワーク サービス、具合です。  
  
## <a name="application-views"></a>アプリケーションのビュー  
 アプリケーションのビューには、次の表で説明する要素が含まれます。  
  
### <a name="elements"></a>要素  
  
|ビュー名|説明|  
|---------------|-----------------|  
|アプリケーションの状態の表示|これは、BizTalk アプリケーションの正常性を表示するダッシュ ボード ビューです。 BizTalk アプリケーションの正常性は、その構成アイテムでオーケストレーション、送信ポート グループ、送信ポート、受信ポート、および受信場所などの正常性に依存します。 詳細ビュー ウィンドウでは、ホストされる BizTalk アプリケーションのプロパティを提供します。|  
|グループの状態の表示|これは、BizTalk グループの正常性を表示するダッシュ ボード ビューです。 BizTalk グループの正常性は、BizTalk ホストと BizTalk アプリケーションの正常性に依存します。 詳細ビュー ウィンドウでは、BizTalk グループのプロパティを提供します。|  
|ホストの状態の表示|これは、BizTalk ホストの正常性を表示するダッシュ ボード ビューです。 BizTalk ホストの正常性は、ホストされている BizTalk アプリケーションのインスタンスの正常性に依存します。 詳細ビュー ウィンドウでは、BizTalk ホストのプロパティを提供します。|

|受信場所の状態の表示|ホステッド アプリケーションの受信場所の構成とランタイムの状態を表示します。|  
|受信ポートの状態の表示|ホステッド アプリケーションの受信ポートの構成とランタイムの状態を表示します。|  
|送信ポート グループの状態の表示|ホステッド アプリケーションの送信ポート グループの構成とランタイムの状態を表示します。|
