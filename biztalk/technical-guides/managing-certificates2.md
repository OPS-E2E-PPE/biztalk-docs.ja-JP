# <a name="managing-certificates"></a>証明書の管理
このセクションでは、BizTalk Server で使用されるデジタル証明書を管理する方法について説明します。 そのトピックは、証明書をインストールする方法 (および、どのフォルダーにそれらをインストールする) について説明します。 構成する方法と[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]MIME/SMIME および AS2、および BizTalk アダプターで使用するために、証明書を使用します。 Microsoft BizTalk Server には、ドキュメントの暗号化および復号化のために公開キー基盤 (PKI) デジタル証明書を使用して、ドキュメントの署名と検証 (否認)、およびパーティの解決を行うことができます。
