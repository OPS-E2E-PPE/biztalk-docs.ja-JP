# <a name="update-references-to-the-bam-archive-database-name"></a>BAM アーカイブ データベース名への参照を更新します。
BAM アーカイブ データベースをバックアップした場合は、システムまたはデータに障害が発生した場合、別のコンピューターにそのバックアップを復元することができ、バックアップの名前を変更することができます。  
  
 BAM アーカイブ データベースを復元する手順を実行[Backup BizTalk Server ジョブでデータベースを復元する方法](../technical-guides/how-to-restore-databases-in-the-backup-biztalk-server-job.md)します。 さらに、新しいサーバー名とデータベース名を BAM SSIS パッケージを更新する必要があります。  
  
## <a name="how-to-update-references-to-bam-archive-database"></a>BAM アーカイブ データベースへの参照を更新する方法  
 BAM アーカイブ データベースへの参照を更新する方法については、次を参照してください。[新しい BAM アーカイブ データベースへの参照の更新](../technical-guides/how-to-move-the-bam-archive-database1.md#BKMK_UpdateArch)します。
