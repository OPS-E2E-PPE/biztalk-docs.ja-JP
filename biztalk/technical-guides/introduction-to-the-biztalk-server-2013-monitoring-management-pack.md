# <a name="introduction-to-the-biztalk-server-2013-monitoring-management-pack"></a>BizTalk Server 2013 Monitoring 管理パックの概要
[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]を主体的および対応の BizTalk Server の展開、アプリケーション、およびその構成アイテムの監視管理パックを提供します。 この管理パックによって提供される監視には、正常性の監視、可用性と構成の監視、パフォーマンス データの収集、および既定のしきい値が含まれています。 サービス指向の監視シナリオに、BizTalk Server コンポーネントの監視を統合することができます。 この管理パックが提供される Web からダウンロードします。  
  
 ヘルス監視機能に加えて、この管理パックには、リアルタイムの診断と検出された問題の解決を有効にするアプリケーションおよび展開のビューなどのダッシュ ボード ビューが含まれます。 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]包括的な BizTalk Server の重要なイベントと一元管理を提供するパフォーマンス カウンターの監視と BizTalk Server の展開とアプリケーションのエクスペリエンスを監視用管理パックを提供します。
