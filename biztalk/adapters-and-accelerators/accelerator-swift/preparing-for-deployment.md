# <a name="preparing-for-deployment"></a>展開の準備
このセクションでは、展開の計画と準備フェーズに関する情報を提供します。 展開を実装する前に、次の作業を行います。  
  
1.  読み取りおよび、既知の問題を理解および機器を準備します。  
  
    -   ハードウェアと、展開、およびネットワークの図と展開ワークシート (ネットワーク ダイアグラムと選択した展開アーキテクチャに基づいて)、展開のバリエーションを反映するために必要なソフトウェアのチェックリストを作成します。  
  
    -   ハードウェアとソフトウェアのチェックリストを取得します。 この購入には、すべてのサービス パック、修正プログラム、および配置に必要なソフトウェア更新プログラムのダウンロードが含まれます。  
  
         インターネットにアクセスできるコンピューターを使用してこれらのソフトウェア コンポーネントをダウンロードして、これらのソフトウェア コンポーネントを簡単に配布の CD-ROM をコピーします。  
  
2.  さまざまな段階に、展開を分離し、各ステージに関連付けられているタスクを特定します。 この方法は、特定の展開段階に従ってタスクを整理することによって、展開プロセスを簡略化します。 たとえば、所定の展開には、次のサーバーの展開段階が含まれています。  
  
3.  ドメイン コント ローラーを構成します。  
  
4.  データベースの構成 (実行時間も含めて[!INCLUDE[btsSQLServerNoVersion](../../includes/btssqlservernoversion-md.md)]クラスターおよびデザイン時のデータベース サーバー)。  
  
5.  BizTalk server を構成します。
