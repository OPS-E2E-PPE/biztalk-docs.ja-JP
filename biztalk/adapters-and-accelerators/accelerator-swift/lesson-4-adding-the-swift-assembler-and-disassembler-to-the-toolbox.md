# <a name="lesson-4-adding-the-swift-assembler-and-disassembler-to-the-toolbox"></a>レッスン 4: SWIFT アセンブラーと逆アセンブラーをツールボックスに追加します。
このレッスンで BizTalk パイプライン デザイナを使用する場合は、これらのコンポーネントをツールボックスで利用できるように (ASM) の SWIFT アセンブラーおよび逆アセンブラー (逆アセンブラー) への参照を追加します。  
  
### <a name="to-add-the-swift-asm-and-dasm-components-to-the-toolbox"></a>ツールボックスに SWIFT ASM と逆アセンブラー コンポーネントを追加するには  
  
1. Visual Studio での**ツール**] メニューのをクリックして **[ツールボックス アイテムの**します。  
  
2. ツールボックス アイテムの選択 ダイアログ ボックスで、上、 **BizTalk パイプライン コンポーネント** タブで  **SWIFT アセンブラー**と**SWIFT 逆アセンブラー**します。  
  
3. **[OK]** をクリックします。  
  
   まだなければ、受信または送信パイプラインを開くには、この場合、ASM や逆アセンブラーを使用できない、アセンブラーおよび逆アセンブラー コンポーネントはツールボックスから、使用可能なようになりました。  
  
   続行する[レッスン 5: SWIFT 逆アセンブラーをカスタム受信パイプラインに追加する](../../adapters-and-accelerators/accelerator-swift/lesson-5-adding-the-swift-disassembler-to-a-custom-receive-pipeline.md)します。
