# <a name="lesson-4-adding-the-swift-assembler-and-disassembler-to-the-toolbox"></a>レッスン 4: SWIFT アセンブラーおよび逆アセンブラーをツールボックスに追加します。
このレッスンでは、BizTalk パイプライン デザイナを使用すると、これらのコンポーネントがツールボックスで使用できるようにに SWIFT アセンブラー (ASM) および逆アセンブラー (DASM) への参照を追加します。  
  
### <a name="to-add-the-swift-asm-and-dasm-components-to-the-toolbox"></a>SWIFT ASM と逆アセンブラー コンポーネントをツールボックスに追加するには  
  
1.  Visual Studio での**ツール** メニューのをクリックして**ツールボックス アイテムの選択**です。  
  
2.  ツールボックス アイテムの選択 ダイアログ ボックスで、 **BizTalk パイプライン コンポーネント** タブで、 **SWIFT アセンブラー**と**SWIFT 逆アセンブラー**です。  
  
3.  **[OK]** をクリックします。  
  
 アセンブラーおよび逆アセンブラー コンポーネントは、ここで、ツールボックスから使用可能ながあるまだなければ、受信パイプラインまたは送信パイプラインを開くには、その場合、ASM または DASM できない場合があります。
