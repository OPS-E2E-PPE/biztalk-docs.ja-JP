# <a name="a4swift-component-configuration-guide"></a>A4SWIFT コンポーネント構成ガイド
このガイドは、Microsoft の構成についての情報を提供します。[!INCLUDE[A4SWIFT_CurrentVersion_FirstRef](../../includes/a4swift-currentversion-firstref-md.md)]します。 A4SWIFT をインストールして (ように、インストール ガイドで説明) は、A4SWIFT 構成ウィザードを完了した後は、この構成のガイドで手順を実行します。 この構成のガイドには、次の手順が含まれています。  

- インストール後の手順のメッセージング シナリオ用の A4SWIFT ランタイムを構成します。  

- Message Repair and New Submission を構成する方法。 これを行うにする必要がありますまず手動で構成する A4SWIFT ランタイム。 FIN Response Reconciliation を構成することは不要です。
