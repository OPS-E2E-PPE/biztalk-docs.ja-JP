# <a name="module-5-adding-a-flat-file-receive-location-and-xml-send-port"></a>第 5 章:受信場所のフラット ファイルを追加して、XML 送信ポート
このモジュールでは、1 つのフラット ファイルの受信ポートと場所、および 1 つの XML 送信ポートを追加します。 ファイル フォルダーから MT103 のフラット ファイルを受信するのにには、場所と受信ポートを使用します。 MT103 メッセージを XML 形式で、送信ファイルのフォルダーに送信するのにには、送信ポートを使用します。
