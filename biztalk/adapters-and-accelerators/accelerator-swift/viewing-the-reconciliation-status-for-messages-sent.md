# <a name="viewing-the-reconciliation-status-for-messages-sent"></a>送信されたメッセージの調整状態を表示します。
送信されるメッセージの調整状態を表示するには。  
  
1.  プログラムは、次のようにクリックします。 **Microsoft BizTalk Server\<バージョン\>**、 をクリックし、 **BAM ポータル Web サイト**します。  
  
2.  BAM ポータル Web サイトの マイ View\XMLV2View\Activity 検索、選択**メッセージ送信アクティビティ**します。  
  
3.  列の選択 でのすべての列を移動**使用可能なデータ**と**表示の項目をマイルス トーン**します。  
  
4.  クリックして**クエリ実行**します。 すべてのメッセージ送信アクティビティに対して保存されているレコードが返されます。  
  
5.  同様に、配信通知、配信レポート、レポートの送信、および概要のアクティビティのデータを表示できます。
