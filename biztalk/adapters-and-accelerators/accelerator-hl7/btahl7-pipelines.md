# <a name="btahl7-pipelines"></a>BTAHL7 のパイプライン
Microsoft BizTalk Accelerator 用 HL7 ([!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]) 受信のセットを提供および HL7 エンコードし、XML でエンコードされた両方の処理のパイプラインを送信します。 これらのパイプラインは、医療ソリューションの BizTalk パイプライン処理を強化します。  
  
- HL7 FF の 2.X (します。BTP) を提供する[!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]2XReceive と[!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]2XSend パイプライン  
  
- HL7XML の 2.XML (します。BTP) を提供する[!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]2XMLReceive と[!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]2XMLSend
