# <a name="administration-and-run-time-tools"></a>管理およびランタイム ツール
Microsoft BizTalk Server と Microsoft [!INCLUDE[btaBTAHL7NoNumber](../../includes/btabtahl7nonumber-md.md)] ([!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]) 展開、構成管理、管理、ビジネス ルールの管理などの管理および実行時のタスクを実行して、監視のための一連のツールを提供します。 詳細については[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]ツール、MicrosoftBizTalk Server のヘルプを参照してください。  
  
## <a name="biztalk-explorer"></a>BizTalk エクスプ ローラー  
 構成して、プロジェクト、パーティ、およびオーケストレーションを管理するには、BizTalk エクスプ ローラーを使用します。 BizTalk エクスプ ローラーでは、アセンブリ、ポート、パーティなどの項目 (階層ツリー) 形式で表示、構成データベース (BizTalk 管理データベースとも呼ばれます) の内容を表示します。  
  
## <a name="btahl7-configuration-explorer"></a>BTAHL7 構成エクスプ ローラー  
 [!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]構成エクスプ ローラーを確認してバッチ定義、受信確認の構成、およびログのデータ ストアの設定を構成します。 使用して[!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]構成エクスプ ローラーを実行できますイニシアチブのライセンス認証、要求、およびバッチ処理はパーティ レベルでの終了。  
  
## <a name="business-rule-engine"></a>ビジネス ルール エンジン  
 BizTalk ビジネス ルール エンジン (BRE) は、ファクトに対してルールを評価し、評価の結果に基づいてアクションを開始するに使用できるランタイム推論エンジンです。  
  
## <a name="windows-performance-monitors"></a>Windows パフォーマンス モニター
