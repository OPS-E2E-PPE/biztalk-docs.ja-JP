# <a name="party-aliases-tab"></a>パーティ エイリアス タブ
使用する、**パーティ エイリアス**タブが選択されているパーティに関する情報を表示します。  
  
> [!NOTE]
>  パーティを作成するのにには、BizTalk エクスプ ローラーを使用します。  
  
 パーティを作成する方法の詳細については、「を管理するパーティを使用して BizTalk エクスプ ローラー」を BizTalk Server のヘルプを参照してください。  
  
 **BTAHL7 構成エクスプ ローラー**  ダイアログ ボックスで、**パーティ エイリアス** タブで、次の操作を行います。  
  
|プロパティ|目的|  
|--------------|----------------|  
|**名前**|パーティの組織の名前を表示します。|  
|**[値]**|パーティ ID の値を表示します。|
