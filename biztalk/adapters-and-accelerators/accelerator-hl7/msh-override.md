# <a name="msh-override"></a>MSH 上書き
前に[!INCLUDE[btsCoName](../../includes/btsconame-md.md)]BizTalk Accelerator 用 HL7 ([!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]) MSH セグメント内の特定の値を置換する場合は、送信ポートに送信 HL7 メッセージをルーティングします。 これを行うときに、メッセージのサブスクリプションには、複数の送信ポートが含まれているし、各送信ポートが受信アプリケーション フィールドに別の値を確認することがあります。
