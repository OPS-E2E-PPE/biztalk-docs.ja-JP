# <a name="operational-logging-message-batching-validation-and-asknowledgment-settings"></a>操作ログ記録、メッセージのバッチ処理、検証および受信確認の設定
Microsoft[!INCLUDE[btaBTAHL7NoNumber](../../includes/btabtahl7nonumber-md.md)]サーバーと一連のエンタープライズ アプリケーション統合 (EAI)、ツール、ビジネス プロセスを自動化し、ビジネス パートナーとの対話を促進することが含まれます。 上に構築された、[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]プラットフォーム、BTAHL7 は HL7 標準を使用して、施設の医療のアプリケーション統合を昇格させます。 HL7 には、交換する、統合、および臨床試験の演習と管理の電子的な情報を取得するための標準が含まれています。  
  
 インターフェイスのユーザーのアナリストは、取引先、メッセージの受信確認、メッセージのバッチ処理、およびビジネス プロセスに必要なその他の詳細を構成します。  
  
 このセクションでは、効率的に監視および BTAHL7 アプリケーション環境を保持することができます、概念と手順の情報を提供します。
