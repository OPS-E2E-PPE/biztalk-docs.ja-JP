# <a name="lesson-5-deploy-the-solution"></a>レッスン 5: ソリューションをデプロイします。
このレッスンで配置する、BizTalk ソリューションを使用して、[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]管理コンソール。 BizTalk オーケストレーション デザイナーでは、論理ポートを作成します。 物理ポートを作成して論理ポートを物理ポートにマッピングすることによって、BizTalk アプリケーションを構成します。  
  
 アプリケーションを構成した後に開始し、アプリケーションをテストします。
