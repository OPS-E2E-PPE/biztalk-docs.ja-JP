# <a name="lesson-5-deploy-the-solution"></a>レッスン 5: ソリューションを配置します。
このレッスンで配置する、BizTalk ソリューションを使用して、[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]管理コンソールです。 BizTalk オーケストレーション デザイナーでは、論理ポートを作成します。 物理ポートが作成され、論理ポートを物理ポートにマップすることによって、BizTalk アプリケーションを構成します。  
  
 アプリケーションを構成した後に開始して、アプリケーションをテストします。
