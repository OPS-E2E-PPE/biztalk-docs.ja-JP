このトピックでは、Wcf-oracledb アダプターを追加する方法を手順については、[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]管理コンソール。  
  
> [!IMPORTANT]
>  WCF カスタム ポートを構成する場合これらのタスクを実行する必要ありません、[!INCLUDE[adapteroracle_short](../../includes/adapteroracle-short-md.md)]します。  
  
## <a name="add-the-oracle-database-adapter"></a>Oracle データベース アダプターを追加します。  
  
1. 開く、[!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]管理コンソール。  
  
2. 展開、 **BizTalk グループ**、展開**プラットフォームの設定**、し、**アダプター**します。  
  
3. 右クリック**アダプター**を選択します**新規**、選び**アダプター**します。  
  
    ![アダプターを追加する](../../adapters-and-accelerators/media/c9610d42-8465-4099-b403-87df6dcd0d99.gif "c9610d42-8465-4099-b403-87df6dcd0d99")  
  
4. **アダプター プロパティ**] ダイアログ ボックスに、アダプターとの間の名前を入力、**アダプター**一覧で、[ **Wcf-oracledb**します。
