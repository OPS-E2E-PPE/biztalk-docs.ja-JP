## <a name="biztalk-explorer"></a>BizTalk エクスプ ローラー  
 構成して、プロジェクト、パーティ、およびオーケストレーションを管理するには、BizTalk エクスプ ローラーを使用します。 BizTalk エクスプ ローラーでは、アセンブリ、ポート、パーティなどのアイテムを表示、構成データベースの内容を表示します。
