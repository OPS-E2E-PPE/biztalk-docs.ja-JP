# <a name="creating-the-contoso-solution"></a>Contoso ソリューションの作成
このセクションでは、Contoso 組織に従う必要のある手順について説明します。 Microsoft® を使用して 2 つの組織の連絡先情報および取引先契約を追加するには、まず[!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)]管理コンソール。 基幹業務 (LOB) スキーマを作成し、それらのスキーマと、それぞれの RosettaNet ベース プロセス PIP (Partner Interface) スキーマの間のマッピングを作成します。 次に、情報を Fabrikam に送信するために、ERP システムと HTTP アダプターと通信するための SQL アダプタを使用するポート情報を構成します。 最後の手順では、BizTalk Server でビジネス ルール エンジン (BRE) を使用するプライベート プロセス オーケストレーションをカスタマイズします。
