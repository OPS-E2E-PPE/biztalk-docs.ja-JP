# <a name="uninstall-or-repair-the-fileact-and-interact-adapter"></a>アンインストールまたは修復 FileAct および InterAct アダプター
アンインストールまたは BizTalk FileAct および InterAct アダプターを修復する手順。  
  
## <a name="uninstall-the-fileact-and-interact-adapter"></a>FileAct および InterAct アダプターをアンインストールします。  
  
1.  **コントロール パネルの** **プログラムのアンインストール**です。  
  
2.  選択、 **BizTalk FileAct と対話するアダプター**、し、**アンインストール**または**変更**です。  
  
3.  選択、**削除**オプションをクリックし **次**です。  
  
4.  つまり、次のように選択します。**アンインストール**です。  
  
5.  選択**完了**完了したときにします。

## <a name="repair-the-fileact-and-interact-adapter-installation"></a>FileAct を修復し、アダプターのインストールの操作  
  
1.  **コントロール パネルの** **プログラムのアンインストール**です。  
  
2.  選択、 **BizTalk FileAct と対話するアダプター**、し、**変更**です。  
3.  **プログラムのメンテナンス** ページで、、**修復**オプションをクリックし **次**です。  
  
4.  つまり、次のように選択します。**アンインストール**です。  
  
5.  選択**完了**完了したときにします。
