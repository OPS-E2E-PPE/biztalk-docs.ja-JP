# <a name="interact-adapter-server-application"></a>InterAct アダプタ サーバー アプリケーション
このセクションでは、サーバー アプリケーションの要求メッセージのコンポジションの一般的な説明を表示します。 これらは、サーバー側 SNL からサーバー アプリケーションに渡される XML ドキュメントです。 そのため、これらのメッセージには、サーバー側で実行される SWIFTNet プリミティブの最初の部分の特性が。  
  
 次の図は、サーバーに要求メッセージを示します。
