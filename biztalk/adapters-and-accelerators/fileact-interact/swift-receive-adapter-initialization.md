# <a name="swift-receive-adapter-initialization"></a>SWIFT 受信アダプターの初期化
SWIFT サーバー アプリケーションの初期化プロセスが SWIFT クライアント アプリケーションと同様にします。 初期化の引数に渡されるコマンド ライン パラメーターとして、受信側実行可能ファイルです。 設計時に、受信場所で構成されている、アプリケーション名は、コマンドライン引数として渡されます。 このアプリケーション名は、受信場所、構成されたプロパティを取得する URI の構築に使用されます。  
  
 コマンドライン引数を使用する方法の詳細については、Microsoft では"構成の受信アダプター Environment 方法"トピックを参照してください。[!INCLUDE[swift_adapter](../../includes/swift-adapter-md.md)]インストールおよび構成ガイドです。
