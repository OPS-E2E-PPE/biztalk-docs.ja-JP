# <a name="swift-send-adapter-uri"></a>SWIFT 送信アダプターの URI
送信アダプターの URI が構築されたに基づいて応答側の名前 (SWIFT://\<レスポンダー名前\>)。 SWIFT で定義された識別名をという名前の応答側 これは、先にメッセージが送信する必要を示します。 SWIFT では、値を使用して、受信者にメッセージをルーティングします。
