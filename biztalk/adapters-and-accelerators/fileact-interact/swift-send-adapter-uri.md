# <a name="swift-send-adapter-uri"></a>SWIFT 送信アダプター URI
送信アダプターの URI は、構築されたベース レスポンダー名 (SWIFT://\<レスポンダー名前\>)。 応答側の名前は、SWIFT によって定義された識別名です。 送信先メッセージする必要があることを示します。 SWIFT では、値を使用して、受信者にメッセージをルーティングします。
