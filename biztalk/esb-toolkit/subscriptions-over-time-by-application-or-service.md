# <a name="subscriptions-over-time-by-application-or-service"></a>アプリケーションまたはサービス別の時間の経過と共にサブスクリプション
このオプションは、アプリケーションの指定されたセットの指定された期間に警告のサブスクリプションの数を表示します。 一定期間、アプリケーション内の特定のサービスのサブスクリプションの数を示す傾向グラフを表示するアプリケーションを選択することができます。 特定のサービスを選択するには、そのサービスのサブスクリプションを一覧表示するレポートが表示されます。 サブスクリプションの一覧は、Microsoft Excel のスプレッドシートとしてエクスポートすることもできます。  
  
### <a name="to-view-charts-and-lists-of-subscriptions-over-time-by-application-or-service"></a>アプリケーションまたはサービス別、時間の経過と共にグラフとサブスクリプションの一覧を表示するには  
  
1.  開く、[ポータル レポート ページ](../esb-toolkit/portal-reports-page.md)します。  
  
2.  をクリックして、**アプリケーションの選択**最初のグラフの上にリンクし選択するか表示されるインストール済みの Microsoft BizTalk Server アプリケーションの一覧でチェック ボックスをオフにします。 またはすべてのアプリケーションを選択して表示されるリンクを使用することもできます。 をクリックして**保存**選択したアプリケーションの情報を表示する**キャンセル**変更を破棄します。  
  
3.  グラフの情報を表示する期間を変更するには、次のように選択します。**時間、日、週、月、四半期、年、** または**すべて**最初のグラフの上のドロップダウン リストでします。  
  
4.  サブスクリプションのアプリケーション内のサービスの各トレンド グラフを表示するには、グラフ キー セクションでアプリケーションの名前をクリックします。  
  
5.  選択したサービスのすべてのサブスクリプションを一覧表示するテーブルを表示するには、このグラフでのサービスの名前をクリックします。  
  
6.  クリックして**Excel にエクスポート**を Microsoft Excel で表示できる形式でのサブスクリプションの一覧をダウンロードします。
