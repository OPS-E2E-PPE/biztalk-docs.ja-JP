# <a name="using-the-biztalk-esb-toolkit-web-services"></a>BizTalk ESB Toolkit Web サービスの使用
[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)] Web サービスとしてのすべてのコア機能を公開します。 これにより、ESB、おそらくマイクロソフト以外のプラットフォームで実行されている ESB 交換に参加し、ESB の機能を活用する外部のエンティティ。 すべての Web サービスは、ESB のインストール ディレクトリの \Web フォルダーの個別のサブフォルダーに配置されます。 [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]次の ESB Web サービスが含まれています。  
  
-   [スケジュール オンランプ Web サービス](../esb-toolkit/the-itinerary-on-ramp-web-services.md)  
  
-   [リゾルバー Web サービス](../esb-toolkit/the-resolver-web-service.md)  
  
-   [変換 Web サービス](../esb-toolkit/the-transformation-web-service.md)  
  
-   [例外処理 Web サービス](../esb-toolkit/the-exception-handling-web-service.md)  
  
-   [BizTalk 操作 Web サービス](../esb-toolkit/the-biztalk-operations-web-service.md)
