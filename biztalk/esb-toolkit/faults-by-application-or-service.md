# <a name="faults-by-application-or-service"></a>アプリケーションまたはサービス別のエラー
このオプションは、指定した時間の経過と共に、指定された一連のアプリケーションに対して生成されたすべてのエラーの集計ビューを表示します。 特定のサービスを選択するには、その特定のサービスのエラーを示すレポートが表示されます。 エラーの一覧は、Microsoft Excel のスプレッドシートとしてエクスポートできます。  
  
### <a name="to-view-charts-and-lists-of-faults-by-application-or-service"></a>アプリケーションまたはサービス別のグラフおよびエラーの一覧を表示するには  
  
1.  開く、[ポータル レポート ページ](../esb-toolkit/portal-reports-page.md)します。  
  
2.  をクリックして、**アプリケーションの選択**最初のグラフの上にリンクし選択するか表示されるインストール済みの Microsoft BizTalk Server アプリケーションの一覧でチェック ボックスをオフにします。 またはすべてのアプリケーションを選択して表示されるリンクを使用することもできます。 をクリックし、**保存**を選択したアプリケーションの情報を表示またはをクリックして**キャンセル**変更を破棄します。  
  
3.  グラフの情報を表示する期間を変更するには、次のように選択します。**時間、日、週、月、四半期、年、** または**すべて**最初のグラフの上のドロップダウン リストでします。  
  
4.  アプリケーションを選択します。アプリケーション内でサービスのそれぞれに対してエラーを表示するグラフが変更されます。  
  
5.  クリックして**Excel にエクスポート**Microsoft Excel で表示できる形式でのエラーの一覧をダウンロードします。
