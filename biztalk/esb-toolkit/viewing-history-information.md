# <a name="viewing-history-information"></a>履歴情報を表示します。
ESB の管理ポータルでは、Universal Description, Discovery, and Integration (UDDI) の登録のためのすべての要求の履歴を保持します。 管理者は、ポータル内では、この一覧を表示できます。  
  
### <a name="to-view-uddi-request-history"></a>UDDI の要求履歴を表示するには  
  
1.  (そのうちポータル管理者は自動的にメンバー)、Microsoft BizTalk Server 管理者アカウント グループのメンバーであるアカウントを使用してポータルにログオンすることを確認します。  
  
2.  をポイント、**レジストリ**ポータルのメイン メニュー タブをクリックして**保留中の要求の管理**を開くには、ポータル[保留中の要求 ページの管理](../esb-toolkit/manage-pending-requests-page.md)です。  
  
3.  クリックして、**要求履歴を表示する**リンクします。
