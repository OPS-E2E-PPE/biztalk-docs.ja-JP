# <a name="registry-settings-page"></a>レジストリの設定 ページ
図 1 はレジストリの設定 ページで、設定を変更することができます、ポータルは、Universal Description, Discovery, and Integration (UDDI) サービスとやり取りします。  
  
 ![レジストリの設定 ページ](../esb-toolkit/media/ch8-registrysettingspage.gif "Ch8 RegistrySettingsPage")  
  
 **図 1**  
  
 **ESB 管理ポータルのレジストリ設定ページ**  
  
 次の一覧では、ESB 管理ポータルのレジストリ設定ページの機能を使用する方法について説明します。  
  
-   **UDDI 詳細**ページの上部にあるセクションでは、テキスト ボックスを含む URI または UDDI サーバーの URL を既定値は、ローカルの Microsoft UDDI サービス。 このセクションの下の UDDI の登録の要求の自動-パブリケーションのパブリケーションで有効にして、UDDI サーバーへのアクセスが現在のユーザーの Microsoft Windows アカウントではなく匿名アカウントを使用するかどうかを指定するための 2 つのチェック ボックスがあります。  
  
-   **電子メール通知**ページのセクションには、ポータルの登録要求の通知を自動的に電子メールを送信する場合に選択したチェック ボックスが含まれています。 このチェック ボックスの下では、一連のテキスト ボックスを使用するには、電子メール サーバーの詳細を提供する、"to"と「差出人」電子メール アドレス、メッセージの件名、およびメッセージの本文。  
  
-   **既定の設定**ページのセクションには、既定の連絡先を入力し、ポータルの登録プロセスでの使用の電子メール アドレス テキスト ボックスが含まれています。  
  
-   ページ内の値を編集した後、レジストリ設定を更新するには、クリックして、**更新**ボタンをクリックします。  
  
-   前のページに戻るには、レジストリ設定を変更することがなく、 をクリックして、**キャンセル**ボタンをクリックします。
