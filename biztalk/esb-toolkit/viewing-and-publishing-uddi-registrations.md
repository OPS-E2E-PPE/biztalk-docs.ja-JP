# <a name="viewing-and-publishing-uddi-registrations"></a>表示し、UDDI 登録の発行

-   [BizTalk エンドポイントを発行する](../esb-toolkit/publishing-biztalk-endpoints.md)  
  
-   [自動登録と承認プロセス](../esb-toolkit/the-auto-enlist-and-approval-process.md)  
  
-   [BizTalk Server 管理コンソールから発行する](../esb-toolkit/publishing-from-the-biztalk-server-administration-console.md)
