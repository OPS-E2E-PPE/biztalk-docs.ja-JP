# <a name="error---property-schema-field-data-type-not-valid"></a>エラー - プロパティ スキーマ フィールドのデータ型が有効ではありません。
**エラー コード**  
  
 BEC2010  
  
 **説明**  
  
 **データ型**プロパティ**フィールド要素**プロパティ スキーマのノードは、指定されたデータ型のいずれかに設定する必要があります。 プロパティの昇格はこれらの型のデータのみに制限されているため、プロパティ スキーマで他のデータ型を使用することはできません。  
  
 **ユーザーの操作**  
  
 指定された選択**フィールド要素**ノード、し、[Microsoft[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]プロパティ] ウィンドウで変更、**データ型**プロパティに指定されたデータ型のいずれか。
