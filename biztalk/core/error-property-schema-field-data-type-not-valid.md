# <a name="error---property-schema-field-data-type-not-valid"></a>エラー - プロパティ スキーマ フィールドのデータ型が無効です。
**エラー コード**  
  
 BEC2010  
  
 **説明**  
  
 **データ型**プロパティの**フィールド要素**プロパティ スキーマのノードは、指定されたデータ型のいずれかに設定する必要があります。 プロパティの昇格は、これらのデータ型のみに制限されているため、プロパティ スキーマで他のデータ型を使用することはできません。  
  
 **ユーザーの操作**  
  
 選択**フィールド要素**ノードと、microsoft[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]プロパティ ウィンドウで、変更、**データ型**プロパティに指定されたデータ型のいずれか。
