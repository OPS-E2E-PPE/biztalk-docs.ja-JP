# <a name="document-spec-type-not-found"></a>ドキュメント仕様の種類が見つかりません。
## <a name="details"></a>詳細  
  
|                 |                                                                                        |
|-----------------|----------------------------------------------------------------------------------------|
|  製品名   |   [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]   |
| 製品バージョン |               [!INCLUDE[btsEDIVersion](../includes/btsediversion-md.md)]               |
|    イベント ID     |                                           -                                            |
|  イベント ソース   | [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] EDI |
|    コンポーネント    |                                       EDI エンジン                                       |
|  シンボル名  |                                           -                                            |
|  メッセージ テキスト   |                            ドキュメント仕様の種類が{0}が見つかりません。                            |
  
## <a name="explanation"></a>説明  
 このエラーは、スキーマが見つからないことを示します。  
  
## <a name="user-action"></a>ユーザーの操作  
 このエラーを解決するには、ドキュメント スキーマを展開します。  
  
1.  開始**Microsoft Visual Studio**します。  
  
2.  作成または既存のプロジェクトを開く  
  
3.  スキーマをプロジェクトに追加します。  
  
4.  Visual Studio でプロジェクトをデプロイします。
