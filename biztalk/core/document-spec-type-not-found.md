## <a name="explanation"></a>説明  
 このエラーは、スキーマが見つからないことを示します。  
  
## <a name="user-action"></a>ユーザーの操作  
 このエラーを解決するには、次のようにしてドキュメント スキーマを展開します。  
  
1.  開始**Microsoft Visual Studio**です。  
  
2.  プロジェクトを作成するか、既存のプロジェクトを開きます。  
  
3.  プロジェクトにスキーマを追加します。  
  
4.  Visual Studio でプロジェクトを展開します。
