# <a name="project-designer-application-tab"></a>プロジェクト デザイナー: [アプリケーション] タブ
アプリケーション タブを使用すると、アセンブリ情報を指定できます。  
  
|プロパティ|目的|  
|--------------|----------------|  
|**アセンブリ名**|アプリケーションのアセンブリ名を更新します。|  
|**既定の Namespace**|アプリケーションの既定の名前空間を変更します。|  
  
> [!CAUTION]
>  値を変更しない、**ターゲット フレームワーク**フィールドです。 このプロパティの値を変更すると、予期しない動作が生じる場合があります。  
  
## <a name="assembly-information-dialog-box"></a>[アセンブリ情報] ダイアログ ボックス  
 クリックすると、アセンブリ情報 ダイアログ ボックスが表示される**アセンブリ情報**ボタンをクリックして、**アプリケーション**ページ。  
  
|プロパティ|目的|  
|--------------|----------------|  
|**[タイトル]**|アプリケーションのアセンブリ名を更新します。|  
|**Description**|アセンブリの説明を入力します。|  
|**Company**|企業名を入力します。|  
|**Product**|製品情報を入力します。|  
|**著作権**|著作権情報を入力します。|  
|**商標**|商標情報を入力します。|  
|**アセンブリのバージョン**|major.minor.build.revision の形式でバージョン番号を入力します。|  
|**ファイル バージョン**|major.minor.build.revision の形式でファイル バージョン番号を入力します。|  
|**GUID**|プロジェクトを作成すると、[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] でアセンブリを識別する一意識別子 (GUID) が生成されます。|  
|**ニュートラル言語**|アセンブリがサポートするカルチャを指定します。|  
|**アセンブリを COM 参照可能**|チェック ボックスをオンにして、アセンブリ内の型が COM にアクセスできるかどうかを指定します。|
