# <a name="single-sign-on-event-10831"></a>シングル サインオン: イベント 10831
## <a name="details"></a>詳細  
  
|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  製品名   |                 エンタープライズ シングル サインオン                  |
| 製品バージョン | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    イベント ID     |                           10831                            |
|  イベント ソース   |                           ENTSSO                           |
|    コンポーネント    |                            なし                             |
|  シンボル名  |                ENTSSO_E_SSO_DATABASE_EXISTS                |
|  メッセージ テキスト   |         指定された SSO データベースは既に存在しています。         |
  
## <a name="explanation"></a>説明  
 構成中に、既に存在しているデータベースを作成しようとしました。  
  
## <a name="user-action"></a>ユーザーの操作  
 システムを確認するか、新しいデータベースに別の名前を付けます。
