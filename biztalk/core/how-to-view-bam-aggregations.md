# <a name="how-to-view-bam-aggregations"></a>BAM 集計を表示する方法
BAM 集計は、ビジネス アナリストが Excel で Excel アドインを使用して定義します。  
  
### <a name="to-view-your-bam-aggregations"></a>BAM 集計を表示するには  
  
1. 実行するコンピューターで[!INCLUDE[btsPlatformsComApis](../includes/btsplatformscomapis-md.md)]; をクリックして**開始**、 をポイント**すべてのプログラム**、右クリックして**Internet Explorer**、順にクリックします**として実行管理者**します。 **ユーザー アカウント制御**ダイアログ ボックスで、をクリックして**続行**します。 Internet Explorer アドレス バーに「`http://<server>/BAM/`ここで、 *\<サーバー >* BAM ポータルを実行しているコンピューター名前を指定します。  
  
    -- または --  
  
    クリックして**開始**、 をクリックして**すべてのプログラム**、 をポイント[!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)]、順にクリックします**BAM ポータル Web サイト**します。  
  
2. **マイ**ナビゲーション ウィンドウで、タスク一覧を表示するビューを展開するビューをクリックします。  
  
3. をクリックして、**集計**タスクの一覧を展開するには、集計が定義されています。  
  
4. BAM ポータルのコンテンツ フレームに集計をロードするには、集計をクリックします。  
  
> [!NOTE]
>  ビューが表示されない場合は、ビューが作成されていないか、ユーザーにアクセス許可が与えられていないかのどちらかです。ビューの作成は通常、ビジネス アナリストが実行するタスクで、権限の許可は通常、管理者が実行するタスクです。
