# <a name="single-sign-on-event-10769"></a>シングル サインオン: イベント 10769
## <a name="details"></a>詳細  
  
|||  
|-|-|  
|製品名|エンタープライズ シングル サインオン|  
|製品バージョン|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|イベント ID|10769|  
|イベント ソース|ENTSSO|  
|コンポーネント|なし|  
|シンボル名|ENTSSO_E_APP_INCOMPLETE_FIELDS|  
|メッセージ テキスト|このアプリケーション用のフィールドが未入力であるため、アプリケーションを有効にできません。|  
  
## <a name="explanation"></a>説明  
 このアプリケーション用のフィールドが未入力であるため、アプリケーションを有効にできません。  
  
## <a name="user-action"></a>ユーザーの操作  
 特定のアプリケーションのフィールドの数を指定したら、アプリケーションを有効にする前にフィールドを作成し設定する必要があります。
