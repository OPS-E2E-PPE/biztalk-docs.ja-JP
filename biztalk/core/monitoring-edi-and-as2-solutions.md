# <a name="monitoring-edi-and-as2-solutions"></a>EDI および AS2 ソリューションの監視
EDI および AS2 状態レポート UI には、受信および送信の EDI および AS2 メッセージの状態を表示することができます。 状態レポートを表示する、**グループの概要**ウィンドウでアクセスできるは、BizTalk グループ ノードをクリックすると、EDI、AS2 およびバッチ処理の状態レポートへのリンクは下部にある、**グループ概要**ウィンドウ。
