# <a name="single-sign-on-event-10656"></a>シングル サインオン: イベント 10656
## <a name="details"></a>詳細  

|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  製品名   |                 エンタープライズ シングル サインオン                  |
| 製品バージョン | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    イベント ID     |                           10656                            |
|  イベント ソース   |                           ENTSSO                           |
|    コンポーネント    |                            N\A                             |
|  シンボル名  |          SSO_INFO_PASSWORD_SYNC_ADAPTERS_STARTING          |
|  メッセージ テキスト   |     外部アダプターのパスワード同期を開始しています。%r     |
