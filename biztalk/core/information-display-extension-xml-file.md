# <a name="information---display-extension-xml-file"></a>情報 - 拡張 XML ファイルの表示
**エラー コード**  
  
 btm1057  
  
 **説明**  
  
 拡張オブジェクトに関連付けられている XML が、対象のファイルに格納されています。  
  
 **ユーザーの操作**  
  
 対象のリンクを使用して、生成された拡張 XML にアクセスできます。この XML により、アセンブリ名マップにプレフィックスが提供されます。
