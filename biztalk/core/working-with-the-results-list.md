# <a name="working-with-the-results-list"></a>結果一覧の操作
グループ ハブ ページで、クエリを実行するときに生成、**クエリ結果**、関連するメッセージ、パイプライン、およびアセンブリなどの検索結果を含むリスト。  
  
> [!NOTE]
>  結果一覧を並べ替えるには、任意の列見出しをクリックします。 クリックするたびに昇順または降順のアルファベットまたは数字の表示列の値の間で切り替えることが。
