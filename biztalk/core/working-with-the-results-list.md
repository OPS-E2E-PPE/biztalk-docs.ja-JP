# <a name="working-with-the-results-list"></a>結果一覧を使用した作業
グループ ハブ ページでクエリを実行するときに生成、**クエリの結果**関連するメッセージ、パイプライン、およびアセンブリなどの検索結果を含むリスト。  
  
> [!NOTE]
>  結果一覧を並べ替えるには、任意の列見出しをクリックします。 クリックするたびに、列の値のアルファベット表示または数値表示の昇順/降順が切り替わります。
