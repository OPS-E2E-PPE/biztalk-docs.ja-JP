# <a name="success---create-xml-instance"></a>成功 - XML インスタンスの作成
**説明**  
  
 スキーマのインスタンスが正常に生成され、XML 出力ファイルが作成されました。作成された XML 出力ファイルには、インスタンス メッセージが含まれています。  
  
 **ユーザーの操作**  
  
 適宜、生成された XML 出力のインスタンス メッセージ ファイルを調査します。
