# <a name="keyboard-shortcuts-general-to-the-design-surface"></a>キーボード ショートカットをデザイン画面に [全般]
次の表では、デザイン画面の領域の処理領域およびポート画面の両方に共通するキーボード ナビゲーションについて説明します。  
  
|キー|効果|  
|---------|------------|  
|Tab|フォーカスと選択対象は、右側 (処理領域またはポート画面) で [次へ] のデザイン画面の領域を非アクティブな選択した図形に移動します。<br /><br /> 例:フォーカスと選択対象が、左ポート画面でのポートでは現在です。 TAB キーが押されたときに、処理領域内で非アクティブなフォーカスがあった図形にフォーカスと選択が移動します。|  
|SHIFT + TAB|同じ タブで、ですが、逆の方向。|  
|PageDown|ページを下にスクロールして、オーケストレーションの後の部分が表示されるようにします。 フォーカスと選択は変更されません。|  
|PageUp|キーを押しながら、ページと同じですが、反対方向にします。 フォーカスと選択は変更されません。|  
|SHIFT + PAGEDOWN|セットずつ左に全体のデザイン画面を水平方向にスクロールします。 フォーカスと選択は変更されません。|  
|SHIFT キーを押しながら PAGEUP|右にデザイン サーフェイスのスクロールをことを除いてと shift キーを押しながらページと同じダウンします。 フォーカスと選択は変更されません。|  
|Enter|通常、プロパティ グリッドにフォーカスを移すことを意味する選択範囲の既定のアクションを実行します。|  
|SPACE|現在、フォーカスがある図形を選択します。 図形コネクタにフォーカスがある場合、効果はありません。|  
|CTRL + SHIFT + F1|展開と選択した図形のスマート タグの折りたたみを切り替えます。 矢印と下向き矢印ことができますし、使用のスマート タグの項目に移動します。 展開すると、Enter または Space キーは、スマート タグ項目 (だけでなく、スマート タグを折りたたむ) の関連するアクションを実行します。|  
|Del|現在の選択範囲を削除します。|  
|CTRL キーを押しながらマウス ホイール|デザイン画面の表示を拡大表示が表示されます。|  
|CTRL キーを押しながらマウス ホイール|縮小ビューをデザイン画面が表示されます。|
