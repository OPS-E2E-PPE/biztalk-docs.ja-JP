# <a name="keyboard-shortcuts-general-to-the-design-surface"></a>キーボード ショートカットをデザイン画面に [全般]
デザイン画面の処理領域およびポート画面領域の両方に共通のキーボード操作を次の表に示します。  
  
|[キー]|結果|  
|---------|------------|  
|Tab|右方向にある次のデザイン画面領域 (処理領域またはポート画面) の、選択が非アクティブな図形にフォーカスと選択対象を移動します。<br /><br /> 例: フォーカスと選択対象は現在、左ポート画面のポートのです。 Tab キーを押すと、処理領域内でフォーカスが非アクティブな図形に、フォーカスと選択対象が移動します。|  
|Shift+Tab|Tab キーと同じですが、方向が逆になります。|  
|PageDown|ページを下方向にスクロールし、オーケストレーションの後の部分を表示します。 選択対象とフォーカスは変わりません。|  
|PageUp|PageDown キーと同じですが、スクロールする方向が逆になります。 選択対象とフォーカスは変わりません。|  
|SHIFT キーを押しながら PAGEDOWN|設定された増分値に応じて、デザイン画面全体を左方向に水平スクロールします。 選択対象とフォーカスは変わりません。|  
|SHIFT キーを押しながら PAGEUP|Shift+PageDown と同じですが、デザイン画面は右にスクロールします。 選択対象とフォーカスは変わりません。|  
|Enter|選択項目に対して既定の操作を実行します。通常は、プロパティ グリッドへのフォーカス移動です。|  
|Space|現在フォーカスがある図形を選択します。 図形コネクタにフォーカスがある場合は何も起こりません。|  
|Ctrl+Shift+F1|選択されている図形のスマート タグの展開と折りたたみを切り替えます。 ↑ キーと↓ キーを使用すると、スマート タグのアイテム間を移動できます。 展開されている状態で Enter キーまたは Space キーを押すと、スマート タグのアイテムに関連付けられている処理が実行され、そのスマート タグが折りたたまれます。|  
|DELETE|現在の選択項目を削除します。|  
|Ctrl+マウス ホイールを上へ</localizedText>|デザイン画面の拡大ビューを表示します。|  
|Ctrl+マウス ホイールを下へ</localizedText>|デザイン画面の縮小ビューを表示します。|
