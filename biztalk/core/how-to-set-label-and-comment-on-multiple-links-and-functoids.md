# <a name="how-to-set-label-and-comment-on-multiple-links-and-functoids"></a>複数のリンクおよび Functoid にラベルおよびコメントを設定する方法
複数の Functoid やリンクに共通のラベルやコメントを設定できます。 このトピックでは、次の操作を実行する方法の詳細について説明します。  
  
 複数の functoid およびリンクを選択する方法については、次を参照してください。[および Functoid を複数のリンクを選択する方法](../core/how-to-select-multiple-links-and-functoids.md)です。  
  
 複数の Functoid およびリンクを選択した後は、以下のことを行うことができます。  
  
-   選択したリンクに共通のラベルを設定します。  
  
-   選択した Functoid に共通のラベルやコメントを設定します。  
  
-   選択した Functoid とリンクに共通のラベルを設定します。  
  
## <a name="prerequisites"></a>前提条件  
 これらの手順では、BizTalk マッパーが実行されている必要があります。  
  
## <a name="to-set-a-common-label-for-selected-links"></a>選択したリンクに共通のラベルを設定するには  
 複数のリンクに対する共通ラベルの設定は、1 つのアトミック操作です。 この操作は元に戻したり、やり直したりできます。  
  
 選択したリンクに共通のラベルを設定するには:  
  
1.  マッパー グリッド ページで、対象のリンクを選択します。  
  
2.  **プロパティ** ウィンドウで適切なラベルを入力、**ラベル**プロパティです。  
  
## <a name="to-set-a-common-label-andor-comment-for-selected-functoids"></a>選択した Functoid に共通のラベルやコメントを設定するには  
 選択した Functoid に対する共通のラベルの設定および共通のコメントの設定は、2 つの異なるアトミック操作です。 これらの操作は元に戻したり、やり直したりできます。  
  
 選択した Functoid に共通のラベルやコメントを設定するには:  
  
1.  マッパー グリッド ページで、対象の Functoid を選択します。  
  
2.  **プロパティ**ウィンドウで、入力に適切なラベルやコメントを**ラベル**と**コメント**プロパティです。  
  
## <a name="to-set-a-common-label-for-selected-functoids-and-links"></a>選択した Functoid およびリンクに共通のラベルを設定するには  
 選択した Functoid とリンクに共通のラベルを設定できます。 **ラベル**のみプロパティは、functoid およびリンクの両方に共通です。 この操作は元に戻したり、やり直したりできます。  
  
 選択した Functoid およびリンクに共通のラベルを設定するには:   
  
1.  マッパー グリッド ページで、対象の Functoid およびリンクを選択します。  
  
2.  **プロパティ** ウィンドウで適切なラベルを入力、**ラベル**プロパティです。
