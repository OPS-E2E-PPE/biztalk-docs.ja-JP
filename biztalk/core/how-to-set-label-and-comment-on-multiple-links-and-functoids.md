# <a name="how-to-set-label-and-comment-on-multiple-links-and-functoids"></a>複数のリンクおよび Functoid にラベルおよびコメントを設定する方法
共通のラベルや複数の functoid やリンクのコメントを設定できます。 このトピックでは、この操作を実行する方法について詳細を提供します。  
  
 複数の functoid およびリンクを選択する方法については、次を参照してください。[および Functoid を複数のリンクを選択する方法](../core/how-to-select-multiple-links-and-functoids.md)します。  
  
 複数の functoid およびリンクを選択した後、次のいずれかの操作を行います。  
  
-   選択したリンクに共通のラベルを設定します。  
  
-   共通のラベルや選択した functoid のコメントを設定します。  
  
-   選択した functoid およびリンクに共通のラベルを設定します。  
  
## <a name="prerequisites"></a>前提条件  
 これらの手順では、BizTalk マッパーが実行されている必要があります。  
  
## <a name="to-set-a-common-label-for-selected-links"></a>選択したリンクに共通のラベルを設定するには  
 1 つのアトミック操作は、複数のリンクに共通のラベルを設定します。 元に戻すまたはこの操作をやり直すことができます。  
  
 選択したリンクに共通のラベルを設定するには。  
  
1.  マッパー グリッド ページで、対象のリンクを選択します。  
  
2.  **プロパティ** ウィンドウで適切なラベルを入力、**ラベル**プロパティ。  
  
## <a name="to-set-a-common-label-andor-comment-for-selected-functoids"></a>共通のラベルや選択した functoid のコメントを設定するには  
 共通のラベルを設定し、選択した functoid の一般的なコメントを設定は、2 つの異なるアトミック操作です。 元に戻すまたはこれらの操作をやり直すことができます。  
  
 選択した functoid に共通のラベルやコメントを設定します。  
  
1.  マッパー グリッド ページで、対象の functoid を選択します。  
  
2.  **プロパティ**ウィンドウで、入力に適切なラベルやコメントを**ラベル**と**コメント**プロパティ。  
  
## <a name="to-set-a-common-label-for-selected-functoids-and-links"></a>選択した functoid およびリンクに共通のラベルを設定するには  
 選択した functoid およびリンクに共通のラベルを設定することができます。 **ラベル**functoid とリンクの両方に共通である専用のプロパティします。 元に戻すまたはこの操作をやり直すことができます。  
  
 選択した functoid およびリンクに共通のラベルを設定するには。  
  
1.  マッパー グリッド ページで、目的の functoid およびリンクを選択します。  
  
2.  **プロパティ** ウィンドウで適切なラベルを入力、**ラベル**プロパティ。
