# <a name="xsd-resources-on-the-web"></a>Web 上の XSD リソース
XML スキーマ定義 (XSD) の詳細を提供する言語の概要、XSD やはこのドキュメントの範囲外です。 詳細については、XSD 仕様と World Wide Web Consortium (W3C) の Web サイトに保持される入門書に直接移動します。 次のチュートリアルは推奨具体的には、最初の基礎と、後者を XSD に関する情報源として 2 つについての簡単な方法として。 XSD のチュートリアルでは、次の W3C ページを参照してください。  
  
- XML Schema Part 0:入門、 [ http://go.microsoft.com/fwlink/?LinkId=16787](http://go.microsoft.com/fwlink/?LinkId=16787)します。  
  
- XML Schema Part 1:構造体、 [ http://go.microsoft.com/fwlink/?LinkId=16788](http://go.microsoft.com/fwlink/?LinkId=16788)します。  
  
- XML Schema Part 2:データ型、 [ http://go.microsoft.com/fwlink/?LinkId=16789](http://go.microsoft.com/fwlink/?LinkId=16789)します。  
  
  BizTalk エディターを使用して、複雑なスキーマを作成する前に XSD の理解があることをお勧めします。
