# <a name="how-to-set-receive-pipelines-for-tibco-rendezvous"></a>TIBCO Rendezvous 用の受信パイプラインを設定する方法
Microsoft BizTalk Adapter for TIBCO Rendezvous では、受信ハンドラーと受信パイプラインを設定する必要があります。  
  
### <a name="to-set-the-pipeline"></a>パイプラインを設定するには  
  
1.  設定、**受信ハンドラー**に**BizTalkServerIsolatedHost**一覧にします。  
  
2.  設定、**受信パイプライン**に**XMLReceive**または同等のパイプライン。  
  
3.  **[OK]**をクリックします。
