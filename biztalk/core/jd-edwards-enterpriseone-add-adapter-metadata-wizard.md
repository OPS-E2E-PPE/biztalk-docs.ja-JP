# <a name="jd-edwards-enterpriseone-add-adapter-metadata-wizard"></a>JD Edwards EnterpriseOne アダプター メタデータの追加ウィザード
JD Edwards EnterpriseOne システムからメタデータ情報にアクセスするアダプターの追加ウィザードを使用します。  
  
|プロパティ|目的|  
|--------------|----------------|  
|アダプターの一覧を選択します。|一覧からアダプターを選択します。|  
|SQL Server|一覧から SQL Server を選択し、をクリックして**次**します。|  
|[データベース]|SQL server と同じデータベースに接続します。|  
|Port|選択、構成された送信ポートとクリックして**次**します。|
