# <a name="parties"></a>パーティ
パーティ、または組織単位、オーケストレーションと連携する BizTalk Server 外部のエンティティを表します。 BizTalk Server では、メッセージを交換する各組織は、パーティによって表されます。 パーティがロールに参加させてと連携する方法を定義できます。
