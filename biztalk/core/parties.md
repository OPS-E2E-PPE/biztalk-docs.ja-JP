# <a name="parties"></a>パーティ
パーティ (つまり、組織単位) は、オーケストレーションと連携する BizTalk Server 外部のエンティティを表します。 BizTalk Server では、メッセージ交換の対象となる各組織は、パーティで表されます。 パーティをロールに参加させてパーティと連携する方法を定義できます。
