# <a name="mappings-enlisted-party-node"></a>マッピング (Enlisted Party ノード)
バインド ファイルの Mappings ノードは、すべてのパーティ ポートとロール ポートの種類の操作の間のマッピングを記述するマッピング ノードの親ノードです。  
  
## <a name="nodes-in-the-mappings-node"></a>Mappings ノード内のノード  
 次の表に、バインド ファイルのこのノードに設定できるプロパティを示します。  
  
|**名前**|**ノードの種類**|**[データ型]**|**[説明]**|**制限**|**コメント**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[マッピング](../core/mapping-mappings-node.md)|レコード|EnlistedPartyMapping (ComplexType)|パーティ ポートとロール ポートの種類の操作の間のマッピングを指定します。|任意|既定値: なし|
