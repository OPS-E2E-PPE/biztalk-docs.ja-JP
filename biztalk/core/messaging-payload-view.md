# <a name="messaging-payload-view"></a>メッセージング ペイロード ビュー
メッセージング ペイロード ビューには、選択したアクション (たとえば、メッセージを送信または受信) に関連付けられている XML メッセージのスキーマが表示されます。 ビューは、一部の図形 オーケストレーション スケジュール ビューのショートカット メニューから利用できます。  
  
## <a name="working-with-the-messaging-payload-view"></a>メッセージング ペイロード ビューの操作  
 イベント ソースの選択 ボタンをクリックし、メッセージング ペイロード スキーマの選択 メニュー項目をクリックして、オーケストレーション スケジュール ビューを選択します。 スキーマを選択するとする要素をドラッグできますこのスキーマからアクティビティのデータ項目フォルダーにこのアクションのメッセージ内の特定の XPath 式からそのデータを抽出することを示します。  
  
 オーケストレーション スケジュール ビューからメッセージング ペイロード ビューを開くには、メッセージ ペイロードを含む図形を右クリックします。 これにより、メッセージング プロパティ スキーマを取得するメッセージング ペイロードのメニュー項目をクリックすることができます、コンテキスト メニューが開きます。  
  
 XPath 式の詳細については、次を参照してください。[メッセージ割り当てにおける Xpath の使用](../core/using-xpaths-in-message-assignments.md)します。
