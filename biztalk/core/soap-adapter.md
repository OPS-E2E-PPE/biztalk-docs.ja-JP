# <a name="soap-adapter"></a>SOAP アダプター
Microsoft BizTalk Server Web サービスは、Web サービス要求の受信と送信時に、SOAP アダプタを使用します。 SOAP アダプターを使用すると、オーケストレーションを Web サービスとして公開し、外部の Web サービスを使用できます。
