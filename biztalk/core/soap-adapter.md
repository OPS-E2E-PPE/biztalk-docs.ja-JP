# <a name="soap-adapter"></a>SOAP アダプター
Microsoft BizTalk Server Web サービスでは、Web サービス要求の送受信時に SOAP アダプタを使用します。 SOAP アダプタを使用すると、オーケストレーションを Web サービスとして公開し、外部 Web サービスを利用できます。
