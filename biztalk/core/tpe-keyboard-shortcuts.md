# <a name="tpe-keyboard-shortcuts"></a>TPE のショートカット キー
追跡プロファイル エディター (TPE) では、マウスの他に、ショートカット キーを使用してナビゲーションや編集を行うことができます。 TPE のすべての機能は、ショートカット キーで実行できます。 ここでは、TPE のショートカット キーについて説明し、キーボードを使用して TPE 内を移動する際のヒントをいくつか紹介します。  
  
## <a name="tpe-tabs"></a>TPE のタブ  
 TPE 内で新しいプロファイルを作成するとき、または既存のプロファイルを開いたとき、最初は、イベント ソース ビューの右側のペインが選択されています。  
  
 アクティビティ ビューが選択されている状態で Tab キーを押すと、次は右側のペインにあるイベント ソース ビューが選択されます。 イベント ソース ビューでは、↑キーと↓キーを使用して、フレーム内を移動したり特定のオーケストレーション図形やスキーマ アイテムを選択できます。  
  
## <a name="tpe-keyboard-shortcuts"></a>TPE のショートカット キー  
 TPE では、マウスの代わりにショートカット キーを使用して操作できます。 次の表に、TPE で使用できるショートカット キーを示します。  
  
|操作|キーボード ショートカット|  
|------------|-----------------------|  
|[新しいプロファイルを作成する]|Alt + F + N|  
|既存の追跡プロファイルを開く|Alt + F + O|  
|追跡プロファイルの保存|Alt + F + S|  
|[名前を付けて保存]|Alt + F + A|  
|アクティビティ定義のインポート|Alt + F + I|  
|ショートカット メニュー項目の削除|Del|  
|ショートカット メニュー項目名の変更|F2|  
|追跡プロファイル エディターのヘルプを開く|F1|  
|追跡プロファイルの適用|Alt + T + A|  
|追跡プロファイルの削除|Alt + T + R|  
|管理データベースの設定|Alt + T + M|  
|メッセージ ペイロード スキーマ|Shift + F10 + M|  
|TPE を終了して閉じる|Alt + F + X|  
|オーケストレーション スケジュールの選択|Alt + S + O|  
|メッセージング ペイロードの選択|Alt + S + M|  
|コンテキスト プロパティの選択|Alt + S + C|  
|メッセージング プロパティの選択|Alt + S + P|  
|コンテキスト プロパティ スキーマ|Shift + F10 + M|  
|メッセージ プロパティ スキーマ|Shift + F10 + P|
