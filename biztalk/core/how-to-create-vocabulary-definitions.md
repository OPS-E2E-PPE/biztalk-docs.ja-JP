# <a name="how-to-create-vocabulary-definitions"></a>ボキャブラリの定義を作成する方法
ボキャブラリの定義ウィザードを使用して、ボキャブラリの定義を作成できます。 ボキャブラリの定義は、定数値、値の範囲、値セット、または .NET アセンブリ、XML ドキュメント、データベース テーブルの要素として定義できます。 パブリック変数を選択する場合があります**取得**と**設定**オプションと同様に、データベースおよび XML の定義ウィザード。  
  
 3 つのタブのいずれかのファクトを選択して、新しいボキャブラリの定義を作成する代わりに、— たとえば、データベース列、XML ノード、または .NET クラスのメンバー: 上に、ファクトをドラッグすること、**ボキャブラリ** タブとボキャブラリの未公開バージョンにドロップします。  
  
> [!NOTE]
>  公開済みのボキャブラリのバージョンにはボキャブラリの定義を追加できません。  
  
### <a name="to-define-a-vocabulary-definition-as-a-constant-value"></a>ボキャブラリの定義を定数値として定義するには  
  
1.  ボキャブラリのバージョンを右クリックし、をクリックして**新しい定義の追加**です。  
  
    > [!NOTE]
    >  ドラッグ アンド ファクト エクスプ ローラーの他のタブの項目をドロップすることができますも: XML スキーマ、データベースおよび .NET クラス  
  
2.  ボキャブラリの定義ウィザードで選択**定数値、値の範囲、または値セット**、順にクリック**次**です。  
  
3.  定義名と定義の説明を編集します。  
  
    > [!NOTE]
    >  定義の表示名の長さは最大 512 文字です。  
  
4.  選択**定数値**、順にクリック**次**です。  
  
5.  利用可能なシステム型のドロップダウン リストから、定義の種類を選択します。  
  
6.  表示名と値を編集し、クリックして**完了**です。  
  
### <a name="to-define-a-vocabulary-definition-as-a-range-of-values"></a>ボキャブラリの定義を値の範囲として定義するには  
  
1.  ボキャブラリのバージョンを右クリックし、をクリックして**新しい定義の追加**です。  
  
2.  ボキャブラリの定義ウィザードで選択**定数値、値の範囲、または値セット**、順にクリック**次**です。  
  
3.  定義名と定義の説明を編集します。  
  
4.  選択**値の範囲**、順にクリック**次**です。  
  
5.  ドロップダウン リストから定義の種類を選択します。  
  
6.  をクリックして**範囲の下限**、順にクリック**編集**範囲の下限の値を指定します。 パラメーター定義ダイアログが表示されます。  
  
7.  選択**定数値の使用**定数の値を入力または選択**公開済みのボキャブラリから定義を使用**ボキャブラリの定義を参照し、をクリックし、 **[ok]**です。  
  
8.  手順 6 および 7 を繰り返します**範囲の上限**です。  
  
    > [!NOTE]
    >  **範囲の上限**値を超える必要があります、**範囲の下限**値。  
  
9. 表示形式文字列を入力するか、をクリックして**既定**をクリックして既定の表示書式文字列に戻す**完了**です。  
  
    > [!NOTE]
    >  書式文字列は中かっこでパラメーターのインデックスを含める必要があります: たとえば、"{0}"と"{1}"—to は高値と安値の範囲のパラメーターのプレース ホルダーとして機能します。

### <a name="to-define-a-vocabulary-definition-as-a-set-of-values"></a>ボキャブラリの定義を値セットとして定義するには  
  
1.  ボキャブラリのバージョンを右クリックし、をクリックして**新しい定義の追加**です。  
  
2.  ボキャブラリの定義ウィザードで選択**定数値、値の範囲、または値セット**、順にクリック**次**です。  
  
3.  定義名と定義の説明を編集します。  
  
4.  選択**値の設定**、順にクリック**次**です。  
  
5.  定義の種類と表示名を編集します。  
  
6.  セットにメンバーを追加するには、選択**定数値の使用**定数の値を入力または選択**公開済みのボキャブラリから定義を使用**ボキャブラリの定義を参照し、 をクリックし、**追加**です。  
  
7.  追加するアイテム数の分だけ、定数やボキャブラリの定義の組み合わせを使用して、手順 6. を繰り返します。  
  
8.  セットの相対順序内のメンバーを移動するには、選択してをクリックして**を**または**ダウン**です。  
  
9. セットからメンバーを削除するには、選択してをクリックして**削除**です。  
  
10. セットが完了したら、をクリックして**完了**です。

### <a name="to-define-a-vocabulary-definition-as-a-net-class-or-class-member"></a>ボキャブラリの定義を .NET クラスまたはクラス メンバーとして定義するには  
  
1.  ボキャブラリのバージョンを右クリックし、をクリックして**新しい定義の追加**です。  
  
2.  ボキャブラリの定義ウィザードで選択**.NET クラスまたはクラス メンバー**、クリックして**次**です。  
  
3.  編集、**定義名**と**説明**フィールドです。  
  
4.  **[参照]**をクリックします。  
  
5.  **.NET アセンブリ**ダイアログ ボックスでは、アセンブリを選択し、をクリックして**OK**です。

    >  アセンブリは、グローバル アセンブリ キャッシュ (GAC) 内に格納されている必要があります。 ビジネス ルール作成ツールは、.NET アセンブリで参照するときに .NET アセンブリを読み込みます、**ファクト エクスプ ローラー**ウィンドウまたは、 **.NET クラスまたはクラス メンバーの定義**のページ、**ボキャブラリ定義**ウィンドウです。  GAC でアセンブリを更新した場合は、ビジネス ルール作成ツールを終了してから再起動し、更新した .NET アセンブリを読み込みます。 ビジネス ルール作成ツールでは、アセンブリが自動更新されません。  
  
6.  アセンブリ ノードを展開します。  
  
7.  クラス、またはクラスを展開およびクラスのメンバーを選択し、をクリックして**OK**です。  
  
8.  をクリックして**次**、し、表示名を指定します。  
  
     詳細については、このトピックの後半にある「パラメーターを使用して、ボキャブラリの定義の表示書式を指定するには」を参照してください。  
  
9. **[完了]**をクリックします。

2.  ボキャブラリの定義ウィザードで選択**XML ドキュメントの要素または属性**、クリックして**次**です。  
  
3.  定義名と定義の説明を入力します。  
  
4.  をクリックして**参照**をスキーマ ファイルを検索およびドキュメントの要素または属性を指定します。  
  
5.  ドロップダウン リストから、スキーマ内の要素または属性の型と互換性のある型を選択します。  
  
    > [!NOTE]
    >  特定の型への有効なキャスト、または特定の型からドキュメントの要素または属性の型への有効なキャストを行うことができなかった場合、実行時にエラーが生成されます。  
  
6.  要素または属性の値を取得するかどうかを示すために、またはその値を設定するために、操作の種類を選択します。  
  
7.  値を設定する場合は、クリックして**次**、し、表示形式を指定します。  
  
8.  **[完了]**をクリックします。

2.  ボキャブラリの定義ウィザードで選択**データベース テーブルまたはデータベース テーブル列の定義**、順にクリック**次**です。  
  
3.  定義名と定義の説明を入力します。  
  
4.  **[参照]**をクリックします。  
  
5.  接続する SQL Server コンピューターを選択します。 SQL Server コンピューターが現在開始されていない場合は、選択、**停止されている場合、SQL Server 開始**チェック ボックスをオンします。  
  
6.  認証の種類を選択します。 SQL Server 認証を選択した場合のログオン名とパスワードを入力し、をクリックして**OK**です。  
  
7.  テーブルまたはテーブルの列にバインドし、そのをクリックする選択**OK**です。  
  
8.  テーブル列を選択した場合は、値を取得するか、または値を設定するかを選択します。 値の取得を選択した場合は、表示書式を入力します。 その値を設定する場合をクリックして**次**表示形式を指定します。  
  
     詳細については、このトピックの後半にある「パラメーターを使用して、ボキャブラリの定義の表示書式を指定するには」を参照してください。

10. **[完了]**をクリックします。
