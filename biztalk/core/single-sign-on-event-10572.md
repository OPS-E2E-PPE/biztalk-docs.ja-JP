# <a name="single-sign-on-event-10572"></a>シングル サインオン: イベント 10572
## <a name="details"></a>詳細  
  
|                 |                                                                                       |
|-----------------|---------------------------------------------------------------------------------------|
|  製品名   |                               エンタープライズ シングル サインオン                               |
| 製品バージョン |              [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]               |
|    イベント ID     |                                         10572                                         |
|  イベント ソース   |                                        ENTSSO                                         |
|    コンポーネント    |                                          なし                                          |
|  シンボル名  |                    SSO_WARN_SSO_DISABLED_FOR_CHANGE_SSO_AFF_ADMIN                     |
|  メッセージ テキスト   | SSO 関連管理者アカウント名を更新するには、最初に SSO を無効にする必要があります。%r |
  
## <a name="explanation"></a>説明  
 SSO 関連管理者アカウント名を更新するには、最初に SSO を無効にする必要があります。  
  
## <a name="user-action"></a>ユーザーの操作  
 SSO を無効にする方法の詳細については、[SSO を無効にする方法](../core/how-to-disable-sso.md)を参照してください。
