# <a name="how-to-rename-nodes"></a>ノードの名前を変更する方法
既存のノードの名前を変更したい場合があります。 このトピックでは、そのための 2 とおりの方法を紹介します。  
  
### <a name="to-rename-a-single-node"></a>単一ノードの名前を変更するには  
  
1.  選択、**レコード**ノード、**フィールド要素**ノード、または**フィールド属性**ノードの名前を変更します。  
  
2.  **BizTalk**  メニューのをクリックして**の名前を変更**です。  
  
3.  新しいノード名を入力し、Enter キーを押します。  
  
    > [!NOTE]
    >  値を変更することによって、ノードを変更することもその**ノード名**プロパティです。  
  
### <a name="to-rename-multiple-nodes-to-have-the-same-name"></a>複数のノードに同じ名前を付けるには  
  
1.  同じ名前にする一連のノードの最初のノードを選択します。  
  
2.  <localizedText>Ctrl</localizedText> キーを押しながら、同じ名前にする他のノードを選択します。  
  
3.  [プロパティ] ウィンドウでの値を変更、**ノード名**プロパティ、および ENTER キーを押して終了します。  
  
     スキーマ ツリー ビューで、選択されたすべてのノードの名前が更新されます。
