# <a name="how-to-rename-nodes"></a>ノードの名前を変更する方法
既存のノードの名前を変更する可能性あります。 このトピックでは、このタスクに 2 つのバリエーションをについて説明します。  
  
### <a name="to-rename-a-single-node"></a>1 つのノードの名前を変更するには  
  
1.  選択、**レコード**ノード、**フィールド要素**ノード、または**フィールド属性**名前を変更するノード。  
  
2.  **BizTalk**  メニューのをクリックして**の名前を変更**します。  
  
3.  ノードは、新しい名前を入力し、ENTER キーを押します。  
  
    > [!NOTE]
    >  値を変更することで、ノードを変更することもできます。 その**ノード名**プロパティ。  
  
### <a name="to-rename-multiple-nodes-to-have-the-same-name"></a>同じ名前を持つ複数のノードの名前を変更するには  
  
1.  同じ名前を指定するノードのセット内の最初のノードを選択します。  
  
2.  キーを押して、CTRL キーを押しし、同じ名前を指定する他のノードを選択します。  
  
3.  [プロパティ] ウィンドウでの値を変更、**ノード名**プロパティ、および ENTER キーを押して終了します。  
  
     選択したすべてのノードは、新しい名前でスキーマ ツリー ビューで更新されます。
