# <a name="how-to-configure-the-installed-web-service"></a>インストールされている Web サービスを構成する方法
Web サービス ファイルをインストールした後、その Web サービスからメッセージを受信するように BizTalk Server を構成する必要があります。  
  
### <a name="to-configure-the-web-service"></a>Web サービスを構成するには  
  
1.  BizTalk Server 管理コンソールで、 **BizTalk グループ**、展開**アプリケーション**、Web サービスを構成する場合、アプリケーションを右クリックします。  
  
2.  をポイント**インポート**、クリックして**バインド**です。  
  
3.  配布フォルダに BindingInfo.xml ファイルを選択し、クリックして**OK**です。  
  
4.  オーケストレーションを開始し、受信場所を有効にします。
