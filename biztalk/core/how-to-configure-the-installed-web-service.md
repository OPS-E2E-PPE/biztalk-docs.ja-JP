# <a name="how-to-configure-the-installed-web-service"></a>インストールされている Web サービスを構成する方法
Web サービス ファイルをインストールした後は、Web サービスからメッセージを受信する BizTalk Server を構成する必要があります。  
  
### <a name="to-configure-the-web-service"></a>Web サービスを構成するには  
  
1.  BizTalk Server 管理コンソールで  **BizTalk グループ**、展開**アプリケーション**、Web サービスを構成するアプリケーションを右クリックします。  
  
2.  をポイント**インポート**、 をクリックし、**バインド**します。  
  
3.  配布フォルダに BindingInfo.xml ファイルを選択し、クリックして**OK**します。  
  
4.  オーケストレーションを起動し、受信場所を有効にします。
