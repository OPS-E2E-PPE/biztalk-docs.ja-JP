# <a name="business-rules-biztalk-server-samples-folder"></a>ビジネス ルール (BizTalk Server Samples フォルダ)
Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] のソフトウェア開発キット (SDK) には、ビジネス ルールのサンプルがいくつか付属しています。 このセクションでは、各ビジネス ルール サンプルが示す機能、サンプルをビルドして実行する方法、および予測される実行結果について詳しく説明します。  
  
## <a name="in-this-section"></a>このセクションの内容  
  
-   [Business Rules こんにちは World1 (BizTalk Server サンプル)](../core/business-rules-hello-world1-biztalk-server-sample.md)です。 サンプルのファクト セットを使用して、BizTalk ルール セットを作成、保存、読み込み、および実行する方法を示します。  
  
-   [Business Rules こんにちは World2 (BizTalk Server サンプル)](../core/business-rules-hello-world2-biztalk-server-sample.md)です。 示しますバージョン、発行、および共有 SQL ルール ストアを使用する設定、拡張マークアップ言語 (XML) ルールの展開方法、**ポリシー**オブジェクト、およびポリシーの動的更新を実行します。  
  
-   [Loans Processing Using Business Rules (BizTalk Server サンプル)](../core/loans-processing-using-business-rules-biztalk-server-sample.md)です。 オーケストレーションで BizTalk ルール セットとファクトを使用して、インスタンス メッセージのフィールドの値を算出する方法を示します。  
  
-   [Medical Claims Processing and Testing Policies (BizTalk Server サンプル)](../core/medical-claims-processing-and-testing-policies-biztalk-server-sample.md)です。 データベース テーブルと受信ドキュメントから導き出したファクトを調べる複数ルールのルール セットを作成する方法を示します。
