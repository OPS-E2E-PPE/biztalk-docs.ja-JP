# <a name="business-rules-biztalk-server-samples-folder"></a>ビジネス ルール (BizTalk Server Samples フォルダー)
Microsoft[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]ソフトウェア開発キット (SDK) のいくつかのビジネス ルールのサンプルが含まれています。 ここでは、各ビジネス ルール サンプルを構築して、サンプルと予想される結果を実行する手順で示されている機能の詳細を示します。  
  
## <a name="in-this-section"></a>このセクションの内容  
  
-   [Business Rules こんにちは World1 (BizTalk Server サンプル)](../core/business-rules-hello-world1-biztalk-server-sample.md)します。 作成、保存、読み込み、およびサンプルのファクト セットを使用して設定、BizTalk ルールを実行する方法を示します。  
  
-   [Business Rules こんにちは World2 (BizTalk Server サンプル)](../core/business-rules-hello-world2-biztalk-server-sample.md)します。 について説明する方法のバージョンに次のように公開、および共有 SQL ルール ストアを使用する設定、拡張マークアップ言語 (XML) ルールを展開、**ポリシー**オブジェクト、およびポリシーの動的更新を実行します。  
  
-   [Loans Processing Using Business Rules (BizTalk Server サンプル)](../core/loans-processing-using-business-rules-biztalk-server-sample.md)します。 BizTalk ルール セットと、オーケストレーション内でファクトを使用して、インスタンス メッセージ内のフィールドの値を計算する方法を示します。  
  
-   [Medical Claims Processing and Testing Policies (BizTalk Server サンプル)](../core/medical-claims-processing-and-testing-policies-biztalk-server-sample.md)します。 データベース テーブルと受信ドキュメントから派生したファクトを調べる複数ルールのルール セットを作成する方法を示します。
