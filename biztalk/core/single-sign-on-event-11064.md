# <a name="single-sign-on-event-11064"></a>シングル サインオン:イベント 11064
## <a name="details"></a>詳細  
  
|                 |                                                                                                                                                 |
|-----------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
|  製品名   |                                                            エンタープライズ シングル サインオン                                                            |
| 製品バージョン |                                           [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                            |
|    イベント ID     |                                                                      11064                                                                      |
|  イベント ソース   |                                                                     ENTSSO                                                                      |
|    コンポーネント    |                                                                       なし                                                                       |
|  シンボル名  |                                                     SSO_INFO_MIIS_PASSWORD_CHANGE_RECEIVED                                                      |
|  メッセージ テキスト   | MIIS.%r から Windows パスワードの変更を受け取りました<br /><br /> 追跡 ID: %1 %r<br /><br /> Windows アカウント: % 2 %r<br /><br /> クライアント ユーザー: %3 |
  
## <a name="explanation"></a>説明  
 これは、パスワードの変更が MIIS から受信されたことを示す情報メッセージです。  
  
## <a name="user-action"></a>ユーザーの操作  
 ユーザーによる操作は不要です。
