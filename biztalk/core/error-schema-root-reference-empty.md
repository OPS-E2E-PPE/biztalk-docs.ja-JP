# <a name="error---schema-root-reference-empty"></a>エラー - スキーマ ルート参照が空です。
**エラー コード**  
  
 BEC2005  
  
 **説明**  
  
 **ルート参照**のプロパティ、**スキーマ**ノードが設定されていません。 ときに、**標準**のプロパティ、**スキーマ**ノードが以外の値に設定されて**XML**を設定する必要があります、**ルート参照**プロパティを子ノードを示す、**スキーマ**ノードはこのスキーマで定義されるメッセージのルートとして使用するためのものです。  
  
 **ユーザーの操作**  
  
 スキーマの適切な設定、**標準**のプロパティ、**スキーマ**ノードを**XML**、設定や、**ルート参照**プロパティ、**スキーマ**の適切な子ノードにノード、**スキーマ**ノード。 これらの子ノードがで使用できる、**ルート参照**プロパティ ボックスの一覧。
