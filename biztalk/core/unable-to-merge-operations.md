## <a name="explanation"></a>説明  
 このエラーは、名前の競合、または異なる通信方向のために、操作を統合できないことを示します。  
  
## <a name="user-action"></a>ユーザーの操作  
 名前の競合や同じ通信方式がないオーケストレーションを公開します。
