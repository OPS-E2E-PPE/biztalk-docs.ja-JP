# <a name="error---cannot-resolve-schemalocation-attribute"></a>エラー - schemaLocation 属性を解決できません。
**エラー コード**  
  
 BEC8001  
  
 **説明**  
  
 BizTalk エディターでは、開いているスキーマによってインポートされたスキーマを見つけることができません。  
  
 **ユーザーの操作**  
  
 をクリックして、**スキーマ**ノード。 [プロパティ] ウィンドウ、 **Imports**プロパティ。 省略記号 (**.**) ボタンがアクティブになります。 ボタンをクリックします。 **Imports**ダイアログが表示されます。 ダイアログ ボックスで、インポートされたスキーマへのパスを編集します。
