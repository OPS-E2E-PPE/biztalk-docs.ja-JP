# <a name="error---cannot-resolve-schemalocation-attribute"></a>エラー - schemaLocation 属性を解決できません。
**エラー コード**  
  
 BEC8001  
  
 **説明**  
  
 BizTalk エディターで、開いているスキーマによりインポートされたスキーマを検出できません。  
  
 **ユーザーの操作**  
  
 クリックして、**スキーマ**ノード。 [プロパティ] ウィンドウ、 **Imports**プロパティです。 省略記号ボタン (**.**) ボタンがアクティブになります。 ボタンをクリックします。 **Imports**ダイアログが表示されます。 このダイアログ ボックスで、インポートされたスキーマへのパスを編集します。
