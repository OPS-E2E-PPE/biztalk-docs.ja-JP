# <a name="what-is-a-scheduled-aggregation"></a>スケジュール済みの集計について
スケジュール済みの集計は、今朝の出荷の概要など、指定した期間のビジネス データのスナップショットを表します。 集計が処理される時期については、データベース管理者に問い合わせてください。集計が処理されると、履歴データを見ることができます。
