# <a name="how-to-configure-the-loop-shape"></a>ループ図形を構成する方法
![](../core/media/ebiz-orch-loop.gif "ebiz_orch_loop")  
ループ図形  
  
### <a name="to-configure-a-loop-shape"></a>ループ図形を構成するには  
  
1.  BizTalk 式エディターが表示されていない場合を右クリックし、**ループ**図形を**ブール式の編集**、[プロパティ] ウィンドウで次のようにクリックしますまたは、**省略記号**(。**...**) ボタンを**式**プロパティ。  
  
2.  BizTalk 式エディターでは、反復条件のブール式を作成します。 詳細については、次を参照してください。[式の要件と制限](../core/requirements-and-limitations-for-expressions.md)します。
