# <a name="transforms-receiveport-node"></a>Transforms (ReceivePort ノード)
バインド ファイルの ReceivePort ノードの Transforms ノードには、入力方向のコレクションが含まれています。 一方向の変換は、バインド ファイルと共にエクスポートされるポートを受信します。  
  
## <a name="nodes-in-the-transforms-node"></a>Transforms ノード内のノード  
 次の表に、バインド ファイルのこのノードに設定できるプロパティを示します。  
  
|**名前**|**ノードの種類**|**[データ型]**|**[説明]**|**制限**|**コメント**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[Transform (Transforms ノード)](../core/transform-transforms-node.md)|レコード|変換 (ComplexType)|BizTalk Server マップまたは変換で、送信元スキーマと送信先スキーマ間のマッピングを表す項目を指定します。|任意|既定値: なし|
