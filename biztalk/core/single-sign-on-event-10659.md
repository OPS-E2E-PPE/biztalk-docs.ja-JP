# <a name="single-sign-on-event-10659"></a>シングル サインオン: イベント 10659
## <a name="details"></a>詳細  

|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  製品名   |                 エンタープライズ シングル サインオン                  |
| 製品バージョン | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    イベント ID     |                           10659                            |
|  イベント ソース   |                           ENTSSO                           |
|    コンポーネント    |                            N\A                             |
|  シンボル名  |          SSO_INFO_PASSWORD_SYNC_WINDOWS_STARTING           |
|  メッセージ テキスト   |    (PCNS から) Windows のパスワード同期は starting.%r    |
