# <a name="btadchangerequestaction"></a>BTAD_ChangeRequestAction
BTAD_ChangeRequestAction には、変更要求アクションを示す値が格納されます。  
  
## <a name="remarks"></a>解説  
 BTAD_ChangeRequestAction で使用できる値を次の表に示します。  
  
|値|Description|  
|-----------|-----------------|  
|作成|インポート (上書き指定なし)|  
|更新|インポートまたはインストール (上書き指定あり)|  
|Del|Uninstall|
