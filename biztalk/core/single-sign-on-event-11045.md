# <a name="single-sign-on-event-11045"></a>シングル サインオン: イベント 11045
## <a name="details"></a>詳細  
  
|||  
|-|-|  
|製品名|エンタープライズ シングル サインオン|  
|製品バージョン|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|イベント ID|11045|  
|イベント ソース|ENTSSO|  
|コンポーネント|なし|  
|シンボル名|SSO_WARN_EXISTING_MAPPING_WINDOWS|  
|メッセージ テキスト|指定された Windows アカウントには既にマッピングが存在するので、マッピングを作成できませんでした。<br /><br /> 追跡 ID: %1 %r<br /><br /> アプリケーション名: %2 %r<br /><br /> Windows アカウント: %3|  
  
## <a name="explanation"></a>説明  
 指定した Windows アカウントには既にマッピングが存在しており、重複するマッピングは使用できません。  
  
## <a name="user-action"></a>ユーザーの操作  
 ユーザーの操作は必要ありません。
