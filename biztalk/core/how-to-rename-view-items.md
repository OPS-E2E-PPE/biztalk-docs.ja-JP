# <a name="how-to-rename-view-items"></a>アイテムの表示名の変更方法
多くの場合、ビジネス アクティビティを収集するデータには、クレジット_カード番号や社会保障番号などの機密情報が含まれています。 つまり、一部の種類のビジネス ユーザーがデータのフィルター選択されたサブセットのみを表示する必要があります。  
  
 BAM ビュー ウィザードでのビュー項目の名前を変更するとします。 このウィザードを使用すると、ビジネス アクティビティ、項目、およびビューを作成します。 詳細については、ウィザードを使用して、次を参照してください。[ビジネス アクティビティの定義および Excel でのビュー](../core/defining-business-activities-and-views-in-excel.md)します。  
  
### <a name="to-rename-activity-view-items"></a>アクティビティの表示項目の名前を変更するには  
  
1.  Excel のメニューで、をクリックして**BAM**、 をクリックして**BAM ビュー**、 をクリックし、**次**が表示されるまで、 **BAM ビューの編集します。表示項目 ページ**ページ。  
  
2.  **[BAM ビューの編集します。表示項目] ページ**の一覧で項目を選択し、順にクリックして、**次**。  
  
3.  2 番目の**BAM ビューの編集します。項目の表示** ページで、名前を変更し、をクリックする項目の表示を選択します。**編集**します。  
  
4.  **名前の変更データ項目を**ボックスで、新しい名前を入力し、をクリックし、 **OK**。
