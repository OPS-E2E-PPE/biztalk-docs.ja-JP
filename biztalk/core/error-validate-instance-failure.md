# <a name="error---validate-instance-failure"></a>エラー - インスタンスの検証エラー
**説明**  
  
 **インスタンスの検証**コマンドは、インスタンス メッセージ ファイルがスキーマに準拠していることを検証できませんでした。  
  
 **ユーザーの操作**  
  
 エラー メッセージを調査して、インスタンスの検証エラーの原因を特定します。
