# <a name="error---validate-instance-failure"></a>エラー - インスタンスの検証エラー
**説明**  
  
 **インスタンスの検証**コマンドがインスタンス メッセージのファイルがスキーマに準拠していることを検証に失敗しました。  
  
 **ユーザーの操作**  
  
 インスタンスの検証エラーの原因を特定するエラー メッセージを調査します。
