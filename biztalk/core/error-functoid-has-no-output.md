# <a name="error---functoid-has-no-output"></a>エラー - Functoid に出力がありません。
**エラー コード**  
  
 btm1013  
  
 **説明**  
  
 対象の Functoid には出力リンクがありません。この Functoid は使用できません。  
  
 **ユーザーの操作**  
  
 対象の Functoid に適切な出力リンクを作成します。
