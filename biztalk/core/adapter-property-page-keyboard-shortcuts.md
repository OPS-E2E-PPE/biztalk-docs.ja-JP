# <a name="adapter-property-page-keyboard-shortcuts"></a>[アダプターのプロパティ] ページのキーボード ショートカット
アダプターのプロパティ ページでキーボード ショートカットを使用すると、標準的な Microsoft Windows アプリケーションのようはメニューを使用して、それ以外の場合のコマンドにアクセスします。 次の表では、使用できるキーボード ショートカットについて説明します。  
  
> [!NOTE]
>  コンマ区切りのキーの組み合わせを入力する必要があります。 たとえば、"CTRL + I, CTRL + R"CTRL + I, を最初に入力する必要がありますとし、CTRL キーを押しながら R キーを入力する必要があります。 "ALT + Z, I, F"などのキーの組み合わせは ALT + Z, 最初に入力する必要がありますが、し入力"I"および"F"、ALT キーを押さなくてもことを意味します。  
  
|操作|キーボード ショートカット|  
|------------|-----------------------|  
|ダイアログ ボックスのボタンと、プロパティ グリッドの上に移動します。 または、お客様が、プロパティ グリッドであり、矢印ボタンを使用して、適切なプロパティの値に移動した後後、は、プロパティ値を編集する TAB キーを使用します。 プロパティの値に省略記号 (...) ボタンまたはドロップダウン メニューがある場合は、代わりに f4 キーを使用します。|Tab|  
|プロパティ グリッドでプロパティ フィールド内を移動します。|方向キー|  
|プロパティ名 フィールドからの省略可能な文字列値が無効にします。|Fn、N または SHIFT + F10, N|  
|省略記号ボタン (…) またはドロップ ダウン ボックスがあるプロパティ値を編集します。他のすべてのユーザーは、TAB キーを使用します。|F4|  
|編集したプロパティの値に変更を保存します。|Enter|  
|ポップアップ メニューを閉じます。|Esc|
