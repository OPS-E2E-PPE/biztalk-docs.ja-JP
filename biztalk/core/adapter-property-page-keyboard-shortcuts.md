# <a name="adapter-property-page-keyboard-shortcuts"></a>[アダプターのプロパティ] ページのキーボード ショートカット
他の標準的な Microsoft Windows アプリケーションと同様、[アダプターのプロパティ] ページでも、メニューのコマンドをキーボード ショートカットで実行できます。 使用できるキーボード ショートカットを次の表に示します。  
  
> [!NOTE]
>  キーの組み合わせがコンマで区切って示されている場合は、その順番でキーを押す必要があることを示します。 たとえば、"CTRL + I, CTRL + R"にすると CTRL + I, を最初に入力する必要がありますし、ctrl キーを押しながら R キーを入力する必要があります。 "ALT + Z, I, F"などのキーの組み合わせ ALT + Z, 最初に入力する必要がありますが、し入力"I"および"F"、ALT キーを押さずことを意味します。  
  
|操作|キーボード ショートカット|  
|------------|-----------------------|  
|プロパティ グリッドに移動し、ダイアログ ボックスのボタン間を移動するか、 プロパティ グリッドで、矢印ボタンを使用して適切なプロパティの値に移動した後、Tab キーを使用してプロパティの値を編集できます。 プロパティ値に省略記号ボタン ([...]) またはドロップダウン メニューがある場合は、代わりに F4 キーを使用します。|Tab|  
|プロパティ グリッドのプロパティ フィールド間を移動します。|方向キー|  
|プロパティ名フィールドの省略可能な文字列値に NULL を設定します。|Fn, N または Shift+F10, N|  
|省略記号ボタン ([...]) またはドロップダウン ボックスがあるプロパティの値を編集します。その他のすべてのプロパティ値では、Tab キーを使用します。|F4|  
|編集したプロパティ値に対する変更を保存します。|Enter|  
|ポップアップ メニューを閉じます。|Esc|
