# <a name="what-you-should-know-before-programming-single-sign-on"></a>シングル サインオンをプログラミングする前に知っておく必要があります。
このドキュメントを効率的に使用するには、以下の知識が必要です。  
  
-   Microsoft [!INCLUDE[btsPlatformsComApis](../includes/btsplatformscomapis-md.md)] オペレーティング システム。 特に、Windows のセキュリティ機能に関する知識が必要です。  
  
-   エンタープライズ シングル サインオンの管理機能。特に、ユーザー インターフェイスを使用する管理操作の実行方法。  
  
 使用する API および開発環境に応じて、以下のインターフェイスに関する知識も必要です。  
  
-   COM (COM)  
  
-   .NET Framework および共通言語ランタイム  
  
-   Windows ネットワーク (特にチケットの送受信方法)
