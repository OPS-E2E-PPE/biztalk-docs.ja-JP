# <a name="how-to-save-a-pipeline"></a>パイプラインを保存する方法
パイプラインを保存するには、.btp ファイルに指定するパイプラインの構成に関する情報が格納されます。  
  
-   パイプラインの名前。  
  
-   テンプレートおよびポリシーに基づく、(送信または受信)。  
  
-   順序と各ステージで使用されるコンポーネントのセットを含む、そのパイプラインの一部であるコンポーネントのシーケンス。  
  
-   各パイプライン コンポーネントの構成情報。  
  
### <a name="to-save-a-pipeline"></a>パイプラインを保存するには  
  
1.  **[ファイル]** メニューの **[保存]** をクリックします。  
  
     - または -  
  
2.  デザイン画面で、パイプラインを表示するには、画面の上部付近のタブをクリックします。  
  
3.  **ファイル** メニューのをクリックして**付けて**します。  
  
4.  **ファイルに名前を付けて** ダイアログ ボックスで、**ファイル**名フィールドに、パイプラインの名前を入力します。 既定の名前は提供します。  
  
5.  **[OK]** をクリックします。
