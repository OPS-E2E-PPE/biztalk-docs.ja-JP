# <a name="x12-997-acknowledgment-error-codes"></a>X12 997 受信確認エラー コード
このトピックでリストのエラー コードは、997 受信確認を X12 のセグメント内で使用されます。 これらのセグメントの詳細については、次を参照してください。 [X12 997 受信確認](../core/x12-997-acknowledgment.md)します。  
  
 各テーブルがどのエラーを示すコードの仕様がサポートされている X12 で指定された[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]EDI と AS2 はサポートされていません。  
  
 **AK304 エラー コード**  
  
 次の表では、(データ セグメント メモ) AK3 セグメントの AK304 データ要素で使用されるエラー コードを示します。  
  
|エラー コード|条件|サポートされていますか。|  
|----------------|---------------|----------------|  
|1|セグメント ID が認識されません|はい|  
|2|セグメントが正しくありません|はい|  
|3|必須のセグメントがありません。|はい|  
|4|最大回数を超えてループが発生します|はい|  
|5|セグメントが最大使用数を超えています|はい|  
|6|セグメントで定義されていないトランザクション セット|はい|  
|7|適切なシーケンスではなくセグメントします。|はい|  
|8|セグメントのデータ要素エラーをが|はい|  
|511|末尾の区切り記号 (カスタム コード) が発生しました|はい|  
  
 **AK403 エラー コード**  
  
 次の表では、(データ要素メモ) AK4 セグメントの AK403 データ要素で使用されるエラー コードを示します。  
  
|エラー コード|条件|サポートされていますか。|  
|----------------|---------------|----------------|  
|1|必須のデータ要素がありません|はい|  
|2|必須の条件付きデータ要素がありません|はい|  
|3|データ要素が多すぎます|はい|  
|4|データ要素が短すぎます|はい|  
|5|データ要素が長すぎます|はい|  
|6|データ要素に無効な文字があります|はい|  
|7|無効なコード値|はい|  
|8|日付が無効です。|はい|  
|9|無効な時刻|はい|  
|10|除外条件に違反しました|はい|  
  
 **AK501 エラー コード**  
  
 次の表では、(トランザクション セット応答トレーラー) AK5 セグメントの AK501 データ要素で使用されるエラー コードを示します。  
  
|エラー コード|条件|サポートされていますか。|  
|----------------|---------------|----------------|  
|A|受理|はい|  
|E|受け入れていましたが、エラーが検出されました|はい<br /><br /> 注:"E"の状態をエラー コードの可能性があります。|  
|M|拒否されると、メッセージ認証コード (MAC) に失敗しました|いいえ|  
|P|一部受理、少なくとも 1 つのトランザクション セットが拒否されました|はい|  
|R|拒否されました。|はい|  
|W|拒否されると、保証は有効性のテストが失敗しました|いいえ|  
|x|復号化を分析した後にコンテンツを拒否します。|いいえ|  
  
 **AK502 から AK506 のエラー コード**  
  
 AK502 から AK506 のデータ要素の AK5 セグメント (トランザクション セット応答トレーラー) で使用されるエラー コードを次の表に示します。  
  
|エラー コード|条件|サポートされていますか?/<br />AK501 との関連性ですか。|  
|----------------|---------------|------------------------------------------|  
|1|トランザクションがサポートされている設定されていません。|[はい]/R|  
|2|トランザクション セット トレーラーがありません。|[はい]/R|  
|3|ヘッダーとトレーラのトランザクション セット制御番号が一致しません|[はい]/R|  
|4|含まれているセグメントの数が、実際の数と一致しません|[はい]/R|  
|5|エラーに 1 つ以上のセグメントが含まれています|[はい]/R|  
|6|不明または無効なトランザクション セット識別子|[はい]/R|  
|7|不明または無効なトランザクション セット制御番号が (重複するトランザクション数が発生した可能性があります)|[はい]/R|  
|8 ~ 27|-|いいえ|

 **AK905 AK909 のエラー コードから**  
  
 AK905 ~ AK909 のデータ要素の AK9 セグメント (機能グループ応答トレーラー) で使用されるエラー コードを次の表に示します。
