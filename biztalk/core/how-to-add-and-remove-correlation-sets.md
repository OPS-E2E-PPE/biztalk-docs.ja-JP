# <a name="how-to-add-and-remove-correlation-sets"></a>追加および関連付けセットを削除する方法
に対して関連付けセットを追加またはオーケストレーションから削除するオーケストレーションの種類 ウィンドウを使用できます。 関連付けセットは、関連付けの種類のインスタンスです。  
  
### <a name="to-add-and-configure-a-correlation-set"></a>追加して、関連付けセットの構成  
  
-   オーケストレーションの種類 ウィンドウで、右クリック**関連付けセット** をクリックし、**新しい関連付けセット**します。  
  
     **関連付けのプロパティ** ダイアログ ボックスが表示されます。 関連付けセットに含めるプロパティを追加します。 関連付けセットおよび対応する関連付けの種類が作成されます。  
  
### <a name="to-remove-a-correlation-set"></a>関連付けセットを削除するには  
  
-   オーケストレーションの種類 ウィンドウで、クリックして削除する関連付けセットを右クリックして**削除**します。
