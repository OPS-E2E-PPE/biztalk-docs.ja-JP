# <a name="how-to-delete-nodes"></a>ノードを削除する方法
スキーマ ツリー内のノードを削除する場合があります。 このトピックでは、このタスクの詳細な手順を説明します。  
  
### <a name="to-delete-a-node"></a>ノードを削除するには  
  
1.  削除するをクリックするノードを右クリックして**削除**します。  
  
2.  クリックして**はい**確認のダイアログ ボックスでします。  
  
> [!CAUTION]
>  削除すると、**レコード**ノードは、そのすべての子ノードも削除されます。  
  
> [!CAUTION]
>  スキーマ ツリーの別の場所への参照であるノードを削除すると、参照元のノードも削除されます。  
  
> [!NOTE]
>  ノードが選択不可の場合は読み取り専用および削除することはできません。
