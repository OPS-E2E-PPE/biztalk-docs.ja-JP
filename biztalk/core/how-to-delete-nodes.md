# <a name="how-to-delete-nodes"></a>ノードを削除する方法
スキーマ ツリーのノードを削除したい場合があります。 このトピックでは、この作業を行うための手順について説明します。  
  
### <a name="to-delete-a-node"></a>ノードを削除するには  
  
1.  削除、およびをクリックするノードを右クリックして**削除**です。  
  
2.  をクリックして**はい**確認のダイアログ ボックスでします。  
  
> [!CAUTION]
>  削除すると、**レコード**ノードは、そのすべての子ノードも削除します。  
  
> [!CAUTION]
>  スキーマ ツリーの他の場所への参照を含むノードを削除すると、参照しているノードも削除されます。  
  
> [!NOTE]
>  淡色表示のノードは、読み取り専用で削除できません。
