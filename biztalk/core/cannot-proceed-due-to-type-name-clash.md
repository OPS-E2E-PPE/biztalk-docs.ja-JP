## <a name="explanation"></a>説明  
 このエラーは、同じ定義済み名前空間の複数のアイテムが同じ名前であることを示します。  
  
## <a name="user-action"></a>ユーザーの操作  
 同じ名前空間のアイテム名を変更するか、別の名前空間に配置します。
