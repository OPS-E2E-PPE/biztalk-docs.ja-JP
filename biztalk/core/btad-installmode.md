# <a name="btadinstallmode"></a>BTAD_InstallMode
BTAD_InstallMode は、BizTalk アプリケーションの展開のインストール モードを表します。  
  
## <a name="remarks"></a>コメント  
 BTAD_InstallMode で使用できる値を次の表に示します。  
  
|値|説明|  
|-----------|-----------------|  
|インストール|BizTalkHostInstance (ローカル コンピューター) の作成および更新|  
|[インポート]|ConfigurationDb (グループの BizTalk 管理データベース) の作成および更新|  
|アンインストール|BizTalkHostInstance (ローカル コンピューター) からの削除|
