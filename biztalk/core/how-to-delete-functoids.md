# <a name="how-to-delete-functoids"></a>Functoid を削除する方法
マップを開発する場合は、既存の functoid を削除する必要があります。 Functoid を削除すると、関連付けられている入力と出力のリンクだけでなく、追加の入力パラメーター、スクリプト仕様、およびテーブル グリッド構成も削除します。  
  
### <a name="to-delete-an-existing-functoid"></a>既存の functoid を削除するには  
  
1.  削除する functoid を含むグリッド ページを表示します。 グリッド ページ間の切り替えなど、グリッド ページの操作の詳細については、次を参照してください。[グリッド ページの操作](../core/working-with-grid-pages.md)します。  
  
2.  クリックして削除する functoid を選択します。  
  
     複数の functoid を選択し、それらを同時に削除できます。 Functoid およびリンクの一括選択の詳細については、次を参照してください。[および Functoid を複数のリンクを選択する方法](../core/how-to-select-multiple-links-and-functoids.md)します。  
  
3.  **編集** メニューのをクリックして**削除**します。  
  
    > [!NOTE]
    >  DEL キーを押してまたは関連するグリッド ページで選択した functoid を右クリックしてをクリックして**削除**コンテキスト メニュー。  
  
    > [!NOTE]
    >  Functoid またはリンクの削除を元に戻すことはできません。 元に戻すまたはユーザーの操作を再実行する方法の詳細については、次を参照してください。[を元に戻すまたはユーザーの操作をやり直す方法](../core/how-to-undo-or-redo-user-operations.md)します。
