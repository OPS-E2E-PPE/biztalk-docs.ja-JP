# <a name="how-to-work-with-multiple-nodes-simultaneously"></a>同時に複数のノードを操作する方法
一部の操作を複数のノードで同時に行うことができます。ただし、関連するノードをすべて選択する必要があります。 このトピックでは、複数のノードを同時に操作する方法について説明します。  
  
### <a name="to-select-multiple-nodes-that-are-not-adjacent"></a>隣接していない複数のノードを選択するには  
  
1.  選択対象となる複数のノードの、最初のノードを選択します。  
  
2.  <localizedText>Ctrl</localizedText> キーを押しながら、残りの対象ノードを選択します。  
  
     これで、隣接していないノードが同時にすべて選択されます。  
  
### <a name="to-select-a-range-of-adjacent-nodes"></a>隣接しているノードをまとめて選択するには  
  
1.  同時に選択する隣接するノードのセットの 1 つの極端な例として、ノードを選択します。  
  
2.  <localizedText>Shift</localizedText> キーを押しながら、選択対象となる最後のノードを選択します。  
  
     手順 1. および手順 2. で選択したノードによって指定された範囲内の隣接しているノードが、同時にすべて選択されます。  
  
### <a name="to-modify-property-values-of-simultaneously-selected-nodes"></a>同時に選択したノードのプロパティの値を変更するには
