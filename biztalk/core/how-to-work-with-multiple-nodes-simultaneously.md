# <a name="how-to-work-with-multiple-nodes-simultaneously"></a>複数のノードを同時に操作する方法
一部の操作を同時に、複数のノードで実行できますが、関連するノードのすべてを選択してください。 このトピックでは、複数のノードを同時に操作する方法について説明します。  
  
### <a name="to-select-multiple-nodes-that-are-not-adjacent"></a>隣接していない複数のノードを選択するには  
  
1.  隣接するいないと同時に選択されるノードのセットでは、最初のノードを選択します。  
  
2.  キーを押すし、CTRL キーを押し隣接するいないと同時に選択されるノードのセット内の残りのノードを選択します。  
  
     隣接していないノードは、すべて同時に選択されます。  
  
### <a name="to-select-a-range-of-adjacent-nodes"></a>隣接するノードの範囲を選択するには  
  
1.  隣接するノードを同時に選択できる一連の極端なノードを選択します。  
  
2.  キーを押して shift </localizedtext しその一方で同時に選択されるノードのセットのノードを選択します。  
  
     手順 1. および 2. で選択したノードで定義されている隣接するノードの範囲は、すべて同時に選択されます。  
  
### <a name="to-modify-property-values-of-simultaneously-selected-nodes"></a>同時に選択したノードのプロパティの値を変更するには  
  
1.  隣接していない複数のノードを選択するには"するには」の手順に従って、複数のノードを選択します。 このドキュメントでは。  
  
2.  関連するプロパティ値を変更、**プロパティ**ウィンドウ。  
  
> [!NOTE]
>  選択したノードすべてに共通するプロパティのみを同時に変更できます。
