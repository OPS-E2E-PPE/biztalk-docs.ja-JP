# <a name="supported-adapter-xsd-element-types"></a>サポートされているアダプターの XSD 要素の型
次の表は、アダプター フレームワークでサポートされる要素の一覧です。 構成スキーマに新しい要素を定義する際には、次の例の `string` 部分を、表に示されている任意の型に置き換えてください。  
  
```  
<xs:element name="uri" type="xs:string">  
```  
  
|型|XML 型|UI の動作|その他の仕様|  
|----------|--------------|-----------------|---------------------|  
|string|なし|編集ボックス。この型のみを受け取ります。|最大および最小を制限するために属性|  
|normalizedString|なし|編集ボックス。この型のみを受け取ります。|最大および最小を制限するために属性|  
|整数 (integer)|なし|編集ボックス。この型のみを受け取ります。|最大および最小を制限するために属性|  
|positiveInteger|なし|編集ボックス。この型のみを受け取ります。|最大および最小を制限するために属性|  
|negativeInteger|なし|編集ボックス。この型のみを受け取ります。|最大および最小を制限するために属性|  
|nonNegativeInteger|なし|編集ボックス。この型のみを受け取ります。|最大および最小を制限するために属性|  
|nonPositiveInteger|なし|編集ボックス。この型のみを受け取ります。|最大および最小を制限するために属性|  
|int|なし|編集ボックス。この型のみを受け取ります。|最大および最小を制限するために属性|  
|unsignedInt|なし|編集ボックス。この型のみを受け取ります。|最大および最小を制限するために属性|  
|long|なし|編集ボックス。この型の 10 進数のみを受け取ります。|最大および最小を制限するために属性|  
|unsignedLong|なし|編集ボックス。この型の 10 進数のみを受け取ります。|最大および最小を制限する属性|  
|short|なし|編集ボックス。この型のみを受け取ります。|最大および最小を制限するために属性|  
|unsignedShort|なし|編集ボックス。この型のみを受け取ります。|最大および最小を制限するために属性|  
|decimal|なし|編集ボックス。この型のみを受け取ります。|最大および最小を制限するために属性|  
|float|なし|編集ボックス。この型のみを受け取ります。|最大および最小を制限するために属性|  
|double|なし|編集ボックス。この型のみを受け取ります。|最大および最小を制限するために属性|  
|boolean|なし|ドロップダウン リスト。ブール値が表示されます。|なし|  
|time|なし|編集ボックス。この型のみを受け取ります。|なし|  
|dateTime|なし|編集ボックス。この型のみを受け取ります。 フィールド領域の最後には省略記号ボタンが表示されます。 省略記号ボタンをクリックすると、カレンダーが表示されます。|なし|  
|date|なし|編集ボックス。この型のみを受け取ります。 フィールド領域の最後には省略記号ボタンが表示されます。 省略記号ボタンをクリックすると、カレンダーが表示されます。|なし|  
|gMonth|なし|編集ボックス。この型のみを受け取ります。|この値は文字列であり、想定どおりの動作が行なわれない場合があります。 月の値を保持するには、制限付きの xsd:int 型を使用する方法もあります。|  
|gYear|なし|編集ボックス。この型のみを受け取ります。|この値は文字列であり、想定どおりの動作が行なわれない場合があります。 年の値を保持するには、制限付きの xsd:int 型を使用する方法もあります。|  
|gYearMonth|なし|編集ボックス。この型のみを受け取ります。|この値は文字列であり、想定どおりの動作が行なわれない場合があります。 年と月の値を保持するには、制限付きの xsd:int 型を使用する方法もあります。|  
|gDay|なし|編集ボックス。この型のみを受け取ります。|この値は文字列であり、想定どおりの動作が行なわれない場合があります。 日の値を保持するには、制限付きの xsd:int 型を使用する方法もあります。|  
|gMonthDay|なし|編集ボックス。この型のみを受け取ります。|この値は文字列であり、想定どおりの動作が行なわれない場合があります。 月と日の値を保持するには、制限付きの xsd:int 型を使用する方法もあります。|  
|名前|なし|編集ボックス。この型のみを受け取ります。|なし|

|[ファイル名]|FileName|フィールド領域の最後には省略記号ボタンが表示されます。 省略記号ボタンをクリックし、 **Windows FileOpen** ファイルを選択できるダイアログ ボックスが表示されます。|なし|  
|Folder Name|FolderName|フィールド領域の最後には省略記号ボタンが表示されます。 省略記号ボタンをクリックし、 **Windows Folder Open**  ダイアログ ボックスでフォルダーを選択することができます。|なし|  
|SSO App ID|SSOAppID|ドロップダウン リスト。SSO アプリケーションの一覧が表示されます。|なし|  
|Password|Password|編集ボックス。クリア テキストではなく "*" が表示されます。|なし|
