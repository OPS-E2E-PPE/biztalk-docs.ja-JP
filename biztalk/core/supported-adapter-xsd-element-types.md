# <a name="supported-adapter-xsd-element-types"></a>サポートされているアダプターの XSD 要素の型
次の表では、アダプター フレームワークでサポートされている要素を示します。 構成スキーマに新しい要素を定義するときに、次の種類のいずれかに置き換える使用`string`次の例。  
  
```  
<xs:element name="uri" type="xs:string">  
```  
  
|型|XML 型|UI の動作|その他の仕様|  
|----------|--------------|-----------------|---------------------|  
|string|なし|編集ボックス型のみを受け取ります。|最大/最小を制限する属性|  
|normalizedString|なし|編集ボックス型のみを受け取ります。|最大/最小を制限する属性|  
|整数 (integer)|なし|編集ボックス型のみを受け取ります。|最大/最小を制限する属性|  
|positiveInteger|なし|編集ボックス型のみを受け取ります。|最大/最小を制限する属性|  
|negativeInteger|なし|編集ボックス型のみを受け取ります。|最大/最小を制限する属性|  
|nonNegativeInteger|なし|編集ボックス型のみを受け取ります。|最大/最小を制限する属性|  
|nonPositiveInteger|なし|編集ボックス型のみを受け取ります。|最大/最小を制限する属性|  
|ssNoversion|なし|編集ボックス型のみを受け取ります。|最大/最小を制限する属性|  
|unsignedInt|なし|編集ボックス型のみを受け取ります。|最大/最小を制限する属性|  
|long|なし|型のみを受け取ります ボックスと 10 進数を編集します。|最大/最小を制限する属性|  
|unsignedLong|なし|型のみを受け取ります ボックスと 10 進数を編集します。|属性を 1 分あたりの最大の制約|  
|short|なし|編集ボックス型のみを受け取ります。|最大/最小を制限する属性|  
|unsignedShort|なし|編集ボックス型のみを受け取ります。|最大/最小を制限する属性|  
|Decimal|なし|編集ボックス型のみを受け取ります。|最大/最小を制限する属性|  
|FLOAT|なし|編集ボックス型のみを受け取ります。|最大/最小を制限する属性|  
|double|なし|編集ボックス型のみを受け取ります。|最大/最小を制限する属性|  
|boolean|なし|ドロップダウン リストがブール値が表示されます。|なし|  
|time|なし|編集ボックス型のみを受け取ります。|なし|  
|dateTime|なし|編集ボックス型のみを受け取ります。 フィールド領域の末尾に省略記号が表示されます。 省略記号をクリックし、カレンダーが表示されます。|なし|  
|日付|なし|編集ボックス型のみを受け取ります。 フィールド領域の末尾に省略記号が表示されます。 省略記号をクリックし、カレンダーが表示されます。|なし|  
|gMonth|なし|編集ボックス型のみを受け取ります。|この値は文字列であるため、したがってが正しく動作しません。 1 か月の値を保持する制限付きの xsd:int 型を使用してください。|  
|gYear|なし|編集ボックス型のみを受け取ります。|この値は文字列であるため、したがってが正しく動作しません。 年の値を保持する制限付きの xsd:int 型を使用してください。|  
|gYearMonth|なし|編集ボックス型のみを受け取ります。|この値は文字列であるため、したがってが正しく動作しません。 年と月の値を保持する制限付きの xsd:int 型を使用してください。|  
|gDay|なし|編集ボックス型のみを受け取ります。|この値は文字列であるため、したがってが正しく動作しません。 日の値を保持する制限付きの xsd:int 型を使用してください。|  
|gMonthDay|なし|編集ボックス型のみを受け取ります。|この値は文字列であるため、したがってが正しく動作しません。 月と日の値を保持する制限付きの xsd:int 型を使用してください。|  
|名前|なし|編集ボックス型のみを受け取ります。|なし|  
|NCName|なし|編集ボックス型のみを受け取ります。|なし|  
|anyURI|なし|編集ボックス型のみを受け取ります。|なし|  
|Sequence|"Sequence"スキーマ要素|なし|なし|  
|グループ|なし|A「+」または"-"記号を展開するか、グループ内のすべてのフィールドを折りたたみます。<br /><br /> プロパティ ページの右側にある編集機能がありません。|なし|  
|ファイル名|FileName|フィールド領域の末尾に省略記号が表示されます。 省略記号をクリックし、 **Windows FileOpen**  ダイアログ ボックスが表示されたら、ファイルの選択可能になります。|なし|  
|フォルダー名|フォルダー名|フィールド領域の末尾に省略記号が表示されます。 省略記号をクリックし、 **Windows Folder Open**  ダイアログ ボックスでフォルダーを選択できます。|なし|  
|SSO アプリ ID|SSOAppID|SSO アプリケーションの一覧を含むドロップダウン リスト|なし|  
|パスワード|パスワード|編集ボックスに"*"クリア テキストではなく表示されます。|なし|
