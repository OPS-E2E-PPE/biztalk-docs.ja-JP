# <a name="how-to-navigate-with-the-keyboard"></a>キーボードで移動する方法
マウスの代わりに、キーボードを使用して、デザイン画面を移動できます。 使用できるキーを次の表に示します。  
  
|キー|効果|  
|---------|------------|  
|下方向キー|など、次の区分線または図形の下にカーソルを移動**開始**と**エンド**図形。|  
|上矢印|など、前の区分線または図形の上にカーソルを移動**開始**と**エンド**図形。|  
|PageUp|ステージの前の開始図形にカーソルのフォーカスを変更します。 パイプラインの前の部分が表示されるように、ページをスクロール効果があります。|  
|PageDown|ステージの次の開始図形にカーソルのフォーカスを変更します。 ページを下へスクロールして、パイプラインの次の部分が表示されるように効果があります。|  
|Home|フォーカスを**開始**パイプラインの形状。|  
|END|フォーカスを**エンド**パイプラインの形状。|
