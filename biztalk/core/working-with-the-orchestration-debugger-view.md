# <a name="working-with-the-orchestration-debugger-view"></a>オーケストレーション デバッガー ビューの操作
結果一覧のセルを右クリックし、オーケストレーション デバッガーを選択して、オーケストレーション デバッガー ビューにアクセスできます。  
  
 オーケストレーション デバッガーは、1 つのインスタンスの技術的な処理アクティビティを表示します。 ビューには、オーケストレーションの表示の画像が表示されます。  
  
 オーケストレーション デバッガー ビューからは、メッセージ フロー ビューに移動できます。
