# <a name="error---unspecified-fatal-error"></a>エラー - 特定できない致命的なエラー
**エラー コード**  
  
 BEC1001、BEC2004、BEC3001  
  
 **説明**  
  
 スキーマのコンパイル中に、特定できない致命的なエラーが発生しました。  
  
 **ユーザーの操作**  
  
 スキーマ ファイルが有効で、壊れていないことを確認してください。
