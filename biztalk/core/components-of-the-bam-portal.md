# <a name="components-of-the-bam-portal"></a>BAM ポータルのコンポーネント
BAM ポータルは、ポータルの 3 つのページに対応する 3 つの機能領域に分割された ASP.NET Web ページです。 このセクションでは、これらのページがについて説明します。アクティビティの検索ページ、集計 ページ、および警告マネージャー ページ。 これらのページを使用して、アクティビティを作成、既存のアクティビティを検索および既存の活動を変更することができます。
