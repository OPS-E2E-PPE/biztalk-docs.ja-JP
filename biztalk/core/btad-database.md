# <a name="btaddatabase"></a>BTAD_Database
アプリケーションのインポート時に処理前および処理後のスクリプトを実行すると、BTAD_Database 環境変数が設定されます。 この変数は、グループの BizTalk 管理データベースの名前を指定します。
