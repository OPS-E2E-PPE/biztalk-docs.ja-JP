# <a name="single-sign-on-event-10845"></a>シングル サインオン:イベント 10845
## <a name="details"></a>詳細  
  
|                 |                                                                                                                       |
|-----------------|-----------------------------------------------------------------------------------------------------------------------|
|  製品名   |                                               エンタープライズ シングル サインオン                                               |
| 製品バージョン |                              [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                               |
|    イベント ID     |                                                         10845                                                         |
|  イベント ソース   |                                                        ENTSSO                                                         |
|    コンポーネント    |                                                          なし                                                          |
|  シンボル名  |                                                  ENTSSO_E_DB_ACCESS2                                                  |
|  メッセージ テキスト   | SSO データベースへのアクセスを試みているときにエラーが発生しました。 詳細については、イベント ログ (コンピューター "%1" 上) を参照してください。 |
  
## <a name="explanation"></a>説明  
 SSO データベースへのアクセスを試みているときにエラーが発生しました。  
  
## <a name="user-action"></a>ユーザーの操作  
 SQL Server からの追加情報の指定したコンピューター上のイベント ログを参照してください。
