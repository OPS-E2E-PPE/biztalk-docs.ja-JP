# <a name="edi-and-as2-biztalk-server-samples-folder"></a>EDI および AS2 (BizTalk Server Samples フォルダ)
Microsoft BizTalk Server には、ソフトウェア開発キット (SDK) の中に次の EDI および AS2 のサンプルが含まれます。 ここでは、EDI のサンプルで示される機能に関する詳細、サンプルの構築と実行の手順、および期待される効果について説明します。  
  
## <a name="in-this-section"></a>このセクションの内容  
  
-   [AS2 over File (BizTalk Server サンプル)](../core/as2-over-file-biztalk-server-sample.md)です。 FILE 受信場所経由で AS2 メッセージを受信する方法を示します。  
  
-   [メッセージ強化サンプル (BizTalk Server サンプル)](../core/message-enrichment-sample-biztalk-server-sample.md)です。 EDI ドキュメントのトランザクション セット メッセージにインターチェンジ ヘッダーを追加する方法を示します。  
  
## <a name="additional-sample-code"></a>その他のサンプル コード  
  
-   [FILE 送信ポート経由で AS2 メッセージの送信](../core/sending-an-as2-message-over-a-file-send-port.md)です。 このトピックでは、AS2 メッセージにヘッダーを書き込む方法と、AS2 ヘッダー コンテキスト プロパティを昇格させる方法について説明します。  
  
-   [送信パーティ解決の AS2 コンテキスト プロパティの書き込み](../core/writing-as2-context-properties-for-outbound-party-resolution.md)です。 このトピックでは、AS2To プロパティまたは UserHttpHeaders プロパティを (AS2 プロパティと共に) コンテキストに書き込み、送信 AS2 メッセージのパーティを解決する方法について説明します。
