# <a name="http-adapter-performance-counters"></a>HTTP アダプターのパフォーマンス カウンター
パフォーマンス カウンターを使用して、サービスによってサイトまたはシステムで実行される作業の具体的な側面を監視できます。 パフォーマンス カウンターは、サーバー パフォーマンスに関する問題を特定してトラブルシューティングする際に役立ちます。  
  
 次のパフォーマンス カウンターは各ホスト インスタンスにアクセスできますが、 **BizTalk:HTTP Receive Adapter**と**BizTalk:HTTP Send Adapter**パフォーマンス オブジェクト カテゴリ。  
  
|**カテゴリ**|**カウンター**|**[説明]**|  
|------------------|-----------------|---------------------|  
|BizTalk:HTTP 受信アダプター|メモリのキュー サイズ|Http の受信メッセージの数には、アダプターの内部メモリ キューが表示されます。|  
||メッセージを受信しました|受信アダプター、HTTP で受信した HTTP 要求の合計数。 要求メッセージは、HTTP で完全に読み取った後、カウンターがインクリメントが HTTP クライアントからの受信アダプター。|  
||メッセージの受信/秒|Http 受信 HTTP 要求の数には、1 秒あたりにアダプターが表示されます。 カウンターは、要求が HTTP クライアントから HTTP によって完全に読み取られたメッセージの受信アダプターにのみ適用されます。|  
||送信されたメッセージ|受信アダプター、HTTP で送信された HTTP 応答の合計数。 カウンターは、HTTP クライアントに正常に送信された応答メッセージに対してのみインクリメントされます。|  
||メッセージの送信/秒|HTTP によって送信された HTTP 応答の数には、1 秒あたりにアダプターが表示されます。 このカウンターは、HTTP クライアントに正常に送信された応答メッセージのみに適用されます。|  
||メッセージをバッチに追加する時間|HTTP メッセージを指定するときまでの平均時間が IIS によってアダプターと、メッセージをバッチに追加するときに受信します。|  
||バッチの作成に時間|Http に要した平均時間の受信アダプタはメッセージ バッチを作成します。|  
|BizTalk:HTTP 送信アダプター|メモリのキュー サイズ|アダプターの内部メモリ キューに送信して HTTP で送信メッセージの数。|  
||受信したメッセージ|送信アダプター、HTTP で受信した HTTP 応答メッセージの合計数。 応答メッセージは HTTP サーバーから HTTP 送信アダプターによって完全に読み取った後、カウンターがインクリメントされます。|  
||メッセージの受信/秒|1 秒あたりにアダプターに送信して、HTTP で受信した HTTP 応答の数。 このカウンターは、HTTP サーバーから HTTP 送信アダプターによって完全に読み取られた応答メッセージのみに適用されます。|  
||送信されたメッセージ|送信アダプター、HTTP で送信された HTTP 要求の合計数。 カウンターは、送信先 URL に到達した要求メッセージに対してのみインクリメントされます。|  
||メッセージの送信/秒|1 秒あたりにアダプターに送信して、HTTP で送信された HTTP 要求の数。 カウンターは、送信先 URL に到達した要求メッセージのみに適用されます。|  
  
## <a name="to-access-performance-counters"></a>パフォーマンス カウンターにアクセスするには  
 パフォーマンス カウンターにアクセスするのにには、次の手順を使用します。  
  
#### <a name="if-you-are-using-windows-2008"></a>Windows 2008 を使用している場合  
  
1.  クリックして**開始**、 をポイント**管理ツール**、順にクリックします**パフォーマンス モニター**します。  
  
2.  **パフォーマンス モニター**  ダイアログ ボックスで、展開**監視ツール**を選択します**パフォーマンス モニター**、 をクリックし、**追加**。  
  
3.  **カウンターの追加**] ダイアログ ボックスから、**使用可能なカウンター**一覧で、展開、 **[BizTalk:HTTP**パフォーマンス カウンター オブジェクトと監視するカウンターを選択します。  
  
4.  **インスタンスの選択したオブジェクト**一覧をクリックして、選択したカウンターを監視する特定のインスタンスを選択**追加**します。  使用可能なカウンターのインスタンスすべてを選択する\<**すべてのインスタンス**\>します。  
  
5.  カウンターを追加すると、次のようにクリックします。 **OK**します。  
  
     選択したパフォーマンス カウンターが表示される、**パフォーマンス モニター**画面。
