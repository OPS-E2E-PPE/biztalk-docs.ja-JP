# <a name="error---schema-file-not-in-build"></a>エラー - スキーマ ファイルがビルドに存在しません
**説明**  
  
 コマンド (**インスタンスの検証**、**インスタンスの生成**、または**スキーマの検証**) ためを実行できませんでした、**ビルド アクション**スキーマ ファイルのプロパティに設定されて**None**、このスキーマがこれらのコマンドに参加していることを防止します。  
  
 **ユーザーの操作**  
  
 Microsoft では、関連するスキーマ ファイルを選択します。[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]ソリューション エクスプ ローラーで、し、[プロパティ] ウィンドウで次のように変更します。、**ビルド アクション**プロパティを、**コンパイル**します。 しようとし、**インスタンスの検証**、**インスタンスの生成**、または**スキーマの検証**コマンドを再実行します。
