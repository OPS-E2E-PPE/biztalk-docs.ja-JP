## <a name="explanation"></a>説明  
 SSO 管理者アカウント名を変更するには、クライアント ユーザーが新しい SSO 管理者アカウントのメンバーである必要があります。  
  
## <a name="user-action"></a>ユーザーの操作  
 SSO 管理者アカウント名を変更するには、新しい SSO 管理者アカウントのメンバーである必要があります。
