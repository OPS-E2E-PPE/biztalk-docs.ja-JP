# <a name="how-to-set-receive-pipelines-for-tibco-enterprise-message-service"></a>TIBCO Enterprise Message Service 用の受信パイプラインを設定する方法
Microsoft の BizTalk Adapter for TIBCO Enterprise Message Service では、受信ハンドラーと受信パイプラインを設定することが必要です。  
  
### <a name="to-set-the-pipeline"></a>パイプラインを設定するには  
  
1.  **受信ハンドラー**一覧で、 **BizTalkServerApplication**します。  
  
2.  **受信パイプライン**一覧で、目的のパイプラインを選択します。  
  
3.  **[OK]** をクリックします。
