# <a name="warning---field-data-type-mismatch"></a>警告 - フィールド データ型が一致しません
**エラー コード**  
  
 BEC1002  
  
 **説明**  
  
 **データ型**以外にする対象のノードのプロパティが見つかりました、**データ型**のプロパティ、**フィールド要素**ノードにはされている上位変換先で、対応するプロパティ スキーマです。 スキーマ内のノードのデータ型に割り当てられているデータ型に一致する必要があります、**フィールド要素**プロパティ フィールドを昇格するノードが使用またはには、少なくとも、割り当てられているデータ型が同じ共通言語ランタイム (CLR) 型に割り当てる必要があります。  
  
 **ユーザーの操作**  
  
 変更、**データ型**のノードのプロパティと**フィールド要素**ノードをそれに昇格されて、同じデータ値を入力するか、少なくともデータに値型は同じ CLR データ型にマップします。
