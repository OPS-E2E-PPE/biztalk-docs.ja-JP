# <a name="orchestration-designer-keyboard-shortcuts"></a>オーケストレーション デザイナーのキーボード ショートカット
オーケストレーション デザイナーのデザイン画面では、完全なユーザー補助のキーボードを使用して提供します。 ユーザー補助機能以降では、メニュー オプションを選択して、Visual Studio で行うことができるデザイン画面を開くと、**新規**オプション、**ファイル**メニュー。  
  
 ユーザー補助機能は、ここで、たとえばを選択さまざまな図形と図形の各部を上矢印と下矢印を押して、デザイン サーフェイス内もサポートされます。 デザイン画面で図形を選択すると、強調表示されて、デザイン サーフェイスとオーケストレーションの種類 ウィンドウ (存在する場合)、中に、**プロパティ**グリッドでは、選択した図形に関する情報も表示されます。  
  
 オーケストレーション デザイナーでは、さまざまなキーボード ショートカットを提供することでユーザー補助機能をサポートしています。 さまざまなキーボード ショートカットを使用している現在のデザイン画面のどの部分に応じて利用できます。  
  
 などの BizTalk 式エディターで Microsoft IntelliSense が使用している場合、Visual Studio で使用できる既存のショートカットの多くを使用することもできます。
