# <a name="msmq-adapter"></a>MSMQ アダプター
MSMQ アダプターからのメッセージ キュー 4.0 を使用できます。[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]します。 BizTalk Server とメッセージ キュー テクノロジの統合により、異種ネットワーク間で通信するために、異なる時刻で実行されているアプリケーションとシステムを一時的にオフラインにすることがあります。  
  
 MSMQ アダプターの機能を以下に示します。  
  
-   パフォーマンスの向上  
  
-   マルチ スレッド操作のサポート  
  
-   リモート トランザクションの読み込みのサポートを含むとサブキュー (メッセージ キュー 4.0 で使用可能) からの受信、メッセージ キュー 4.0 機能のサポート。  
  
-   4 MB を超えるメッセージを処理するためのサポート
