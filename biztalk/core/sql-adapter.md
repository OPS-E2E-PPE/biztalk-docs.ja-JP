# <a name="sql-adapter"></a>SQL アダプタ
[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] では SQL アダプターをサポートしなくなりました。 同じ機能を実現するには、[!INCLUDE[adaptersql](../includes/adaptersql-md.md)] を使用することをお勧めします。  
  
 ただし、[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] では、EDI Control メッセージ SQL 受信パイプラインや AS2 再送信制御メッセージ受信パイプラインなどの [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] に同梱されるコンポーネントを使用する SQL アダプターはサポートしています。  
  
 詳細については、次を参照してください。 [Microsoft BizTalk Adapter Pack ヘルプ](http://go.microsoft.com/fwlink/?LinkId=187919)です。
