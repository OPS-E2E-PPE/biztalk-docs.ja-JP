# <a name="error---input-for-mass-copy-functoid-not-valid"></a>エラー - 入力一括コピー Functoid が無効です。
**エラー コード**  
  
 btm1069  
  
 **説明**  
  
 関連する入力パラメーター**一括コピー** functoid が有効でないまたは関連する入力パラメーターが多すぎる**一括コピー** functoid が指定されています。 **一括コピー** functoid が正確に 1 つの入力をいる必要があります: からのリンクを**レコード**送信元スキーマ内のノード。  
  
 **ユーザーの操作**  
  
 再構成、関連する**一括コピー** functoid から 1 つの入力パラメーターがある指定されるように、**レコード**送信元スキーマ内のノード。
