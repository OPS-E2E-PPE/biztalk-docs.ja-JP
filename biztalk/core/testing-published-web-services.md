# <a name="testing-published-web-services"></a>公開済み Web サービスのテスト
Web サービスを開発するとき、Web サービスを再発行してテストしたい場合があります。 Web サービスを再発行するたびにインターネット インフォメーション サービス (IIS) をリセットする必要があります。  
  
 Web サービス クライアント アプリケーションを作成するか、Internet Explorer などのブラウザーを使用して、公開済み Web サービスをテストすることができます。
