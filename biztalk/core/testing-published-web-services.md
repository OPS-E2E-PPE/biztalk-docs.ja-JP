# <a name="testing-published-web-services"></a>公開済み Web サービスのテスト
Web サービスを開発するとき、Web サービスを再公開することによって、Web サービスをテストできます。 Web サービスを再公開する場合は、その都度インターネット インフォメーション サービス (IIS) をリセットする必要があります。  
  
 公開済み Web サービスのテストは、Web サービス クライアント アプリケーションを作成したり、Internet Explorer などのブラウザを使用したりすることで実行できます。
