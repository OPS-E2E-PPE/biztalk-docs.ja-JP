-   [EnvelopeProcessing (BizTalk Server サンプル)](../core/envelopeprocessing-biztalk-server-sample.md)します。 BizTalk Server パイプラインを使用してエンベロープを XML メッセージをフラット ファイル メッセージに変換する方法を示します。
