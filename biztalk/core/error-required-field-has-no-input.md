# <a name="error---required-field-has-no-input"></a>エラー - 必須フィールドに入力がありません。
**エラー コード**  
  
 btm1028  
  
 **説明**  
  
 送信先スキーマでは、対象のノードは必須と指定されていますが、受信リンク、定数値、または既定値がありません。したがって、ノードはこのマップで提供される出力インスタンス メッセージに含められません。  
  
 **ユーザーの操作**  
  
 適宜、送信先スキーマの対象ノードを省略可能に変更するか、受信リンク、定数値、または既定値をそのノードに設定します。
