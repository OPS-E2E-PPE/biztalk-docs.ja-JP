# <a name="single-sign-on-event-10773"></a>シングル サインオン: イベント 10773
## <a name="details"></a>詳細  
  
|                 |                                                                             |
|-----------------|-----------------------------------------------------------------------------|
|  製品名   |                          エンタープライズ シングル サインオン                          |
| 製品バージョン |         [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]          |
|    イベント ID     |                                    10773                                    |
|  イベント ソース   |                                   ENTSSO                                    |
|    コンポーネント    |                                     なし                                     |
|  シンボル名  |                        ENTSSO_E_SSO_MUST_BE_DISABLED                        |
|  メッセージ テキスト   | この機能を実行する前に、エンタープライズ シングル サインオンを無効にする必要があります。 |
  
## <a name="explanation"></a>説明  
 この機能を実行する前に、エンタープライズ シングル サインオンを無効にする必要があります。  
  
## <a name="user-action"></a>ユーザーの操作  
 詳細については、[SSO を無効にする方法](../core/how-to-disable-sso.md)を参照してください。
