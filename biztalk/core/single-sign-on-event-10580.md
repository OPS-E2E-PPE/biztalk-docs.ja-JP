|-----------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
|  製品名   |                                                                   エンタープライズ シングル サインオン                                                                   |

|    イベント ID     |                                                                             10580                                                                             |

## <a name="explanation"></a>説明  
 これは情報メッセージであり、SSO システム内で発生する重要なセキュリティ関連イベントの追跡に役立つことがあります。 このメッセージは、アプリケーションが有効になっていたことを示します。  
  
## <a name="user-action"></a>ユーザーの操作  
 ユーザーによる操作は不要です。
