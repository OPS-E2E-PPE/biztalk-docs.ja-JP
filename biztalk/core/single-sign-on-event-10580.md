# <a name="single-sign-on-event-10580"></a>シングル サインオン:イベント 10580
## <a name="details"></a>詳細  
  
|                 |                                                                                                                                                               |
|-----------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
|  製品名   |                                                                   エンタープライズ シングル サインオン                                                                   |
| 製品バージョン |                                                  [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                                   |
|    イベント ID     |                                                                             10580                                                                             |
|  イベント ソース   |                                                                            ENTSSO                                                                             |
|    コンポーネント    |                                                                              なし                                                                              |
|  シンボル名  |                                                                     SSO_INFO_APP_ENABLED                                                                      |
|  メッセージ テキスト   | アプリケーションが enabled.%r<br /><br /> 追跡 ID: %1 %r<br /><br /> クライアント コンピューターの場合: % 2 %r<br /><br /> クライアント ユーザー: % 3 %r<br /><br /> アプリケーション名: %4 |
  
## <a name="explanation"></a>説明  
 これは、情報メッセージであり、SSO システム内でその発生の重要なセキュリティ関連イベントを追跡するために便利です。 このメッセージは、アプリケーションが有効になっていることを示します。  
  
## <a name="user-action"></a>ユーザーの操作  
 ユーザーによる操作は不要です。
