# <a name="warning---record-body-xpath-not-found"></a>警告 - レコードのボディ XPath が見つかりません。
**エラー コード**  
  
 BEC1008  
  
 **説明**  
  
 **ボディ XPath**空にするこのエンベロープ スキーマに含まれているルート ノードのプロパティが見つかりましたまたは存在しないノードを参照します。 指定したとおり、エンベロープ スキーマ、**エンベロープ**のプロパティ、**スキーマ**ノード、有効な値を設定するために必要な**ボディ XPath**指定されたルートのプロパティスキーマ内のノードを参照します。  
  
 ルート参照ノードが指定されていない場合、**ルート参照**のプロパティ、**スキーマ**ノード、スキーマでは、既定のルート参照ノードの最初のルート ノードがそので有効な値を設定する必要が**ボディ XPath**プロパティです。 **ボディ XPath**プロパティ値をエンベロープ内のメッセージの各スキーマの識別に使用します。  
  
 **ユーザーの操作**  
  
 対象のルート ノードを選択し、値を選択、**ボディ XPath**正しく関連付けられているメッセージ本文のスキーマを識別するプロパティです。
