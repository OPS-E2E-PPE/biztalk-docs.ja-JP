# <a name="single-sign-on-event-10786"></a>シングル サインオン: イベント 10786
## <a name="details"></a>詳細  
  
|                 |                                                                                                         |
|-----------------|---------------------------------------------------------------------------------------------------------|
|  製品名   |                                        エンタープライズ シングル サインオン                                        |
| 製品バージョン |                       [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                        |
|    イベント ID     |                                                  10786                                                  |
|  イベント ソース   |                                                 ENTSSO                                                  |
|    コンポーネント    |                                                   なし                                                   |
|  シンボル名  |                                       ENTSSO_E_ACCOUNT_NOT_VALID                                        |
|  メッセージ テキスト   | アカウント名が無効であるか存在しません。 詳細については、イベント ログ (コンピューター "%1" 上) を参照してください。 |
  
## <a name="explanation"></a>説明  
 アカウント名が無効であるか存在しません。  
  
## <a name="user-action"></a>ユーザーの操作  
 アカウントの名前とスペルを確認し、もう一度やり直してください。
