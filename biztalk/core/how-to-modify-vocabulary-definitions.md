# <a name="how-to-modify-vocabulary-definitions"></a>ボキャブラリの定義を変更する方法
表示名を変更したり、表示名に関連付けられているバインドを変更するのには、ボキャブラリの定義ウィザードを使用して、ボキャブラリの未公開バージョンのボキャブラリの定義を変更できます。  
  
> [!NOTE]
>  すべての要素を変更できます。  
  
 このトピックには、次のタスクの手順が含まれています。  
  
-   ボキャブラリの定義を変更するには  
  
-   ボキャブラリまたはボキャブラリの定義に、ファクトを追加するには  
  
### <a name="to-modify-a-vocabulary-definition"></a>ボキャブラリの定義を変更するには  
  
1.  ボキャブラリの定義を右クリックし、をクリックし、**変更**します。  
  
2.  新しいボキャブラリの定義を作成すると、ボキャブラリの定義ウィザードを使用します。  
  
### <a name="to-add-a-fact-to-a-vocabulary-or-vocabulary-definition"></a>ボキャブラリまたはボキャブラリの定義に、ファクトを追加するには  
  
1.  [ファクト エクスプ ローラー] ウィンドウ、**ボキャブラリ**タブをクリックし、更新する未公開のボキャブラリのバージョンを選択します。  
  
2.  クリックして**データベース**、 **XML スキーマ**、または **.NET クラス**します。  
  
3.  必要なファクトにデータのソース階層内を移動します。  
  
    > [!NOTE]
    >  実際には、データベース テーブルまたはデータベース テーブルの列、XML ドキュメントの要素または属性、または、.NET クラスまたはクラス メンバーができます。  
  
4.  ファクトをクリックし、マウス ボタンを押したままです。  
  
5.  経由で、ファクトをドラッグして、**ボキャブラリ**タブ。  
  
     **ボキャブラリ**タブが表示されます。  
  
6.  ボキャブラリのバージョンを変更するのには、という事実をドラッグします。
