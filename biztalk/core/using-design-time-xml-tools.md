# <a name="using-design-time-xml-tools"></a>デザイン時 XML ツールの使用
EDI XML ツールを使用すると、EDI のスキーマとマップに対してデザイン時の操作を実行できます。 これらのツールは、Visual Studio のソリューション エクスプローラー ペインでスキーマまたはマップを右クリックしたときに表示されるメニューを使用して実行します。
