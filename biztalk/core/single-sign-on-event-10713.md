# <a name="single-sign-on-event-10713"></a>シングル サインオン:イベント 10713
## <a name="details"></a>詳細  

|                 |                                                                                                                                           |
|-----------------|-------------------------------------------------------------------------------------------------------------------------------------------|
|  製品名   |                                                         エンタープライズ シングル サインオン                                                         |
| 製品バージョン |                                        [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                         |
|    イベント ID     |                                                                   10713                                                                   |
|  イベント ソース   |                                                                  ENTSSO                                                                   |
|    コンポーネント    |                                                                    該当なし                                                                    |
|  シンボル名  |                                                   SSO_INFO_PS_DUPLICATE_EXTERNAL_CHANGE                                                   |
