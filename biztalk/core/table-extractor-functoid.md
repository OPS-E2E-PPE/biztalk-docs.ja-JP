# <a name="table-extractor-functoid"></a>テーブル抽出 Functoid
**テーブル抽出**functoid によってときにルーピング グリッドの各行から抽出されるデータを**テーブル ループ**functoid が行を表示します。 必要な**テーブル抽出**functoid が各出力フィールドです。 たとえば、入力インスタンス メッセージでは、1 つの形式でアドレスが必要があるが、出力インスタンス メッセージの各形式のタグ付けされた 2 つの形式のアドレスが必要とします。 必要がありますが、**テーブル抽出**functoid、タグおよびアドレスの各要素。 構成の詳細については、**テーブル抽出**functoid を参照してください[Table-Driven ループの構成](../core/table-driven-looping-configuration.md)します。 参照してください[Table-Driven ループ例](../core/table-driven-looping-example.md)します。
