# <a name="building-and-running-orchestrations"></a>ビルドとオーケストレーションの実行
設計したオーケストレーションをコンパイルして実行することにより、その完成度と正確さをテストできます。  
  
 オーケストレーションをコンパイルしてエラーが発生しなければ、それをテスト環境に展開してから必要な修正を加え、最終的に、実稼働環境に展開することになります。
