# <a name="business-activity-monitoring-biztalk-server-samples-folder"></a>ビジネス アクティビティ監視 (BizTalk Server Samples フォルダー)
Microsoft BizTalk Server には、ソフトウェア開発キット (SDK) のいくつかのビジネス アクティビティ監視 (BAM) のサンプルが含まれています。 ここでは、各 BAM のサンプルを構築して、サンプルと予想される結果を実行する手順で示されている機能の詳細を示します。  
  
## <a name="in-this-section"></a>このセクションの内容  
  
-   [BAM API (BizTalk Server サンプル)](../core/bam-api-biztalk-server-sample.md)します。 について説明する方法。  
  
    -   ビジネス アナリストは、ウィザードを使用して、関心のあるデータを定義します、集計を調べ、およびアクティビティ データを照会します。  
  
    -   情報技術 (IT) 管理者は、SQL、データ変換サービス (DTS)、およびオンライン分析処理 (OLAP) インフラストラクチャを作成します。  
  
    -   開発者は、ビジネスの実装をインストルメント化します。  
  
-   [BAM エンド ツー エンド (BizTalk Server サンプル)](../core/bam-end-to-end-biztalk-server-sample.md)します。 BAM を使用して複数のコンポーネントからイベントを関連付ける方法を示します。  
  
-   [BAM API (BizTalk Server サンプル) オーケストレーション式から](../core/bam-api-from-an-orchestration-expression-biztalk-server-sample.md)します。 について説明する方法。  
  
    -   BizTalk Server オーケストレーションから BAM API を使用します。  
  
    -   個別のアクティビティ インスタンスとしてメッセージ内の繰り返し項目を追跡します。  
  
    -   追跡プロファイルと、API から追跡データを使用してオーケストレーションから追跡した BAM データ間の関係を確立します。
