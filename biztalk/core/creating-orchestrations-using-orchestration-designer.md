# <a name="creating-orchestrations-using-orchestration-designer"></a>オーケストレーション デザイナーを使用してオーケストレーションの作成
[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] ワークフローでのオーケストレーション  
  
 ![BizTalk Server のオーケストレーション](../core/media/ebiz-dev-busprcsf.gif "ebiz_dev_busprcsf")  
  
 BizTalk オーケストレーション デザイナーは、ビジネス プロセスの図式を簡単に作成できるツールです。作成した図式は、自動的に基本コードに反映され、また実行可能モジュールに組み込むことができます。 BizTalk オーケストレーション デザイナーでは、実行する各種アクションに対応したさまざまな図形を利用できます。  
  
 ここでは、BizTalk オーケストレーションおよび BizTalk オーケストレーション デザイナーの理解と使用に役立つ概要情報、リファレンス情報、およびタスク関連の情報について説明します。
