# <a name="single-sign-on-event-11067"></a>シングル サインオン: イベント 11067
## <a name="details"></a>詳細  
  
|                 |                                                                                     |
|-----------------|-------------------------------------------------------------------------------------|
|  製品名   |                              エンタープライズ シングル サインオン                              |
| 製品バージョン |             [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]              |
|    イベント ID     |                                        11067                                        |
|  イベント ソース   |                                       ENTSSO                                        |
|    コンポーネント    |                                         なし                                         |
|  シンボル名  |                      SSO_ERROR_PASSWORD_SYNC_MIIS_START_FAILED                      |
|  メッセージ テキスト   | Windows のパスワード同期を (MIIS から) 開始できませんでした。%r<br /><br /> エラー コード: %1 |
  
## <a name="explanation"></a>説明  
 これは、Windows のパスワード同期を (MIIS から) 開始できなかったことを示す情報メッセージです。  
  
## <a name="user-action"></a>ユーザーの操作  
 ユーザーによる操作は不要です。
