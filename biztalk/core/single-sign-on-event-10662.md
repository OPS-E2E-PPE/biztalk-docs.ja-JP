# <a name="single-sign-on-event-10662"></a>シングル サインオン:イベント 10662
## <a name="details"></a>詳細  

|                 |                                                                                                                                                                                   |
|-----------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|  製品名   |                                                                             エンタープライズ シングル サインオン                                                                             |
| 製品バージョン |                                                            [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                                             |
|    イベント ID     |                                                                                       10662                                                                                       |
|  イベント ソース   |                                                                                      ENTSSO                                                                                       |
|    コンポーネント    |                                                                                        該当なし                                                                                        |
|  シンボル名  |                                                                     SSO_INFO_WINDOWS_PASSWORD_CHANGE_RECEIVED                                                                     |
|  メッセージ テキスト   | PCNS.%r から Windows パスワードの変更を受け取りました<br /><br /> 追跡 ID: %1 %r<br /><br /> Windows アカウント: % 2 %r<br /><br /> 追加データ: % 3 %r<br /><br /> クライアント ユーザー: %4 |

## <a name="explanation"></a>説明  
 この情報イベントは、パスワード変更通知サービスから Windows パスワードの変更を受け取ったことを示します。  

## <a name="user-action"></a>ユーザーの操作  

-   ユーザー操作は必要ありません。
