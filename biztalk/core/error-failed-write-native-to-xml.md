# <a name="error---failed-write-native-to-xml"></a>エラー - XML へのネイティブな書き込みに失敗しました
**エラー コード**  
  
 btm1043  
  
 **説明**  
  
 XML ファイルの書き込み中に、ファイルの書き込みエラーが発生しました。  
  
 **ユーザーの操作**  
  
 ディスクに十分な空き領域があることと、表示されたファイルを作成できるアクセス許可があることを確認します。
