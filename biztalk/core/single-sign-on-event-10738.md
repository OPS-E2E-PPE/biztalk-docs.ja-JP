# <a name="single-sign-on-event-10738"></a>シングル サインオン:イベント 10738
## <a name="details"></a>詳細  

|                 |                                                                                                                                                                  |
|-----------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|  製品名   |                                                                    エンタープライズ シングル サインオン                                                                     |
| 製品バージョン |                                                    [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                                    |
|    イベント ID     |                                                                              10738                                                                               |
|  イベント ソース   |                                                                              ENTSSO                                                                              |
|    コンポーネント    |                                                                               該当なし                                                                                |
|  シンボル名  |                                                             SSO_INFO_PS_SET_WINDOWS_PASSWORD_ADAPTER                                                             |
|  メッセージ テキスト   | Windows パスワードは SSO database.%r で正常に更新されました<br /><br /> 追跡 ID: %1 %r<br /><br /> アダプター: % 2 %r<br /><br /> Windows アカウント: %3\\%4 |

## <a name="explanation"></a>説明  
 この情報イベントは、Windows パスワードが SSO データベースに正常に更新されたことを示します。

- ユーザー操作は必要ありません。
