# <a name="receivelocations-receiveport-node"></a>Receivelocation (ReceivePort ノード)
バインド ファイルの ReceivePort ノードの ReceiveLocations ノードは、すべてのバインド ファイルと共にエクスポートされる受信場所に関する特定の情報を含む ReceiveLocation ノードの親ノードです。  
  
## <a name="nodes-in-the-receivelocations-node"></a>ReceiveLocations ノード内のノード  
 次の表に、バインド ファイルのこのノードに設定できるプロパティを示します。  
  
|**名前**|**ノードの種類**|**[データ型]**|**[説明]**|**制限**|**コメント**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[ReceiveLocation](../core/receivelocation-receivelocations-node.md)|レコード|ReceiveLocation (ComplexType)|バインド ファイルと共にエクスポートされる受信場所に関する情報を指定します。|任意|既定値: なし|
