# <a name="single-sign-on-event-10852"></a>シングル サインオン: イベント 10852
## <a name="details"></a>詳細  
  
|||  
|-|-|  
|製品名|エンタープライズ シングル サインオン|  
|製品バージョン|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|イベント ID|10852|  
|イベント ソース|ENTSSO|  
|コンポーネント|なし|  
|シンボル名|ENTSSO_E_CREATE_FIELD_NO_DIRECT_PASSWORD_SYNC|  
|メッセージ テキスト|このアプリケーションは "直接パスワード同期" フラグが設定されているため、フィールドを作成できません。|  
  
## <a name="explanation"></a>説明  
 アプリケーションの直接パスワード同期フラグが設定されていると、フィールドを作成できません。  
  
## <a name="user-action"></a>ユーザーの操作  
 ユーザーが実行できる操作はありません。 このオプションはサポートされていません。
