# <a name="pipeline-designer-keyboard-shortcuts"></a>パイプライン デザイナーのキーボード ショートカット
マウスの代わりにキーボードを使用して、デザイン画面を操作できます。 次の表に、使用できるキーボード ショートカットを示します。  
  
|キーボード ショートカット|結果|  
|-----------------------|------------|  
|下方向キー|含む、次の区分線または図形の下にカーソルを移動**開始**と**終了**図形です。|  
|↑|など、前の区分線または図形の上にカーソルを移動**開始**と**終了**図形です。|  
|PageUp|ステージの前の開始図形にカーソルのフォーカスを移動します。 このキーは、ページを上方向にスクロールして、パイプラインの前の部分を表示します。|  
|PageDown|ステージの次の開始図形にカーソルのフォーカスを移動します。 このキーは、ページを下方向にスクロールして、パイプラインの次の部分を表示します。|  
|Home|フォーカスを**開始**パイプラインの形状。|  
|END|フォーカスを**終了**パイプラインの形状。|
