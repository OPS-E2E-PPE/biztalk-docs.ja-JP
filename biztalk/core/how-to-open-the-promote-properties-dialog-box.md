# <a name="how-to-open-the-promote-properties-dialog-box"></a>開く方法、昇格のプロパティ ダイアログ ボックス
プロパティの昇格がで管理されている、**プロパティの昇格** ダイアログ ボックス。 このトピックでは、昇格させたプロパティを管理するこのダイアログ ボックスを開くための手順が説明します。  
  
> [!NOTE]
>  使用すると、**クイック昇格**機能、BizTalk エディターの新しい追加**プロパティ フィールド**プロモーションを開かなくても、**プロパティの昇格**ダイアログ ボックスで、プロモーションはときに表示されます ダイアログ ボックスで、次へ が開かれます。  
  
 プロパティの昇格 ダイアログ ボックスを開くの 2 つの方法はあります。  
  
### <a name="to-open-the-promote-properties-dialog-box-from-the-schema-node"></a>スキーマ ノードからプロパティの昇格 ダイアログ ボックスを開く  
  
1.  スキーマ ツリーで、選択、 **\<スキーマ\>** ノード。  
  
2.  [プロパティ] ウィンドウで、省略記号ボタンをクリックします ( **...** ) の値フィールドのボタン、**プロパティの昇格**プロパティ。  
  
     **プロパティの昇格** ダイアログ ボックスが表示されます。  
  
### <a name="to-open-the-promote-properties-dialog-box-from-any-node"></a>任意のノードからプロパティの昇格 ダイアログ ボックスを開く  
  
1.  スキーマ ツリーで、任意のノードを選択します。  
  
2.  選択したノードを右クリックし、をクリックして**昇格**、] をクリックし、 **[昇格の表示**  
  
     **プロパティの昇格** ダイアログ ボックスが表示されます。
