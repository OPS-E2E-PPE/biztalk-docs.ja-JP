# <a name="error---multiple-loop-paths"></a>エラー - 複数のループ パス
**エラー コード**  
  
 btm1030  
  
 **説明**  
  
 送信先スキーマのノードに、複数の送信元ループ パスがあります。  
  
 **ユーザーの操作**  
  
 使用して、**ループ**functoid を明示的にする、ループの実行、送信元スキーマ内のノードを指定します。
