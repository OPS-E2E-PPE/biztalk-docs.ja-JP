### <a name="to-save-a-vocabulary-version"></a>ボキャブラリのバージョンを保存するには  
  
-   バージョンを右クリックし、クリックして**保存**します。  
  
### <a name="to-publish-a-vocabulary-version"></a>ボキャブラリのバージョンを発行するには  
  
-   バージョンを右クリックし、クリックして**発行**します。  
  
### <a name="to-create-an-updated-version-of-a-vocabulary"></a>ボキャブラリの更新バージョンを作成するには  
  
1.  既存のバージョンを右クリックし、をクリックし、**コピー**します。  
  
2.  ボキャブラリ フォルダーを右クリックし、**貼り付け**します。  
  
     コピーされるバージョンと同じ要素を新しいバージョンが作成されます。  
  
### <a name="to-create-an-empty-new-version-of-a-vocabulary"></a>ボキャブラリの空の新しいバージョンを作成するには  
  
-   ボキャブラリ フォルダーを右クリックし、**新しいバージョンの追加**します。
