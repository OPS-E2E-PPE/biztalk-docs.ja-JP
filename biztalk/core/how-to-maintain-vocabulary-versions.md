# <a name="how-to-maintain-vocabulary-versions"></a>ボキャブラリのバージョンを維持する方法
ボキャブラリの定義をボキャブラリのバージョンに追加することにより、将来の開発に備えて特定のバージョンをルール ストアに保存できます。また、そのバージョンを公開して、厳密に定義されたデータ関連の用語のセットを作成し、ポリシーを作成する際に、これらの用語をルールに追加することができます。 ただし、ボキャブラリをバージョンに追加したとしても、既存のルールでは、古いバージョンが参照される点に注意してください。  
  
 このトピックでは、次の操作の手順について説明します。  
  
-   ボキャブラリのバージョンを保存するには  
  
-   ボキャブラリのバージョンを公開するには  
  
-   既存のボキャブラリ バージョンをコピーするには  
  
-   ボキャブラリの空のバージョンを新規作成するには  
  
### <a name="to-save-a-vocabulary-version"></a>ボキャブラリのバージョンを保存するには  
  
-   バージョンを右クリックし、をクリックして**保存**です。  
  
### <a name="to-publish-a-vocabulary-version"></a>ボキャブラリのバージョンを公開するには  
  
-   バージョンを右クリックし、をクリックして**発行**です。  
  
### <a name="to-create-an-updated-version-of-a-vocabulary"></a>既存のボキャブラリ バージョンをコピーするには  
  
1.  既存のバージョンを右クリックし、をクリックして**コピー**です。  
  
2.  ボキャブラリ フォルダーを右クリックし、をクリックして**貼り付け**です。  
  
     コピー元と同じ要素を持つ新しいバージョンが作成されます。  
  
### <a name="to-create-an-empty-new-version-of-a-vocabulary"></a>ボキャブラリの空のバージョンを新規作成するには  
  
-   ボキャブラリ フォルダーを右クリックし、をクリックして**新しいバージョンの追加**です。  
  
    > [!NOTE]
    >  公開されたボキャブラリの定義しか使用できません。
