# <a name="single-sign-on-event-11046"></a>シングル サインオン: イベント 11046
## <a name="details"></a>詳細  
  
|||  
|-|-|  
|製品名|エンタープライズ シングル サインオン|  
|製品バージョン|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|イベント ID|11046|  
|イベント ソース|ENTSSO|  
|コンポーネント|なし|  
|シンボル名|SSO_WARN_EXISTING_MAPPING_EXTERNAL|  
|メッセージ テキスト|指定された外部アカウントには既にマッピングが存在するため、マッピングを作成できませんでした。%r<br /><br /> 追跡 ID: %1 %r<br /><br /> アプリケーション名: %2 %r<br /><br /> 外部アカウント: %3|  
  
## <a name="explanation"></a>説明  
 指定された外部アカウントにはマッピングが既に存在しており、重複するマッピングは使用できません。  
  
## <a name="user-action"></a>ユーザーの操作  
 ユーザーの操作は必要ありません。
