# <a name="how-to-set-pipeline-component-properties"></a>パイプライン コンポーネントのプロパティを設定する方法
各コンポーネントには、使用する前に構成することが必要なプロパティがあります。  
  
### <a name="to-set-pipeline-component-properties"></a>パイプライン コンポーネントのプロパティを設定するには  
  
-   [プロパティ] ウィンドウで、**パイプライン コンポーネントのプロパティ**セクションで、をクリックして、**値**セル、**プロパティ**行、および新しい値を入力します。 場合、**値**セルには、矢印ボタンが含まれている、選択肢の一覧のボタンをクリックします。 場合、**値**セルには、省略記号が含まれています (**.**) ボタンをクリックし、スキーマ コレクション プロパティ エディターを使用して、プロパティを構成するボタンをクリックします。 詳細については、次を参照してください。[スキーマ コレクション プロパティ エディターを使用する方法](../core/how-to-use-the-schema-collection-property-editor.md)です。
