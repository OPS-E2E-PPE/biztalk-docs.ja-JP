# <a name="synchronizing-passwords"></a>パスワードの同期
パスワードの同期は、パスワード同期アダプターを使用して行います。 このアダプターには、特定の Windows 以外のリモート システムと通信するための機能や、そのシステムにパスワード情報の更新を指示するための機能が必要です。
