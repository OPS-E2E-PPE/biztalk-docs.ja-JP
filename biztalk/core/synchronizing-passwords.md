# <a name="synchronizing-passwords"></a>パスワードの同期
パスワードを同期するには、パスワード同期アダプターを使用します。 このアダプターは、Windows 以外のシステムに固有で、リモート通信できる必要があり、パスワード情報を更新するには、そのシステムに指示することもあります。
