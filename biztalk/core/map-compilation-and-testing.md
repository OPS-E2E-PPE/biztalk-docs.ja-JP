# <a name="map-compilation-and-testing"></a>マップのコンパイルとテスト
BizTalk マッパーを使用すると、マップを検証してエラーや警告の状況を確認できます。 入力インスタンス データに対してマップをテストすることもできます。  
  
 マップやスキーマのプロパティ設定は、検証およびテストに大きな影響を与えます。 このセクションの各トピックでは、これらのプロパティについて取り上げます。
