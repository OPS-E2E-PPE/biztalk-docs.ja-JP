# <a name="single-sign-on-event-10501"></a>シングル サインオン: イベント 10501
## <a name="details"></a>詳細  

|                 |                                                                                                                                                                                                                      |
|-----------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|  製品名   |                                                                                              エンタープライズ シングル サインオン                                                                                               |
| 製品バージョン |                                                                              [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                                                              |
|    イベント ID     |                                                                                                        10501                                                                                                         |
|  イベント ソース   |                                                                                                        ENTSSO                                                                                                        |
|    コンポーネント    |                                                                                                         N\A                                                                                                          |
|  シンボル名  |                                                                                              SSO_INFO_SERVICE_STARTING                                                                                               |
|  メッセージ テキスト   | SSO サービスが開始しました。%r<br /><br /> コンピューター名: %1 %r<br /><br /> SQL Server 名: % 2 %r<br /><br /> SSO データベース名: % 3 %r<br /><br /> SSL を使用しています。 SQL Server の接続には、プロトコルの暗号化が設定されます。 |

## <a name="explanation"></a>説明  
 この情報イベントは、示された構成プロパティを使用して SSO サービスを起動していることを示します。  

## <a name="user-action"></a>ユーザーの操作  
 このエラー イベントを解決するには、次の手順を実行します。  

- 必要なアクションはありません。
