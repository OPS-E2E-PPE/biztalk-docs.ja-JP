# <a name="single-sign-on-event-10501"></a>シングル サインオン:イベント 10501
## <a name="details"></a>詳細  

|                 |                                                                                                                                                                                                                      |
|-----------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|  製品名   |                                                                                              エンタープライズ シングル サインオン                                                                                               |
| 製品バージョン |                                                                              [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                                                              |
|    イベント ID     |                                                                                                        10501                                                                                                         |
|  イベント ソース   |                                                                                                        ENTSSO                                                                                                        |
|    コンポーネント    |                                                                                                         該当なし                                                                                                          |
|  シンボル名  |                                                                                              SSO_INFO_SERVICE_STARTING                                                                                               |
|  メッセージ テキスト   | SSO サービスが starting.%r<br /><br /> コンピューター名: %1 %r<br /><br /> SQL Server 名: % 2 %r<br /><br /> SSO データベース名: % 3 %r<br /><br /> SSL を使用します。 プロトコルの暗号化は SQL Server の接続に適用します。 |

## <a name="explanation"></a>説明  
 この情報イベントは、示された構成プロパティで、SSO サービスが開始されていることを示します。  

## <a name="user-action"></a>ユーザーの操作  
 このエラー イベントを解決するには、次の操作を行います。  

- 必要なアクションはありません。
