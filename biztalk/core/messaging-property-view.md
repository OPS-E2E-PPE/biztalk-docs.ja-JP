# <a name="messaging-property-view"></a>メッセージング プロパティ ビュー
メッセージング プロパティ ビューには、プロパティに関連付けられた XML メッセージのスキーマが表示されます。 ビューは、一部の図形 オーケストレーション スケジュール ビューのショートカット メニューから利用できます。  
  
## <a name="working-with-messaging-properties"></a>メッセージング プロパティの使用  
 クリックして、メッセージング プロパティ ビューを選択、**イベント ソースの選択**クリックし、 **Select Message Property**メニュー項目。 スキーマを選択するメッセージ プロパティを選択します。 スキーマを選択するとする要素をドラッグできますこのスキーマからアクティビティのデータ項目フォルダーにこのアクションのメッセージ内の特定の XPath 式から特定のデータを抽出することを示します。  
  
 オーケストレーション スケジュール ビューからメッセージ プロパティのビューを開くには、メッセージ ペイロードを含む図形を右クリックします。 これにより、図形に関連付けられているペイロードの一覧を取得するメッセージのペイロードのメニュー項目をクリックすることができます、コンテキスト メニューが開きます。  
  
 使用可能なメッセージ プロパティの一覧は広範にすることはできます。 検索対象のオーケストレーションの名前の一部がわかっている場合でを入力、**文字列**テキスト ボックスをクリックして、**検索**ボタンをクリックします。 これにより、入力した部分文字列が含まれるメッセージのプロパティのみが選択されます。
