## <a name="explanation"></a>説明  
 SSO 関連管理者アカウントがグローバル情報アダプターに対して無効です。  
  
## <a name="user-action"></a>ユーザーの操作  
 警告メッセージには、SSO 関連管理者アカウントおよび無効なアカウントに関する情報が含まれます。 この情報を確認し、必要に応じて修正を行い、もう一度更新を試行します。
