# <a name="error---map-without-any-links-or-constants"></a>エラー - マップにリンクまたは定数
**エラー コード**  
  
 btm1000  
  
 **説明**  
  
 マップにスキーマ間のリンクがまったく指定されていないか、送信先スキーマに定数値が指定されていません。 そのため、このマップから出力を作成できません。  
  
 **ユーザーの操作**  
  
 適切なリンクを追加し、適宜、マップのグリッドに Functoid を追加するか、送信先スキーマに定数を追加して、送信元スキーマに準拠しているインスタンス メッセージを送信先スキーマに準拠したインスタンス メッセージに変換する方法を指定します。
