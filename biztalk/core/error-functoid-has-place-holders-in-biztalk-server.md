# <a name="error--the-quot0quot-functoid-has-place-holders-replace-these-with-appropriate-links-or-constants"></a>エラー、 &quot;{0}&quot; functoid がプレース ホルダーです。 適切なリンクまたは定数で置き換えてください。
**エラー コード**  
  
 btm1090  
  
 **ユーザーの操作**  
  
 プレースホルダー リンクが設定されている Functoid には、UI に警告タグが設定されます。 プレースホルダー リンクは、適切なリンクまたは定数に置き換える必要があります。
