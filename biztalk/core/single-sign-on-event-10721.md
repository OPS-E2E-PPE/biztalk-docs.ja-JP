# <a name="single-sign-on-event-10721"></a>シングル サインオン:イベント 10721
## <a name="details"></a>詳細  

|                 |                                                                                 |
|-----------------|---------------------------------------------------------------------------------|
|  製品名   |                            エンタープライズ シングル サインオン                            |
| 製品バージョン |           [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]            |
|    イベント ID     |                                      10721                                      |
|  イベント ソース   |                                     ENTSSO                                      |
|    コンポーネント    |                                       該当なし                                       |
|  シンボル名  |                        SSO_INFO_REPLAY_FILE_CLOSED_FULL                         |
|  メッセージ テキスト   | Full.%r のため、再生ファイルは閉じられました<br /><br /> ファイル名: %1 |

## <a name="explanation"></a>説明  
 この情報イベントは、いっぱいのために、再生ファイルが閉じられたことを示します。 必要に応じて、新しい再生ファイルが作成されます。  

## <a name="user-action"></a>ユーザーの操作  

- ユーザー操作は必要ありません。
