# <a name="error---exception-caught"></a>エラー - 例外がキャッチされました
**エラー コード**  
  
 btm1023  
  
 **説明**  
  
 例外が発生しました。マップの検証またはテスト中に、致命的なエラーが発生しました。  
  
 **ユーザーの操作**  
  
 対象の例外に応じて修正操作を行ってください。
