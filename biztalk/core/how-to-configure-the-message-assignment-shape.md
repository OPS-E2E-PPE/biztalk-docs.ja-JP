# <a name="how-to-configure-the-message-assignment-shape"></a>メッセージの割り当て図形を構成する方法
![](../core/media/ebiz-orch-assign.gif "ebiz_orch_assign")  
メッセージの割り当て図形  
  
> [!NOTE]
>  A**メッセージの割り当て**図形を配置できる内でのみ、**メッセージの構築**図形。 デザイン サーフェイスでは、メッセージの割り当てをその他の場所にドラッグした場合、新しい**メッセージの構築**図形が作成されます。  
  
### <a name="to-configure-a-message-assignment-shape"></a>メッセージの割り当て図形を構成するには  
  
1.  BizTalk 式エディターが表示されていない場合を右クリックし、**メッセージの割り当て**図形し、クリックして**式の編集**または、[プロパティ] ウィンドウで、**省略記号**(**...**) ボタンを**式**プロパティ。  
  
2.  BizTalk 式エディターでは、値を代入する式を作成します。 詳細については、次を参照してください。[式の要件と制限](../core/requirements-and-limitations-for-expressions.md)します。
