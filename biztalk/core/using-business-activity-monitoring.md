# <a name="using-business-activity-monitoring"></a>ビジネス アクティビティの使用の監視
ビジネス アクティビティ監視 (BAM) は、異種 IT 実装と、IT 環境の独立したビジネス プロセスの可視性を提供します。 これにより、このようなメッセージとインスタンスの監視の監視操作と BAM が区別されます。 BAM の重要な側面は、ビジネス データをリアルタイムに近いと呼ばれる待機時間ゼロを表示できることです。  
  
 開始する前に知っておくべき情報で[ビジネス アクティビティ監視](../core/business-activity-monitoring.md)します。
