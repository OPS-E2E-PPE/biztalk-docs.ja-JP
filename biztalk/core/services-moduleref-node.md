# <a name="services-moduleref-node"></a>Services (ModuleRef ノード)
バインド ファイルの Services ノードには、バインド ファイルと共にエクスポートされる各サービスに関する特定の情報が含まれています。  
  
## <a name="nodes-in-the-services-node"></a>[サービス] ノード内のノード  
 次の表に、バインド ファイルのこのノードに設定できるプロパティを示します。  
  
|**名前**|**ノードの種類**|**[データ型]**|**[説明]**|**制限**|**コメント**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[サービス](../core/service-services-node.md)|レコード|ServiceRef (ComplexType)|このサービス説明ノードのコンテナー ノードです。|任意|既定値:なし|
