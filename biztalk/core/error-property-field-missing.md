# <a name="error---property-field-missing"></a>エラー - プロパティ フィールドが見つかりません
**エラー コード**  
  
 BEC2008  
  
 **説明**  
  
 このスキーマのノードに昇格されて、**フィールド要素**対応するプロパティ スキーマが存在しないノード。 この条件が発生することがときに、**フィールド要素**ノードを削除、またはプロパティの昇格の対象として使用された後、プロパティ スキーマの名前に変更します。  
  
 **ユーザーの操作**  
  
 プロパティ スキーマを変更か、存在しないように**フィールド要素**ノード、または使用して、**プロパティの昇格** ダイアログ ボックスを削除またはそれ以外の場合、関連するプロパティの昇格を変更します。
