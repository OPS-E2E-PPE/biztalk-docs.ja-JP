# <a name="single-sign-on-event-11056"></a>シングル サインオン:イベント 11056
## <a name="details"></a>詳細  
  
|                 |                                                                                                                                                                                                                                                                                                                          |
|-----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|  製品名   |                                                                                                                                                エンタープライズ シングル サインオン                                                                                                                                                 |
| 製品バージョン |                                                                                                                                [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                                                                                                                |
|    イベント ID     |                                                                                                                                                          11056                                                                                                                                                           |
|  イベント ソース   |                                                                                                                                                          ENTSSO                                                                                                                                                          |
|    コンポーネント    |                                                                                                                                                           なし                                                                                                                                                            |
|  シンボル名  |                                                                                                                                           SSO_WARN_PS_DIRECT_GET_CREDS_FAILED                                                                                                                                            |
|  メッセージ テキスト   | パスワードが直接変更、外部アカウントのないため、既存の外部資格情報を SSO database.%r から取得できませんでした。<br /><br /> 追跡 ID: %1 %r<br /><br /> アプリケーション名: % 2 %r<br /><br /> Windows アカウント: % 3 %r<br /><br /> 外部アカウント: % 4 %r<br /><br /> エラー コード: %5 |
  
## <a name="explanation"></a>説明  
 パスワードが直接変更されません、外部アカウントのため、既存の外部資格情報を SSO データベースから取得できませんでした。  
  
## <a name="user-action"></a>ユーザーの操作  
 提供される情報を使用し、マイクロソフト製品サポート サービスにお問い合わせください。
