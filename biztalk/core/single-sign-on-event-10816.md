|                 |                                                               |
|-----------------|---------------------------------------------------------------|
|  製品名   |                   エンタープライズ シングル サインオン                   |
| 製品バージョン |  [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]   |
|    イベント ID     |                             10816                             |
|  イベント ソース   |                            ENTSSO                             |
|    コンポーネント    |                              なし                              |
|  シンボル名  |          ENTSSO_E_EXTERNAL_UPDATE_FAILED_MORE_RECENT          |
|  メッセージ テキスト   | SSO データベース内の外部資格情報が最新です。 |
  
## <a name="explanation"></a>説明
