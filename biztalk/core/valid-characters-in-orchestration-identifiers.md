# <a name="valid-characters-in-orchestration-identifiers"></a>オーケストレーション識別子で有効な文字
項目、変数、型名などのオーケストレーション識別子にはいくつかの制約があります。  
  
-   識別子の最初の文字は英字またはアンダースコア文字 (_) でなければなりません。  
  
-   後続の文字には、英数字、接続文字、組み合わせ文字、または書式設定文字を使用できます。  
  
-   オーケストレーション識別子ではほとんどの句読点文字を使用できません。
