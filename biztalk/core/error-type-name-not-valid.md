# <a name="error---type-name-not-valid"></a>エラー - 型名が有効ではありません。
**エラー コード**  
  
 BEC2011  
  
 **説明**  
  
 **型名**このスキーマ ファイルのプロパティが無効です。 の値、**型名**プロパティは、自動的に生成された c# クラス名の名前として使用する必要がある必要があります、有効な c# 識別子予約済みの BizTalk キーワードをすることはできません。  
  
 **ユーザーの操作**  
  
 Microsoft® で関連するスキーマ ファイルを選択[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]ソリューション エクスプ ローラーで、し、[プロパティ] ウィンドウで次のように変更します。、**型名**プロパティ値が有効な c# 識別子は、予約済みの BizTalk キーワードではありません。
