# <a name="checklist-undeploy-a-biztalk-application"></a>チェックリスト:BizTalk アプリケーションの展開解除します。

|手順|リファレンス|  
|----------|---------------|  
|BizTalk グループから BizTalk アプリケーションを削除します。 このタスクでは、BizTalk 管理コンソールと BizTalk Server データベースからアプリケーションを削除します。|[BizTalk グループから BizTalk アプリケーションを削除する方法](../core/how-to-delete-a-biztalk-application-from-the-biztalk-group.md)|  
|BizTalk アプリケーションがインストールされているすべてのコンピューターからアンインストールします。|[BizTalk アプリケーションをアンインストールする方法](../core/how-to-uninstall-a-biztalk-application.md)|  
|必要に応じて、グローバル アセンブリ キャッシュ (GAC) からアプリケーションに読み込まれたアセンブリも削除します。|[GAC からアセンブリをアンインストールする方法](http://msdn.microsoft.com/library/464706a8-f902-4d05-a724-19169facd2b4)|  
|上記の手順で自動的に削除されなかったその他のファイルや設定を、ローカル コンピューターから削除します。|[その他のファイルと BizTalk アプリケーションの設定を削除する方法](../core/how-to-remove-other-files-and-settings-for-a-biztalk-application.md)|
