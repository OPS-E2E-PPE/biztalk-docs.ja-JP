# <a name="configuring-the-wcf-adapter-to-intercept-bam-data"></a>BAM データを受信するための WCF アダプタの構成
このセクションでは、BAM WCF インターセプタと連携するように BizTalk WCF アダプタを構成するために必要な手順について説明します。  
  
 基本的な手順は次のとおりです。  
  
-   BAM の動作を machine.config ファイルに追加します。  
  
-   BizTalk WCF アダプタを作成します。  
  
-   BAM WCF インターセプタを構成します。
