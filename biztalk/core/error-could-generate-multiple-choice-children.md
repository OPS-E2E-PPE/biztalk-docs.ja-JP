# <a name="error---could-generate-multiple-choice-children"></a>エラー - が複数の Choice 子要素を生成
**エラー コード**  
  
 btm1027  
  
 **説明**  
  
 送信先スキーマへのリンクには内で複数のノードに不適切な**選択肢**グループ ノードを生成します。  
  
 **ユーザーの操作**  
  
 そのため、送信先スキーマへのリンクを修正内で 1 つのノードだけである、**選択肢**グループ ノードを生成することができます。
