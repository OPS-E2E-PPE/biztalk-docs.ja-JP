# <a name="success---test-map"></a>成功 - マップのテスト
**エラー コード**  
  
 btm1053  
  
 **説明**  
  
 マップに対するマップのテスト操作が成功し、出力インスタンス メッセージ ファイルが作成されました。  
  
 **ユーザーの操作**  
  
 適宜、マップのテスト操作によって作成された、出力インスタンス メッセージ ファイルを確認してください。
