# <a name="error---xsl-transformation-failure"></a>エラー - XSL 変換エラー
**エラー コード**  
  
 btm1050  
  
 **説明**  
  
 変換処理中に、XSLT エラーが発生しました。  
  
 **ユーザーの操作**  
  
 表示されている XSLT エラーに基づき、マップまたは変換元スキーマと変換先のスキーマ、またはその両方を変更し、エラーの原因を解決します。
