# <a name="error---test-map-failure"></a>エラー - マップのテスト エラー
**エラー コード**  
  
 btm1054  
  
 **説明**  
  
 マップに対する、マップのテスト操作に失敗しました。  
  
 **ユーザーの操作**  
  
 関連するエラー メッセージを参考にして、このマップのテスト操作が失敗した原因を特定し、マップか、このマップに関連付けられている送信元スキーマおよび送信先スキーマ、またはその両方に必要な変更を加えます。
