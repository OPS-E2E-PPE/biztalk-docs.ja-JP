# <a name="configuring-a-port-to-receive-edi-messages-and-acknowledgments"></a>EDI メッセージおよび受信確認を受信するポートの構成
EDI インターチェンジを受信するために、一方向の受信ポートまたは要求 - 応答 (双方向) の受信ポートのいずれかを作成できます。  
  
-   作成、一方向受信ポートの場合 (有効な場合)、EDI 受信確認を送信する一方向の送信ポートを作成することも、します。 クリアする必要があります、**要求-応答で送信パイプラインに確認をルーティングの受信ポート**アグリーメントのプロパティです。  
  
-   関連付けられた送信パイプラインを介して EDI 受信確認 (有効な場合) を返す場合は、要求 - 応答の受信ポートおよび受信場所を作成します。 選択する必要があります、**要求-応答で送信パイプラインに確認をルーティングの受信ポート**アグリーメントのプロパティです。
