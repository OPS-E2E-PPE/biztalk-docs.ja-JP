# <a name="how-to-add-web-messages"></a>Web メッセージの追加方法
[オーケストレーションの種類] ウィンドウでは、Web メッセージ変数を作成します。 Web メッセージの種類は、いずれかの Web メッセージの種類に設定する必要があります。  
  
### <a name="to-add-a-web-message"></a>Web メッセージを追加するには  
  
1.  オーケストレーションを開き、[、**ビュー** ] メニューのをクリックして**その他のウィンドウ**、順にクリック**オーケストレーション**です。  
  
2.  右クリックし、**メッセージ**ノードをクリックして**新しいメッセージ**です。  
  
3.  Message _*x*下に表示されます、**メッセージ**ノード。  
  
4.  **プロパティ** ウィンドウから、**メッセージの種類**ドロップダウン ボックスで、Web メッセージの種類を選択します。  
  
5.  使用する Web メッセージの種類までドロップダウン ボックスをスクロールし、Web メッセージの種類をクリックして選択します。
