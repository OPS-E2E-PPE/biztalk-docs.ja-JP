# <a name="arithmetic-operators"></a>算術演算子
ビジネス ルール フレームワークでは、ビジネス ルールの作成時に、加算、減算、乗算、除算、剰余の各演算子を使用できます。 次の表に、算術演算子を示します。  
  
|算術演算子|Description|  
|-------------------------|-----------------|  
|**[追加]**|2 つの引数間で加算を行うための演算子を表します。|  
|**減算**|2 つの引数間で減算を行うための演算子を表します。|  
|**乗算**|2 つの引数間で乗算を行うための演算子を表します。|  
|**Divide**|2 つの引数間で除算を行うための演算子を表します。|  
|**残りの部分**|2 つの引数間での除算の剰余の演算子を表します。|  
  
 オペランドの型が異なる場合、短い方の型の数値が長い方の型に合わせて自動的に変換されます。 たとえば場合、**追加**の最初のオペランドに演算子が使用される**int**型との 2 番目のオペランド**長い**に最初のオペランドの型は型が変換されます**長い**実行する前に、**追加**操作します。 ルール エンジンでは、両方のオペランドの型を同一のものに変換できる場合は、同時変換もサポートしています。 たとえば場合、**追加**の最初のオペランドに演算子が使用される**int**型との 2 番目のオペランド**uint** する型、両方のオペランドの型を変換**長い**実行する前に、**追加**操作します。

1.  [ファクト エクスプ ローラー] ウィンドウ、**ボキャブラリ**タブです。  
  
2.  展開**ボキャブラリ**、展開**関数**、展開**バージョン 1.0 - 公開済み**、し、ドラッグ、**追加/削除/乗算/除算/剰余**条件ペイン/アクション ウィンドウにします。  
  
3.  左側および右側の演算子の値を指定します。
