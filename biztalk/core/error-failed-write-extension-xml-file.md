# <a name="error---failed-write-extension-xml-file"></a>エラー - 拡張 XML ファイルの書き込みに失敗しました
**エラー コード**  
  
 btm1058  
  
 **説明**  
  
 このマップによって生成された拡張オブジェクト XML をファイルに書き込もうとしたときに、エラーが発生しました。  
  
 **ユーザーの操作**  
  
 表示されているエラーに基づき、適切な操作を行ってエラーの原因を修正します。 たとえば、ハード ディスク上の領域の一部を解放します。
