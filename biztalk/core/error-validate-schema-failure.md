# <a name="error---validate-schema-failure"></a>エラー - スキーマの検証エラー
**説明**  
  
 スキーマが正常に検証されませんでした。  
  
 **ユーザーの操作**  
  
 表示されたエラー メッセージを調査して、スキーマの検証エラーの原因を特定します。
