# <a name="how-to-select-multiple-functoids"></a>複数の Functoid を選択する方法
大規模なマップを扱うとき、複数の Functoid に対して同じ操作を実行したい場合があります。 このような場合は、必要となるすべての Functoid を選択して操作を実行できます。 たとえば、接続した一連の Functoid と、それらを接続するリンクをグリッド ページ間で移動するような場合です。  
  
### <a name="to-select-multiple-functoids-at-the-same-time"></a>複数の Functoid を同時に選択するには  
  
1.  BizTalk マッパーのグリッド ページで、最初の Functoid をクリックして選択します。 Functoid が強調表示されます。  
  
2.  BizTalk マッパーの同じグリッド ページで、Ctrl キーまたは Shift キーを押したまま、別の Functoid をクリックして選択範囲に追加します。 この Functoid も強調表示されます。  
  
3.  手順 2. を繰り返しながら、対象となるすべての Functoid を選択します。  
  
    > [!NOTE]
    >  接続した一連の Functoid と、それらを接続するリンクをグリッド ページ間でまとめて移動するには、対象となる複数の Functoid およびリンクがグリッド ページで選択されている必要があります。
