# <a name="interceptor-management-commands"></a>インターセプター管理コマンド
新しい BAM インターセプター機能をサポートするために、BAM 管理ユーティリティに 4 つの新しいコマンドが追加されました。  
  
 これらのコマンドは、展開、取得、およびインターセプターの削除をサポートします。 コマンドは、構成されているインターセプターを一覧にも提供されます。  
  
-   インターセプターの展開:インターセプター構成をデプロイします。  
  
-   get interceptorlist:インターセプトが展開されているアクティビティの一覧を取得します。  
  
-   get インターセプター:インターセプター構成を取得します。  
  
-   削除インターセプター:インターセプター構成を削除します。  
  
> [!NOTE]
>  含めることによって任意の BM ユーティリティ コマンドのトレースを有効にすることができます、 **-トレース: &#124;オフ**パラメーター スイッチ。 トレース スイッチを使用するには、構成ファイルのトレース設定が上書きされます。 スイッチは、通常の BM コマンドと組み合わせて使用できます。  
  
> [!NOTE]
>  ユーザー アカウント制御 (UAC) をサポートするシステムでは、管理者特権を使用してこのツールを実行することが必要な場合があります。  
  
## <a name="deploy-interceptor-command"></a>deploy-interceptor コマンド  
 **使用方法**  
  
 **bm.exe deploy-interceptor -FileName:\<Configuration XML Filename\> [-Force:True ] [-Server:\<server\>] [-Database:\<database\>]**  
  
 **パラメーター**  
  
|パラメーター|説明|  
|---------------|-----------------|  
|ファイル名:\<構成 XML ファイル名\>|インターセプター構成を含む XML ファイルの名前。|  
|Force: True|省略可能:イベント ソース名の競合が検出されたときに、インターセプター構成を展開します。|  
|サーバー:\<サーバー\>|省略可能:インターセプターを展開先サーバーの名前。 サーバーは、bm.exe を実行するコンピューターと同じドメインにある必要があります。|  
|データベース:\<データベース\>|省略可能:インターセプターを構成する BAM プライマリ インポート データベースの名前。|  
  
 このコマンドは、指定したサーバーおよびデータベースにインターセプター構成をデプロイします。 デプロイ時に、BAM 管理ユーティリティは、次の検証を実行します。  
  
- XSD 検証:インターセプター構成は、共通のインターセプター構成スキーマに対して検証されます。  
  
- アクティビティが存在する検証 (は、プライマリ インポート データベースに展開) のチェックポイントが有効であると (存在し、一致するデータ型である)。  
  
  イベント ソース名の競合が検出されると、競合を説明する警告がスローされます。 競合がある場合、配置が失敗しない限り、 **– Force:true**パラメーター フラグが使用されます。  
  
> [!NOTE]
>  **– Force:true**パラメーターは可能性のある同じ名前のイベント ソースを参照するインターセプター構成を削除します。 使用する必要があります、 **get インターセプター**コマンドを使用する前に既存のインターセプター構成のバックアップを作成、 **– Force:true**パラメーター。  
  
 **使用例**  
  
```  
bm.exe deploy-interceptor  -FileName:myInceptor.xml  
bm.exe deploy-interceptor  -FileName:myInceptor.xml -Force:True  
```  
  
## <a name="get-interceptorlist-command"></a>get-interceptorlist コマンド  
 **使用方法**  
  
 **bm.exe get-interceptorlist [-Server:\<server\>] [-Database:\<database\>]**  
  
 **パラメーター**  
  
|パラメーター|説明|  
|---------------|-----------------|  
|サーバー:\<サーバー\>|省略可能:展開されたインターセプターの一覧を返す元のサーバーの名前。 サーバーは、bm.exe を実行するコンピューターと同じドメインにある必要があります。|  
|データベース:\<データベース\>|省略可能:展開されたインターセプターの取得元となる BAM プライマリ インポート データベースの名前。|  
  
 このコマンドは、アクティビティおよびインターセプションが有効になっている、関連付けられたイベント ソースの一覧を返します。  
  
 **例**  
  
```  
bm.exe get-interceptorlist   
```  
  
## <a name="get-interceptor-command"></a>get-interceptor コマンド  
 **使用方法**  
  
 **bm.exe の get インターセプター [-サーバー:\<server\>] [-データベース:\<データベース\>]-filename:\<構成 XML ファイル名\>[-アクティビティ。\<アクティビティ名\>] [-EventSource:\<イベント ソース名\>]**  
  
 **パラメーター**  
  
|パラメーター|説明|  
|---------------|-----------------|  
|サーバー:\<サーバー\>|省略可能:展開されたインターセプターの取得元となるサーバーの名前。 サーバーは、bm.exe を実行するコンピューターと同じドメインにある必要があります。|  
|データベース:\<データベース\>|省略可能:展開されたインターセプターの取得元となる BAM プライマリ インポート データベースの名前。|  
|ファイル名:\<構成 XML ファイル名\>|インターセプター構成を記述する XML ファイルの名前。|  
|アクティビティ:\<アクティビティ名\>|省略可能:構成されたインターセプターを返す対象のアクティビティを指定します。 組み合わせて使用できる、 **EventSource**パラメーターを返す構成をさらに詳しく指定できます。|  
|EventSource:\<イベント ソース名\>|省略可能:構成されたインターセプターを返す対象のイベント ソースを指定します。 組み合わせて使用できる、**アクティビティ**パラメーターを返す構成をさらに詳しく指定できます。|  
  
 アクティビティ名、またはイベント ソース名が指定されていない場合は、すべてのイベント ソースとアクティビティに対するインターセプター構成を含む有効な構成ファイルが返されます。  
  
 場合のみ、アクティビティ名を指定すると、コマンドはそのアクティビティのすべてのイベント ソースの構成ファイル、有効なインターセプターを返します。  
  
 のみ、イベント ソースの名前を指定すると、コマンドは、ファイルを返します有効なインターセプター構成したイベント ソースのすべてのアクティビティ間で。  
  
 アクティビティ名とイベント ソース名の両方を指定しない場合、コマンドを返します有効なインターセプターそのアクティビティには、そのイベント ソースの構成ファイル。  
  
 **使用例**  
  
```  
bm.exe get-interceptor   
bm.exe get-interceptor  -Activity:ShippingPO  
```  
  
## <a name="remove-interceptor-command"></a>remove-interceptor コマンド  
 **使用方法**  
  
 **bm.exe の削除インターセプター [-サーバー:\<server\>] [-データベース:\<データベース\>] [-アクティビティ。\<アクティビティ名\>] [-EventSource:\<イベント ソース名\>]**  
  
 **パラメーター**  
  
|パラメーター|説明|  
|---------------|-----------------|  
|サーバー:\<サーバー\>|省略可能:インターセプターが構成されているサーバーの名前。 サーバーは、bm.exe を実行するコンピューターと同じドメインにある必要があります。|  
|データベース:\<データベース\>|省略可能:インターセプターが構成されているデータベースの名前。|  
|アクティビティ:\<アクティビティ名\>|省略可能:指定のインターセプターを削除するアクティビティを指定します。 組み合わせて使用できる、 **EventSource**パラメーターを返す構成をさらに詳しく指定できます。|  
|EventSource:\<イベント ソース名\>|省略可能:指定のインターセプターを削除するイベント ソースを指定します。 組み合わせて使用できる、**アクティビティ**パラメーターを返す構成をさらに詳しく指定できます。|  
  
 場合のみ、アクティビティ名を指定すると、そのアクティビティのすべてのイベント ソースのインターセプターが削除されます。  
  
 場合のみ、イベント ソースの名前を指定すると、すべてのアクティビティのイベント ソース部分のみが削除されます。  
  
 **例**  
  
```  
bm.exe remove-interceptor   
```
