# <a name="how-to-search-for-activities"></a>アクティビティを検索する方法
BAM Web ポータルでビジネス アクティビティの検索機能を使用すると、BAM データベースに保存された追跡データ (インスタンス) を検索できます。  
  
 この検索では、現在実行中のプロセスのライブ データと保存されたデータの両方に対してクエリが実行されます。  
  
### <a name="to-search-the-bam-database"></a>BAM データベースを検索するには  
  
1.  Internet explorer で、**アドレス**ボックスに、BAM Web ポータルの URL を入力します。  
  
2.  **マイ ビュー**ナビゲーション ウィンドウで、ビュー、タスクの一覧を表示するビューを展開してをクリックします。  
  
3.  クリックして、**アクティビティの検索**タスク アクティビティの一覧を表示する.  
  
4.  アクティビティをクリックして、コンテンツ フレームにアクティビティの検索ページを読み込みます。  
  
5.  **クエリ**ビルダーは、検索のクエリ パラメーターを選択します。  
  
6.  **列の選択**検索で返されるデータ項目を表す列を選択します。  
  
7.  クリックして、**クエリの実行**上部にあるボタン、コンテンツ フレームの右。  
  
     結果が表示されます、**結果**アクティビティの検索コンテンツ ページの領域。  
  
> [!NOTE]
>  最大 20 個の列を選択することができます。  
  
> [!NOTE]
>  このリリースでは、論理演算子のグループ化はサポートされません。
