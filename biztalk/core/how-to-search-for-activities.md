# <a name="how-to-search-for-activities"></a>アクティビティを検索する方法
BAM Web ポータルでビジネス アクティビティの検索を使用して、検索、追跡データを BAM データベースに格納されているインスタンスとも呼ばれます。  
  
 格納されているデータだけでなく、プロセス、現在実行中のライブ データの両方に対する検索クエリ。  
  
### <a name="to-search-the-bam-database"></a>BAM データベースを検索するには  
  
1.  Internet explorer で、**アドレス**ボックスに、BAM Web ポータルの URL を入力します。  
  
2.  **マイ**ナビゲーション ウィンドウで、タスク一覧を表示するビューを展開するビューをクリックします。  
  
3.  をクリックして、**アクティビティの検索**タスク アクティビティの一覧を表示する.  
  
4.  コンテンツ フレームにアクティビティの検索ページを読み込むアクティビティをクリックします。  
  
5.  **クエリ**ビルダーは、検索のクエリ パラメーターを選択します。  
  
6.  **列の選択**検索で返されるデータ項目を表す列を選択します。  
  
7.  をクリックして、**クエリの実行**上部にあるボタン コンテンツ フレームの右。  
  
     結果が表示されます、**結果**アクティビティの検索コンテンツ ページの領域。  
  
> [!NOTE]
>  最大 20 個の列を選択することができます。  
  
> [!NOTE]
>  このリリースでは、論理演算子のグループ化はサポートしません。
