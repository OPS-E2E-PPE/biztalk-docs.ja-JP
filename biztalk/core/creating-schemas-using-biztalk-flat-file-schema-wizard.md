# <a name="creating-schemas-using-biztalk-flat-file-schema-wizard"></a>BizTalk フラット ファイル スキーマ ウィザードを使用してスキーマを作成します。
BizTalk フラット ファイル スキーマ ウィザードは、次を提供することでフラット ファイル スキーマを作成するプロセスを簡略化する設計されています。  
  
- フラット ファイル インスタンスの入力として使用する機能  
  
- 区切り文字および位置指定フラット ファイル スキーマを操作するためのビジュアル デ ザイン サーフェイス  
  
- 注釈をスキーマに要素を追加し、フラット ファイルを定義する対話型ウィザード ベースのプロセスに関連します。  
  
  このセクションでは、BizTalk フラット ファイル スキーマ ウィザードの概要手順と、ウィザードを起動するためのウィザード、およびドキュメント インスタンスからフラット ファイル スキーマを作成する方法のチュートリアルを操作するための考慮事項を示します。
