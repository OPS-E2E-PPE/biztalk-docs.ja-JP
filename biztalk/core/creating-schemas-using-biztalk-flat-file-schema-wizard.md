# <a name="creating-schemas-using-biztalk-flat-file-schema-wizard"></a>BizTalk フラット ファイル スキーマ ウィザードでのスキーマの作成
BizTalk フラット ファイル スキーマ ウィザードでは、次の機能によって、フラット ファイル スキーマを簡単な手順で作成できます。  
  
-   フラット ファイル インスタンスを入力として使用できる機能  
  
-   区切り文字および位置指定フラット ファイル スキーマを操作するため、ビジュアル デザイン画面  
  
-   注釈をスキーマに要素を追加し、フラット ファイルを定義する対話型ウィザード ベースのプロセスに関連します。  
  
 ここでは、BizTalk フラット ファイル スキーマ ウィザードの概要と、このウィザードの起動方法、このウィザードを使用する際の注意事項、およびドキュメント インスタンスからフラット ファイル スキーマを作成する手順について説明します。
