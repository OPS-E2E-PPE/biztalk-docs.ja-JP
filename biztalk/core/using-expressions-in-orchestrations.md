# <a name="using-expressions-in-orchestrations"></a>オーケストレーションでの式の使用
BizTalk 式エディターを使用して XLANG/s 式を入力してロジックを追加し、オーケストレーションの変数およびメッセージの値を操作またはテストできます。 ただし、式図形を使用して高度なオーケストレーション ロジックを実行することはお勧めしません。高度なオーケストレーション ロジックは多くの場合、オーケストレーション図面自体に表示されます。 ビジネス プロセスの透明性を保ち、再構築を簡単にするため、単純なモジュール形式の式を使用することをお勧めします。
