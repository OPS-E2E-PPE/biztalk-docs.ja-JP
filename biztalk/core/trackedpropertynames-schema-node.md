# <a name="trackedpropertynames-schema-node"></a>TrackedPropertyNames (Schema ノード)
バインド ファイルの TrackedPropertyNames ノードには、追跡、親スキーマでプロパティを指定する要素が含まれています。  
  
## <a name="nodes-in-the-trackedpropertyname-node"></a>TrackedPropertyName ノード内のノード  
 次の表に、バインド ファイルのこのノードに設定できるプロパティを示します。  
  
|**名前**|**ノードの種類**|**[データ型]**|**[説明]**|**制限**|**コメント**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|string|要素|xs:string|追跡するプロパティ名を指定します。|任意|既定値: 空|
