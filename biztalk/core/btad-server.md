# <a name="btadserver"></a>BTAD_Server
アプリケーションのインポート時に処理前および処理後のスクリプトを実行すると、BTAD_Server 環境変数が設定されます。 この環境変数で、グループの BizTalk 管理データベースを実行している SQL Server インスタンスの名前を指定します。
