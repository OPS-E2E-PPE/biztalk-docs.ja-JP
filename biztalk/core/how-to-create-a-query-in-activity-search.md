# <a name="how-to-create-a-query-in-activity-search"></a>アクティビティの検索でクエリを作成する方法
ビジネス エンドユーザーおよび業務に関するイベントや状態の通知を受け取る必要があるその他のユーザーは、クエリを使用して、アラートを基になるアクティビティの検索を作成します。  
  
## <a name="prerequisites"></a>前提条件  
 展開済みのビューが必要です。  
  
### <a name="to-open-a-query"></a>クエリを開くには  
  
1. クリックして**開始**、 をポイント**すべてのプログラム**、 をクリックして[!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)]、順にクリックします**BAM ポータル Web サイト**します。  
  
2. **マイ ビュー**フレーム、ポータルのそのビューに使用できるメニューを展開する既存のビューをクリックします。  
  
3. クリックして**アクティビティの検索**ビューで使用できるアクティビティの一覧を展開します。  
  
4. 一覧からアクティビティをクリックします。 選択したアクティビティのアクティビティの検索ページが読み込まれます。  
  
5. ページの上部にある、コンテンツ フレームで、**参照**ボタンをクリックする、**ファイルの選択** ダイアログ ボックス。  
  
6. クエリを以前保存したフォルダーに移動します。  
  
7. [保存] をクリックしてクエリ。  
  
8. をクリックして、**オープン**ボタンをクリックします。 左側にテキスト ボックスのクエリへのパスが読み込まれます、**参照**ボタンをクリックします。 テキスト ボックスに直接クエリへのパスを入力することもできます。  
  
9. をクリックして、**クエリを開く**ボタンの右側に、**参照**ボタンをクリックします。  
  
### <a name="to-construct-a-query"></a>クエリを作成するには  
  
1. クリックして**開始**、 をポイント**すべてのプログラム**、 をクリックして[!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)]、順にクリックします**BAM ポータル Web サイト**します。  
  
2. **マイ ビュー**構成されているビューを現在の一覧がある、ポータルのフレーム。 それぞれのビューには、ビューに対して実行できる 3 つのタスクがあります。 ビューが現在折りたたまれている場合は、タスクの一覧を展開する既存のビューをクリックします。  
  
3. クリックして**アクティビティの検索**ビューで使用できるアクティビティの一覧を展開します。  
  
4. 一覧からアクティビティをクリックします。 選択したアクティビティのアクティビティの検索ページが読み込まれます。  
  
5. コンテンツのフレームで、**クエリ**ボックスからフィールドを選択して、**ビジネス データ**ドロップダウン リストで、クエリの比較に使用します。  
  
6. **演算子**ドロップダウン リストで、使用する比較演算子を選択します。  
  
7. **値**ボックスに、比較対象値を入力します。 比較するフィールドに適切な値を入力することのみできます。 (ビジネス データ項目が日付フィールドの場合、比較データは日付または日付時刻の形式のカルチャ設定に応じて)。  
  
8. クエリに追加するその他の句があれば、 をクリックして、**追加**クエリ ボックスの右側にボタンをクリックします。 これにより、次の句に新しい行が追加されます。 または OR を使用してした句を結合するかどうかを選択できます。  
  
   > [!NOTE]
   >  複雑なクエリを形成する句をグループ化することはできません。 クエリは、または、OR 演算子で結合された句の単純なセットです。  
  
   > [!NOTE]
   >  これらの手順で、[戻る] ボタンをクリックし、受信する場合、"警告: ページの有効期限が切れて"元のコンテンツに戻るには f5 キーを押すことができます。 何度も f5 キーを押す必要があります。  
  
9. 列の選択 で、データまたはマイルス トーンから選択、**使用可能なデータとマイルス トーン**としてデータを返すクエリのリスト ボックス。 複数の項目を選択するには、SHIFT キーを押したまま取得するグループの最初と最後の項目をクリックします。 CTRL キーを押したまま、返されるアイテムを選択して、一覧から複数の項目を選択することもできます。  
  
10. 使用して、 **>>** を選択したアイテムを移動するボタン、**表示する項目**ボックスの一覧。 選択しを使用して、この一覧から項目を削除することができます、 **<<** ボタン、データとマイルス トーン ボックスに移動して、クリックします。  
  
11. クエリが返すすべての項目を選択すると、列の表示順序を判断する結果を並べ替えることができます。 返されたデータ セット内の最初の位置に列を移動するクリックして をクリックして選択、**上へ移動**される項目の一覧の上部にあるボタンをクリックします。 同様に、返されたデータ セット内の後の位置に項目を移動するクリックし、をクリックして項目を選択、**下へ移動**  ボタンをクリックして表示する位置にあること。  
  
### <a name="to-save-a-query"></a>クエリを保存するには  
  
1.  ページの上部にある、コンテンツ フレームで、**クエリの保存**ボタンをクリックする、**ファイルを保存** ダイアログ ボックス。  
  
2.  **ファイルのダウンロード**ダイアログ ボックスで、をクリックして、**保存**ボタン。  
  
3.  クエリを保存する場所に移動します。  
  
4.  クエリの既定の名前をそのまま使用できますかに新しい名前を入力することができます、**ファイル名**テキスト ボックス。  
  
5.  **[保存]** ボタンをクリックします。  
  
### <a name="to-execute-a-query"></a>クエリを実行するには  
  
1. クリックして**開始**、 をポイント**すべてのプログラム**、 をクリックして[!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)]、順にクリックします**BAM ポータル Web サイト**します。  
  
2. **マイ ビュー**フレーム、ポータルのそのビューに使用できるメニューを展開する既存のビューをクリックします。  
  
3. クリックして**アクティビティの検索**ビューで使用できるアクティビティの一覧を展開します。  
  
4. 一覧からアクティビティをクリックします。 選択したアクティビティのアクティビティの検索ページが読み込まれます。  
  
5. 既存のクエリを開くか、新しいクエリを作成します。  
  
6. ポータルのコンテンツのフレームで、**クエリの実行**ボタンをクリックします。
