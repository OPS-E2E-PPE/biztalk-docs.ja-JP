# <a name="how-to-create-a-query-in-activity-search"></a>アクティビティの検索でクエリを作成する方法
業務に関するイベントや状態の通知を受け取る必要があるビジネス エンド ユーザーおよびその他のユーザーは、クエリを使用して、警告の基礎となるアクティビティの検索を作成します。  
  
## <a name="prerequisites"></a>前提条件  
 展開済みのビューが必要です。  
  
### <a name="to-open-a-query"></a>クエリを開くには  
  
1. クリックして**開始**、 をポイント**すべてのプログラム**、 をクリックして[!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)]、順にクリックします**BAM ポータル Web サイト**します。  
  
2. **マイ ビュー**フレーム、ポータルのそのビューに使用できるメニューを展開する既存のビューをクリックします。  
  
3. クリックして**アクティビティの検索**ビューで使用できるアクティビティの一覧を展開します。  
  
4. 一覧からアクティビティを選択します。 アクティビティを選択すると、そのアクティビティの [アクティビティの検索] ページが読み込まれます。  
  
5. ページの上部にある、コンテンツ フレームで、**参照**ボタンをクリックする、**ファイルの選択** ダイアログ ボックス。  
  
6. 以前に保存したクエリが格納されているフォルダーに移動します。  
  
7. [保存] をクリックしてクエリ。  
  
8. をクリックして、**オープン**ボタンをクリックします。 左側にテキスト ボックスのクエリへのパスが読み込まれます、**参照**ボタンをクリックします。 また、このボックスにクエリのパスを直接入力することもできます。  
  
9. をクリックして、**クエリを開く**ボタンの右側に、**参照**ボタンをクリックします。  
  
### <a name="to-construct-a-query"></a>クエリを作成するには  
  
1. クリックして**開始**、 をポイント**すべてのプログラム**、 をクリックして[!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)]、順にクリックします**BAM ポータル Web サイト**します。  
  
2. **マイ ビュー**構成されているビューを現在の一覧がある、ポータルのフレーム。 それぞれのビューには、ビューに対して実行できる 3 つのタスクがあります。 ビューが折りたたまれている場合は、既存のビューをクリックし、タスクの一覧を表示します。  
  
3. クリックして**アクティビティの検索**ビューで使用できるアクティビティの一覧を展開します。  
  
4. 一覧からアクティビティを選択します。 アクティビティを選択すると、そのアクティビティの [アクティビティの検索] ページが読み込まれます。  
  
5. コンテンツのフレームで、**クエリ**ボックスからフィールドを選択して、**ビジネス データ**ドロップダウン リストで、クエリの比較に使用します。  
  
6. **演算子**ドロップダウン リストで、使用する比較演算子を選択します。  
  
7. **値**ボックスに、比較対象値を入力します。 このボックスには比較するフィールドに適した値しか入力できません。 ビジネス データ項目が日付型のフィールドの場合、比較するデータはカルチャ設定に適した形式の日付または日時のデータとなります。  
  
8. クエリに追加するその他の句があれば、 をクリックして、**追加**クエリ ボックスの右側にボタンをクリックします。 このボタンをクリックすると、次の句のための新しい行が追加されます。 AND 演算子または OR 演算子を使用して、追加した句を結合するかどうかを指定できます。  
  
   > [!NOTE]
   >  複数の句をグループ化して、複雑なクエリを作成することはできません。 クエリは AND 演算子または OR 演算子で結合した単純な一連の句になります。  
  
   > [!NOTE]
   >  この一連の手順の途中で [戻る] ボタンをクリックしたときに、"警告: ページの有効期限切れ" というメッセージが表示された場合は、F5 キーを押して元のコンテンツを取得することができます。 ただし、場合によっては、F5 キーを複数回押す必要があります。  
  
9. 列の選択 で、データまたはマイルス トーンから選択、**使用可能なデータとマイルス トーン**としてデータを返すクエリのリスト ボックス。 Shift キーを押しながら、グループ内の最初のアイテムと最後のアイテムをクリックすると、データとして返す複数のアイテムを選択できます。 また、Ctrl キーを押しながら、データとして返すアイテムを個別に選択して、複数のアイテムを選択することもできます。  
  
10. 使用して、 **>>** を選択したアイテムを移動するボタン、**表示する項目**ボックスの一覧。 選択しを使用して、この一覧から項目を削除することができます、 **<<** ボタン、データとマイルス トーン ボックスに移動して、クリックします。  
  
11. クエリで返すアイテムをすべて選択したら、結果を調整して、列の表示順序を決めることができます。 返されたデータ セット内の最初の位置に列を移動するクリックして をクリックして選択、**上へ移動**される項目の一覧の上部にあるボタンをクリックします。 同様に、返されたデータ セット内の後の位置に項目を移動するクリックし、をクリックして項目を選択、**下へ移動**  ボタンをクリックして表示する位置にあること。  
  
### <a name="to-save-a-query"></a>クエリを保存するには  
  
1.  ページの上部にある、コンテンツ フレームで、**クエリの保存**ボタンをクリックする、**ファイルを保存** ダイアログ ボックス。  
  
2.  **ファイルのダウンロード**ダイアログ ボックスで、をクリックして、**保存**ボタン。  
  
3.  クエリを保存する場所に移動します。  
  
4.  クエリの既定の名前をそのまま使用できますかに新しい名前を入力することができます、**ファイル名**テキスト ボックス。  
  
5.  をクリックして、**保存**ボタンをクリックします。  
  
### <a name="to-execute-a-query"></a>クエリを実行するには  
  
1. クリックして**開始**、 をポイント**すべてのプログラム**、 をクリックして[!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)]、順にクリックします**BAM ポータル Web サイト**します。  
  
2. **マイ ビュー**フレーム、ポータルのそのビューに使用できるメニューを展開する既存のビューをクリックします。  
  
3. クリックして**アクティビティの検索**ビューで使用できるアクティビティの一覧を展開します。  
  
4. 一覧からアクティビティを選択します。 アクティビティを選択すると、そのアクティビティの [アクティビティの検索] ページが読み込まれます。  
  
5. 既存のクエリを開くか、または新しいクエリを作成します。  
  
6. ポータルのコンテンツのフレームで、**クエリの実行**ボタンをクリックします。
