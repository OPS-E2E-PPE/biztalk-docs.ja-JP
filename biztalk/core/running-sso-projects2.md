# <a name="running-sso-projects"></a>SSO プロジェクトの実行
Internet Explorer からシングル サインオン (SSO) のサンプル プロジェクトを実行できます。  
  
## <a name="run-a-sample-from-a-web-browser"></a>Web ブラウザーからサンプルを実行します。
  
1.  ブラウザーを開きます。  
  
2.  次の URL に移動します。

     この場合、資格情報を指定する必要はありません。
