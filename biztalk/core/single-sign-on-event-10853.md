# <a name="single-sign-on-event-10853"></a>シングル サインオン: イベント 10853
## <a name="details"></a>詳細  
  
|||  
|-|-|  
|製品名|エンタープライズ シングル サインオン|  
|製品バージョン|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|イベント ID|10853|  
|イベント ソース|ENTSSO|  
|コンポーネント|なし|  
|シンボル名|ENTSSO_E_MAPPING_NO_ACCESS|  
|メッセージ テキスト|指定されたユーザーが Application Users アカウントのメンバーではないため、マッピングを作成できません。|  
  
## <a name="explanation"></a>説明  
 指定されたユーザーが Application Users アカウントのメンバーではないため、マッピングを作成できません。  
  
## <a name="user-action"></a>ユーザーの操作  
 システム管理者に連絡し、アプリケーション ユーザー アカウントのメンバーを探して、このマッピングを作成します。
