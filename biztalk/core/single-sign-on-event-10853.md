# <a name="single-sign-on-event-10853"></a>シングル サインオン:イベント 10853
## <a name="details"></a>詳細  
  
|                 |                                                                                                            |
|-----------------|------------------------------------------------------------------------------------------------------------|
|  製品名   |                                         エンタープライズ シングル サインオン                                          |
| 製品バージョン |                         [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                         |
|    イベント ID     |                                                   10853                                                    |
|  イベント ソース   |                                                   ENTSSO                                                   |
|    コンポーネント    |                                                    なし                                                     |
|  シンボル名  |                                         ENTSSO_E_MAPPING_NO_ACCESS                                         |
|  メッセージ テキスト   | 指定したユーザーがアプリケーション ユーザー アカウントのメンバーではないために、マッピングを作成できません。 |
  
## <a name="explanation"></a>説明  
 指定したユーザーがアプリケーション ユーザー アカウントのメンバーではないために、マッピングを作成できません。  
  
## <a name="user-action"></a>ユーザーの操作  
 このマッピングを作成するには、アプリケーション ユーザー アカウントのメンバーの検索に役立つ、システム管理者に依頼します。
