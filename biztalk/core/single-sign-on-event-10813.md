# <a name="single-sign-on-event-10813"></a>シングル サインオン:イベント 10813
## <a name="details"></a>詳細  
  
|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  製品名   |                 エンタープライズ シングル サインオン                  |
| 製品バージョン | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    イベント ID     |                           10813                            |
|  イベント ソース   |                           ENTSSO                           |
|    コンポーネント    |                            なし                             |
|  シンボル名  |         ENTSSO_E_WINDOWS_UPDATE_FAILED_MORE_RECENT         |
|  メッセージ テキスト   |  SSO データベース内の Windows パスワードは最近です。  |
  
## <a name="explanation"></a>説明  
 期限切れのパスワードは、Windows システムから受信したし、は無視されます。  
  
## <a name="user-action"></a>ユーザーの操作  
 ユーザー操作は必要ありません。
