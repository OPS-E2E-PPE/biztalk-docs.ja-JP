# <a name="error---duplicate-distinguished-field-promotion"></a>エラー - 識別フィールドの重複の昇格
**エラー コード**  
  
 BEC2015  
  
 **説明**  
  
 このスキーマのノードは、識別フィールドとして複数回昇格しています。  
  
 **ユーザーの操作**  
  
 使用して、**識別フィールド**のタブ、**プロパティの昇格** ダイアログ ボックスにアクセス、**プロパティの昇格**のプロパティ、 **スキーマ**をすべて削除が、このスキーマ内の同じノードに関連付けられている識別フィールドの昇格のいずれかのノード。
