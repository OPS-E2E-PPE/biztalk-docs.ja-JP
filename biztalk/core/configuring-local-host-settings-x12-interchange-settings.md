# <a name="configuring-local-host-settings-x12-interchange-settings"></a>ローカルホスト設定の構成 (X12 インターチェンジの設定)
ローカル ホスト設定は、EDI インターチェンジを処理する方法を制御します。 このページの設定は、受信者の設定 (着信インターチェンジ) と送信者の設定 (送信インターチェンジの場合) の 2 つのカテゴリに分けることができます。 受信者の設定の一部として、受信のバッチのトランザクション セットに分割または保持されるかどうかを指定できます。 保持される場合を指定できるかどうか[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]インターチェンジまたはトランザクション エラーが発生した場合のセットを中断します。 送信者の設定の一部として、送信メッセージの制御番号を生成する方法を指定できます。

>  ここで説明する設定は、HIPAA インターチェンジにも適用されます。
