# <a name="single-sign-on-event-11048"></a>シングル サインオン:イベント 11048
## <a name="details"></a>詳細  
  
|                 |                                                                                                                                                                                    |
|-----------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|  製品名   |                                                                             エンタープライズ シングル サインオン                                                                              |
| 製品バージョン |                                                             [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                                             |
|    イベント ID     |                                                                                       11048                                                                                        |
|  イベント ソース   |                                                                                       ENTSSO                                                                                       |
|    コンポーネント    |                                                                                        なし                                                                                         |
|  シンボル名  |                                                                              SSO_ERROR_SSOCSTX_FAILED                                                                              |
|  メッセージ テキスト   | SSOCSTX を作成できませんでした。 問題を解決するには、SSO を再インストールするか、SSOCSTX.dll を COM + ライブラリ application.%r にインストール<br /><br /> 追加データ: 1 %r<br /><br /> エラー コード: %2 |
  
## <a name="explanation"></a>説明  
 可能性があります、これは、インストール エラーが発生します。  
  
## <a name="user-action"></a>ユーザーの操作  
 問題を解決するには、SSO を再インストールするか、SSOCSTX.dll を COM + ライブラリ アプリケーションにインストールします。
