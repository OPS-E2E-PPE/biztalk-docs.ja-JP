# <a name="how-to-administer-password-synchronization"></a>パスワード同期を管理する方法
MMC スナップインまたはコマンドラインでパスワード同期を管理することができます。  
  
 MMC スナップインでは、アダプターとそのプロパティの一覧を表示します。 アダプターを右クリックし、メニューを使用して、次のコマンドを実行できます。  
  
- アダプターを作成します。  
  
- プロパティを設定します。  
  
- 更新  
  
- DELETE  
  
- [有効化]  
  
- Disable  
  
- アプリケーション アダプターを追加します。  
  
- アダプターからアプリケーションを削除します。  
  
- 通知をリセットします。  
  
- アダプタ グループにアダプターを追加します。  
  
- アダプタ グループからアダプターを削除します。  
  
  パスワード同期の管理に SSOPS コマンド ライン ユーティリティを使用することもできます。 このセクションのコマンドのほとんどは、管理者のみ用に用意されています。  
  
  多くのコマンドでは、コマンドの出力は、2 つの列では、画面に表示されます。 画面設定によっては、データの切り捨てになる可能性があります、最良の結果をする必要がありますサイズを変更する画面バッファー サイズ/Windows 120 文字にします。  
  
  SSOPS コマンドは、次の表に表示されます。 プロシージャとさらに詳しい説明は、このトピックの残りの部分であります。  
  
|コマンド|関数|  
|-------------|--------------|  
|-リスト|既存のアダプタの一覧|  
|表示|アダプターの情報を表示します|  
|-作成|新しいアダプターを作成します。|  
|-setprops|アダプターのプロパティを設定します|  
|-更新|アダプターの既存の更新|  
|-削除|既存のアダプターを削除します。|  
|-有効にします。|アダプターを使用にします。|  
|-を無効にします。|アダプターを無効にします。|  
|-addapp|アダプタのアプリケーションを追加します。|  
|-deleteapp|アダプタのアプリケーションを削除します。|  
|リセット|リセットの通知キューまたは抑制キュー|  
|-addtogroup」|アダプターをアダプター グループを追加します。|  
|-deletefromgroup|アダプタ グループからアダプターを削除します。|
