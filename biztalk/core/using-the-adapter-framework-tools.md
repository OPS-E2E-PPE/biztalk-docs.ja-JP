# <a name="using-the-adapter-framework-tools"></a>アダプター フレームワーク ツールを使用します。
BizTalk アダプター フレームワークでは、次のツールやビューを使用して、カスタム アダプターを管理および開発します。  
  
-   **BizTalk Server 管理コンソールです。** これは、独立したアプリケーションから使用可能な**開始**メニュー。 このコンソールを使用して、[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] のインストールに関するほとんどすべてのことを管理します。 新しいアダプターのインストール、関連付けられている受信ハンドラーと送信ハンドラーの管理、およびポート、アセンブリ、オーケストレーションの管理を行うことができます。  
  
-   **アダプター メタデータのウィザードを追加します。** このウィザードは、Microsoft では、BizTalk プロジェクト内から使用可能な[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]します。 使用可能になる、ソリューション エクスプ ローラー ウィンドウで、プロジェクト名を右クリックして[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]、クリックして**生成した項目の追加**です。 アダプター メタデータの追加ウィザードは、アプリケーション アダプターと通信する開発者が使用します。 このウィザードを使用して、これらのアプリケーション アダプターに対応するスキーマを各プロジェクトに組み込みます。  
  
-   **アダプターのプロパティ ページ。** を通じて利用、[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]管理コンソールです。 これらのページを使用して、アダプターの受信ハンドラー、受信場所、送信ハンドラー、および送信ポートを構成できます。
