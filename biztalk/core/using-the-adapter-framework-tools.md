# <a name="using-the-adapter-framework-tools"></a>アダプター フレームワーク ツールの使用
BizTalk アダプター フレームワークは、次のツールとビューを使用して、カスタム アダプターの開発し、管理します。  
  
- **BizTalk Server 管理コンソール。** これは、独立したアプリケーションから使用可能な**開始**メニュー。 ほぼすべての管理に使用されます、[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]インストールします。 新しいアダプターをインストールすることができます。関連付けられている受信を管理し、送信ハンドラー。およびポート、アセンブリ、およびオーケストレーションを管理します。  
  
- **アダプター メタデータのウィザードを追加します。** このウィザードは、Microsoft での BizTalk プロジェクト内から利用できる[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]します。 ソリューション エクスプ ローラー ウィンドウで、プロジェクト名を右クリックして使用可能な[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]、 をクリックし、**生成した項目の追加**します。 アダプター メタデータの追加ウィザードはアプリケーション アダプターと通信する開発者向けに用意します。 プルのスキーマをプロジェクトにこれらのアプリケーション アダプターに対応するのには、このウィザードを使用します。  
  
- **アダプターのプロパティ ページ。** これらは、利用、[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]管理コンソール。 これらのページを使用して構成する受信ハンドラー、受信場所、送信ハンドラー、および送信アダプター用ポート。
