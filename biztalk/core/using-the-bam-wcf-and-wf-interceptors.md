# <a name="using-the-bam-wcf-and-wf-interceptors"></a>BAM WCF インターセプタと WF インターセプタの使用
BAM インターセプターでは、[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] の BAM インターセプター機能が Windows Workflow Foundation (WF)、Windows Communication Framework (WCF) などのランタイム環境に拡張されます。 BAM インターセプタを使用すると、WF または WCF のソリューションを再コンパイルせずにビジネス プロセスを追跡できます。統合は、アプリケーション イベントを BAM アクティビティにマップし、データ、関連付け ID、継続トークンなどの必須アイテムと省略可能アイテムを定義する一連の要素と XML を使用して、構成ファイル経由で行われます。
