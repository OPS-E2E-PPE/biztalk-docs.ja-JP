# <a name="single-sign-on-event-11065"></a>シングル サインオン: イベント 11065
## <a name="details"></a>詳細  
  
|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  製品名   |                 エンタープライズ シングル サインオン                  |
| 製品バージョン | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    イベント ID     |                           11065                            |
|  イベント ソース   |                           ENTSSO                           |
|    コンポーネント    |                            なし                             |
|  シンボル名  |            SSO_INFO_PASSWORD_SYNC_MIIS_STARTING            |
|  メッセージ テキスト   |    Windows のパスワード同期を (MIIS から) 開始しています。%r    |
  
## <a name="explanation"></a>説明  
 これは、Windows のパスワード同期を (MIIS から) 開始していることを示す情報メッセージです。  
  
## <a name="user-action"></a>ユーザーの操作  
 ユーザーによる操作は不要です。
