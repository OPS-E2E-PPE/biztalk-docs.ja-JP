# <a name="single-sign-on-event-10836"></a>シングル サインオン:イベント 10836
## <a name="details"></a>詳細  
  
|                 |                                                                        |
|-----------------|------------------------------------------------------------------------|
|  製品名   |                       エンタープライズ シングル サインオン                        |
| 製品バージョン |       [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]       |
|    イベント ID     |                                 10836                                  |
|  イベント ソース   |                                 ENTSSO                                 |
|    コンポーネント    |                                  なし                                   |
|  シンボル名  |                    ENTSSO_E_FILE_PASSWORD_TOO_SHORT                    |
|  メッセージ テキスト   | 指定したパスワードが短すぎる、7 文字以上にする必要があります。 |
  
## <a name="explanation"></a>説明  
 バックアップ ファイルのパスワードは 7 文字以上である必要があります。  
  
## <a name="user-action"></a>ユーザーの操作  
 長さは少なくとも 7 文字のパスワードを入力します。
