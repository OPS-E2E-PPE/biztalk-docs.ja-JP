# <a name="error---schema-root-reference-nonexistent"></a>エラー - スキーマ ルート参照が存在しません
**エラー コード**  
  
 BEC2006  
  
 **説明**  
  
 **ルート参照**のプロパティ、**スキーマ**ノードが存在しないルート ノードを参照しています。 参照先のルート ノードが削除またはとして設定された後に名前が変更された、**ルート参照**プロパティの値。  
  
 **ユーザーの操作**  
  
 設定、**ルート参照**のプロパティ、**スキーマ**ノードを再度します。 そのプロパティのドロップダウン リストには、ルート ノードのすべての現在有効な選択肢が含まれています。
