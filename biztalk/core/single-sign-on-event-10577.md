# <a name="single-sign-on-event-10577"></a>シングル サインオン:イベント 10577

|                 |                                                                                                                                                                                                                                                   |
|-----------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|  製品名   |                                                                                                             エンタープライズ シングル サインオン                                                                                                             |
| 製品バージョン |                                                                                            [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                                                                             |
|    イベント ID     |                                                                                                                       10577                                                                                                                       |
|  イベント ソース   |                                                                                                                      ENTSSO                                                                                                                       |
|    コンポーネント    |                                                                                                                        なし                                                                                                                        |
|  シンボル名  |                                                                                                          SSO_INFO_CHANGED_SSO_AFF_ADMIN                                                                                                           |
|  メッセージ テキスト   | 更新された SSO 関連管理者 account.%r<br /><br /> 新しい SSO 関連管理者: % 1 %r<br /><br /> 古い SSO 関連管理者: % 2 %r<br /><br /> 追跡 ID: % 3 %r<br /><br /> クライアント コンピューターの場合: % 4 %r<br /><br /> クライアント ユーザー: %5 |
  
## <a name="explanation"></a>説明  
 これは、情報メッセージであり、SSO システム内でその発生の重要なセキュリティ関連イベントを追跡するために便利です。 このメッセージは、SSO 関連管理者アカウントが更新されたことを示します。  
  
## <a name="user-action"></a>ユーザーの操作  
 ユーザーによる操作は不要です。
