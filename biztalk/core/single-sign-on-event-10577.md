|                 |                                                                                                                                                                                                                                                   |
|-----------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|  製品名   |                                                                                                             エンタープライズ シングル サインオン                                                                                                             |

|  イベント ソース   |                                                                                                                      ENTSSO                                                                                                                       |
|    コンポーネント    |                                                                                                                        なし                                                                                                                        |

## <a name="explanation"></a>説明  
 これは情報メッセージであり、SSO システム内で発生する重要なセキュリティ関連イベントの追跡に役立つことがあります。 このメッセージは、SSO 関連管理者アカウントが更新されたことを示します。  
  
## <a name="user-action"></a>ユーザーの操作  
 ユーザーによる操作は不要です。
