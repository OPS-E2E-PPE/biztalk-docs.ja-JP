# <a name="creating-send-ports"></a>送信ポートの作成

## <a name="overview"></a>概要
シングル サインオン (SSO) および関連アプリケーションを使用する送信ポートを作成した場合は、ユーザー名とパスワードのパラメーターは指定しません。 選択した **[はい]** の**SSO を使用する**し、一覧から、TIBCO Enterprise Message Service システムを表すために作成した SSO 関連アプリケーションを選択します。
