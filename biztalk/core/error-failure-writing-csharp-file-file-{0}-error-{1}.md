# <a name="error---failure-writing-csharp-file-ltfile---0gt-error-1"></a>エラー - CSharp ファイルの書き込みに失敗しました (&lt;file:---{0}&gt;)。 エラー: {1}
**エラー コード**  
  
 btm1060  
  
 **説明**  
  
 マッパー コンパイラが csharp ファイルをビルド タスクによって指定された場所で作成することでエラーが発生します。  
  
 **ユーザーの操作**  
  
 メッセージに含まれるエラー メッセージを確認してください (エラー {1})。 示す例外がある場合、プロジェクトの出力フォルダーに対する書き込みアクセス許可しないがあります。
