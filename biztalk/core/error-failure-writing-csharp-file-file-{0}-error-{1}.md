# <a name="error---failure-writing-csharp-file-ltfile---0gt-error-1"></a>エラー - CSharp ファイルの書き込みに失敗しました (&lt;file:---{0}&gt;)。 エラー: {1}
**エラー コード**  
  
 btm1060  
  
 **説明**  
  
 ビルド タスクによって指定された場所にマッパー コンパイラが csharp ファイルを作成できない場合にエラーが発生します。  
  
 **ユーザーの操作**  
  
 メッセージ (エラー {1}) に含まれるエラー メッセージを確認します。 未承認の例外の場合、プロジェクトの出力フォルダーへの書き込みアクセス許可がない可能性があります。
