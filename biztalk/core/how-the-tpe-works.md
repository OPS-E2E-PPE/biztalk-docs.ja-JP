# <a name="how-the-tpe-works"></a>TPE の動作
特定の顧客注文のトラブルシューティングまたは特定のビジネス イベントと、出荷処理中のマイルス トーンの傾向の分析は、たとえば、ビジネス アナリストを指定できます表示するデータのビュー。 データを抽出するには、ビジネス アナリストが必要です、開発者は TPE を使用して、特定のビジネス イベント データと実際のオーケストレーション間のマッピングを実行します。 作成し、保存、テスト、および追跡プロファイルを展開しても、後に、ビジネス アナリストはビジネス プロセスの特定の側面を分析して収集されたデータを取得できます。
