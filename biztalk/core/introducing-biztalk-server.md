## <a name="connecting-diverse-systems"></a>多様なシステムを接続します。  
 最新のビジネス プロセスの大半は、ソフトウェアに少なくとも部分的に依存します。 これらのプロセスの一部は、1 つのアプリケーションでサポートされています、その他の多くはさまざまなソフトウェア システムに依存します。 多くの場合、このソフトウェアは、さまざまなプラットフォームとさまざまなテクノロジを使用して、異なるタイミングに作成されました。 これらのビジネス プロセスを自動化するには、多様なシステムを接続する必要があります。  
  
 さまざまな名称では、この課題をアドレス指定: ビジネス プロセス オートメーション (BPA)、ビジネス プロセス管理 (BPM)、およびその他のユーザー。 名前に関係なく、2 つのシナリオは、アプリケーションの統合最も重要です。 1 つには、エンタープライズ アプリケーション統合 (EAI) とも呼ば、1 つの組織内のアプリケーションは接続します。 別の組織にアプリケーションを呼び出された他の企業間取引 (B2B) 統合に接続します。
