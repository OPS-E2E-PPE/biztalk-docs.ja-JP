# <a name="installing-sso"></a>SSO のインストール
以下のセクションでは、エンタープライズ シングル サインオン機能のインストールに関する情報を説明します。 SSO と他の機能とシステム間の複雑な関係が原因と、システムのセキュリティをその重要性のためには、エンタープライズ シングル サインオンをインストールする前に慎重にこのセクションを読む必要があります。  
  
 また、エンタープライズ シングル サインオンをインストールするための最新のソフトウェア前提条件を確認することをお勧めします。
