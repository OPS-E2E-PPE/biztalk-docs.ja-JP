# <a name="simple-type-derivation"></a>単純型の派生
XML スキーマ定義 (XSD) 言語は、次のように既存の単純型の派生に基づく、単純型のバリエーションを定義するためのいくつかのメカニズムを提供します。  
  
- **制限**します。 制約メカニズムを使用して単純型の派生には、その型の有効な値の制限の概要が含まれます。 例は、数値型の最小および最大値または文字列型の最小値と最大長です。  
  
- **リスト**します。 リスト メカニズムを使用して単純型の派生で構成されるとして定義されるインスタンス メッセージに 1 つの属性または要素値は、特定の種類の値をスペース区切りのリスト。  
  
- **共用体**します。 ユニオン メカニズムを使用して単純型の派生では、1 つの属性または要素値をいくつかの指定した種類のいずれか 1 つの値として定義されるインスタンス メッセージにできます。  
  
  このセクションでは、これらの詳細については、[プロパティ] ウィンドウで、適切なノードのプロパティを設定してこれらの派生を定義する方法と、対応する XSD の構築方法を含む単純型の派生について説明します。
