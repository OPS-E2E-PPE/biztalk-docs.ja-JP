 **説明**  
  
 最初の入力パラメーターを指定された**値抽出**functoid からの出力 (ADO レコード セット) があります、**データベース検索**functoid。

 最初の入力をすべて確認**値抽出**functoid からのリンクを**データベース検索**functoid をマップのグリッド ページの左側の。
