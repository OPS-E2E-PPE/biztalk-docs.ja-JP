# <a name="error---failed-write-xslt-file"></a>エラー - XSLT ファイルを書き込めませんでした。
**エラー コード**  
  
 btm1056  
  
 **説明**  
  
 このマップによって生成された XSLT を、指定されたファイルに書き込もうとしたときに、エラーが発生しました。  
  
 **ユーザーの操作**  
  
 表示されているエラーに基づき、適切な操作を行ってエラーの原因を修正します。 たとえば、ハード ディスク上の領域の一部を解放します。
