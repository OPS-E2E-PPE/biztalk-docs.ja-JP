# <a name="requirements-for-using-the-bam-add-in-for-excel"></a>Excel 用 BAM アドインを使用するための要件
ビジネス アナリストは、Excel 用の BAM アドインを使用して、追跡するイベントやデータを定義できます。また、ユーザーがアクセスできる追跡データの論理グループと集計のビューを定義できます。  
  
## <a name="bam-add-in-for-excel-requirements"></a>Excel 用 BAM アドインの要件  
 [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] がインストールされているかどうかに関係なく、すべてのコンピューターで Excel 用 BAM アドインを実行できます。 ただし、Excel 用 BAM アドインを実行するには、次に示すファイルを先にインストールする必要があります。 これらのファイルの場所については、Microsoft Excel ドキュメントを参照してください。  
  
-   Excel.exe  
  
-   Vbe7.dll  
  
-   Stdole2.tlb  
  
-   Mso.dll  
  
-   Fm20.dll  
  
-   Scrrun.dll  
  
-   Mscomctl.ocx  
  
-   Msado15.dll
