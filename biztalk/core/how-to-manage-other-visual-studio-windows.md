# <a name="how-to-manage-other-visual-studio-windows"></a>その他の Visual Studio の Windows を管理する方法
BizTalk エディターで一部の操作を使用して、他の Microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] windows アプリ、つまり、出力ウィンドウおよびタスク一覧 ウィンドウ。 これらのウィンドウを使用して、サンプル インスタンス メッセージの生成と検証のスキーマおよびインスタンスなどの操作からの出力を表示します。 このトピックでは、これらのウィンドウの管理についての詳細な手順を説明します。  
  
### <a name="to-open-the-output-window"></a>出力ウィンドウを開く  
  
- [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]、をクリックして、**ビュー** メニューをクリックして**出力**です。  
  
  > [!NOTE]
  >  出力ウィンドウ内の行に対応するファイルを表示する場合は、その行をダブルクリックします。  
  
### <a name="to-open-the-task-list-window"></a>タスク一覧 ウィンドウを開く  
  
- [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]、をクリックして、**ビュー** メニューをクリックして**タスク一覧**です。  
  
  > [!NOTE]
  >  タスク一覧 ウィンドウで特定のタスク行に対応する BizTalk スキーマのノードを表示する場合は、そのタスク行をダブルクリックします。  
  
  既定では、出力ウィンドウとタスク一覧 ウィンドウでの同じセクションが共有、[!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]シェルです。 同時に両方のウィンドウが開いている場合はそれらの間で切り替えるには、名前付きのタブに表示されます。
