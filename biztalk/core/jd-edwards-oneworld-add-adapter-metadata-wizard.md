# <a name="jd-edwards-oneworld-add-adapter-metadata-wizard"></a>JD Edwards OneWorld アダプター メタデータの追加ウィザード
JD Edwards OneWorld システムからメタデータ情報にアクセスするアダプター メタデータの追加ウィザードを使用します。  
  
|プロパティ|目的|  
|--------------|----------------|  
|アダプターの一覧を選択します。|一覧からアダプターを選択します。|  
|SQL Server|一覧から SQL Server を選択し、をクリックして**次**します。|  
|[データベース]|SQL server と同じデータベースに接続します。|  
|Port|選択、構成された送信ポートとクリックして**次**します。|
