# <a name="business-process-management-solution"></a>ビジネス プロセス管理ソリューション
ビジネス プロセス管理ソリューションでは、サービスの注文処理などのビジネス プロセスを管理する BizTalk アプリケーションを設計する方法を示します。 ソリューションでは、プロセス マネージャを構築する方法について説明し、プロセスを段階に分ける方法のガイダンスを提供します。 ソリューションには、広範な高度な例外処理と同様に割り込み可能なオーケストレーションを構築する方法も説明します。  
  
 パターンとデザインの選択肢、構築、およびソリューションの実行に関する情報の詳細な説明については、ソリューションの概要を説明します。  
  
> [!NOTE]
>  パターンとこれらのセクションに記載されているガイダンスは、そのビジネス ソリューションの特定の種類の方法を説明するために、します。 パターンは、単純なメカニズムでは、特定の問題に適用するためのものし、は通常、その他のパターンと組み合わせてです。 アプリケーションに接続するのにはないです。 「現状有姿のコード例が提供されると、実稼働環境用のものではありません。 パターンの説明だけを目的とし、そのため、例外処理、ログ記録、セキュリティ、および検証などの余分なコードは含まれません。 Microsoft では、「現状有姿コード例を運用環境にデプロイすることはできません。
