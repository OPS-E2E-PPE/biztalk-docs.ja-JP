# <a name="creating-and-configuring-send-port-groups"></a>送信ポート グループの作成および構成

## <a name="overview"></a>概要
このセクションでは、BizTalk Server 管理コンソールを使用して送信ポート グループを BizTalk アプリケーションに追加する方法と、送信ポート グループを構成および削除する方法について説明します。 送信ポート グループは、送信ポートの名前付きコレクションです。送信ポート グループを使用すると、複数の送信先に同じメッセージを送信できます。 送信ポート グループは、単独の送信ポートの場合と同様に、オーケストレーション ポートにバインドしたり、コンテンツ ベースのルーティング (CBR) に使用できます。 送信ポート グループの基礎知識については、次を参照してください。[送信ポートおよび送信ポート グループ](../core/send-ports-and-send-port-groups.md)です。
