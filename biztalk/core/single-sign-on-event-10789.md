# <a name="single-sign-on-event-10789"></a>シングル サインオン: イベント 10789
## <a name="details"></a>詳細  
  
|||  
|-|-|  
|製品名|エンタープライズ シングル サインオン|  
|製品バージョン|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|イベント ID|10789|  
|イベント ソース|ENTSSO|  
|コンポーネント|なし|  
|シンボル名|ENTSSO_E_DTC_IMPORT_FAILED2|  
|メッセージ テキスト|DTC トランザクションをインポートできませんでした。 MSDTC がリモート操作用に正しく構成されていることを確認してください。 詳細については、イベント ログ (コンピューター "%1" 上) を参照してください。|  
  
## <a name="explanation"></a>説明  
 DTC トランザクションをインポートできませんでした。  
  
## <a name="user-action"></a>ユーザーの操作  
 MSDTC がリモート操作用に正しく構成されていることを確認してください。詳細については、指定されたコンピューターのイベント ログを確認してください。  
  
 MSDTC の問題に関するヘルプを参照するには、次を参照してください。 [MSDTC の問題のトラブルシューティング](../core/troubleshooting-problems-with-msdtc.md)です。
