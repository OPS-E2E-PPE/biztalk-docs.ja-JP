# <a name="information---display-xslt-file"></a>情報 - XSLT ファイルの表示
**エラー コード**  
  
 btm1055  
  
 **説明**  
  
 送信元スキーマに準拠するインスタンス メッセージから送信先スキーマに準拠するインスタンス メッセージへの、指定のマッピングを実行するために作成された XSLT が、対象のファイルに書き込まれました。  
  
 **ユーザーの操作**  
  
 作成された XSLT には、指定されたリンクを使用してアクセスできます。この XSLT を使用して、XSLT デバッガーなどの他のツールでさまざまな作業を実行できます。
