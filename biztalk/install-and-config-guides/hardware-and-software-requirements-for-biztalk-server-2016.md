# <a name="hardware-and-software-requirements-for-biztalk-server-2016"></a>BizTalk Server 2016 のハードウェアおよびソフトウェア

## <a name="hardware-requirements"></a>ハードウェア要件
次の表では、BizTalk Server コンピューターのハードウェアの最小要件を示します。 運用環境で、トラフィックの量は、サーバーの大きいハードウェア要件を必要があります。 

| リソース  |最小要件 |
| --- | --- |
|コンピューターとプロセッサ | Intel Pentium 互換の CPU を使用しているコンピューター: <ul><li>1 GHz 以上 (シングル プロセッサ向け)</li><li>900 MHz 以上 (ダブル プロセッサ向け)</li><li>700 MHz 以上 (クアッド プロセッサ向け)</li></ul> **注**:ハイパースレッディングおよびデュアルコア プロセッサがサポートされます。<br/><br/>64 ビット バージョンの BizTalk Server では、x64 ベースのシステムで実行されている 64 ビット オペレーティング システムが必要です。 コンピューターは、AMD64 と互換性がある Cpu に基づく (x86 64)、拡張 Memory 64-bit Technology (EM64T) プロセッサ アーキテクチャは x64 ベースのシステムと見なされます。<br/><br/>Itanium ベース システムでは、BizTalk Server はサポートされていません。 |
|Memory | 2 GB 以上|
|ハード_ディスク  |オペレーティング システムおよびすべての前提条件ソフトウェアを含む完全なインストールでは、ハード_ディスクの空き容量の 10 GB です。 ハード_ディスクは NTFS 形式である必要があります。|

> [!TIP] 
> ここにリストされているのは最小のハードウェア要件です。 環境はそれぞれ異なり、ご使用の環境ではこれらを超える要件が必要となる可能性は十分あります。 「[BizTalk Server ソリューションのインストール、サイズ変更、配置、およびメンテナンスを行うための推奨事項](http://social.technet.microsoft.com/wiki/contents/articles/666.recommendations-for-installing-sizing-deploying-and-maintaining-a-biztalk-server-solution.aspx)」を参照してください。

## <a name="service-pack-and-cumulative-update-support"></a>サービス パックと累積的な更新プログラムのサポート

すべてのサービス パック、累積的更新プログラム、セキュリティ更新プログラムおよび修正プログラムは、BizTalk Server でサポートされます。 Windows、SQL Server、Visual Studio、およびインストールされている任意のプログラムの最新の更新プログラムのインストールを強くお勧めします。 Microsoft 製品のサービス パックは、その製品の基本サポートに基づいてサポートされます。 参照してください[サポート ライフ サイクルの](http://go.microsoft.com/fwlink/p/?LinkID=151890)の BizTalk Server、SQL Server、Visual Studio、およびその他のマイクロソフト製プログラム。

[BizTalk Server のサービス パックと累積更新プログラムの一覧](https://support.microsoft.com/help/2555976)
